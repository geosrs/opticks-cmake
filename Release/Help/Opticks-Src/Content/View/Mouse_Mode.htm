Mouse Mode

Mouse mode enables you to perform specialized functions using your mouse when that function is enabled, usually by clicking a button on one of the Opticks toolbars.

Plug-ins may add their own mouse mode functionality.

Here are the mouse mode buttons found in Opticks.

Function Button Toolbar See...
Zoom in on mouse Display Toolbar Zoom
Zoom out on mouse Display Toolbar Zoom
Zoom rectangle Display Toolbar Zoom
Pan Display Toolbar Pan
Free Rotate Display Toolbar Rotate
Layer Edit Mode Toolbox Toolbar Layer Edit Mode
Measurement Mode Measurement Toolbar Measurement Toolbar

To turn off the mouse mode, click the Edit Mode Off button on the Toolbox Toolbar.
