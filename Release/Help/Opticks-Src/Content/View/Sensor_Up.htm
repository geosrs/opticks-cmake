Sensor Up

Sensor Up rotates the image display based on an angle entered into the raster layer metadata. Currently this angle must be entered manually.

To enter the rotation angle:

  1. In the Session Explorer, select the Elements Tab,

  1. Select the image and right-click. Select Properties.
  2. On the Properties window, select the Metadata tab.

  1. Click the Modify Values button.

The following message box might be displayed. Click OK.

  1. Select the metadata value "Special" and click the Add Child button.

  1. In the Add Metadata Value window, enter the information as follows:

Name: SensorUpAngle

Type: double

Value: The angle you want the image to rotate counter-clockwise. For this example we'll enter "33." You can also enter a negative value to rotate clockwise. The value should be in degrees.

Click OK.

  1. In the Properties window, click OK.

To apply the rotation:

Click the Sensor Up button on the Display Toolbar.
