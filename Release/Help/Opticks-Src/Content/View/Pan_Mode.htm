Pan Mode

The Pan Mode allows you to determine when the data display refreshes during a pan.

To select the pan mode:

  1. On the Opticks Main Application window, go to the View menu. Select Pan Mode.
  2. Select the pan mode from the sub-menu.

The pan modes are:

  • Pan Delay. The display refreshes after you release the mouse.
  • Pan Instant. The display refreshes as you pan the display. This is the default.
