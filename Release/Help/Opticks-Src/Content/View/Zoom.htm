Zoom

The Zoom selections on the View menu allow you to zoom in and out on the display. (You can zoom in to the pixel level.)

To select a Zoom option:

On the Opticks Main Application window, go to the View menu. Select Zoom and select the zoom option from the sub-menu.

OR

Click the appropriate button on the Display Toolbar.

The View menu zoom options are:

  • Center. Allows you to zoom in or out around the center of the display. Use the sub-menu to select In or Out.
  • Point. Allows you to zoom in or out around a point on the display. Use the sub-menu to select In or Out, then use your mouse to select the point.
  • Percent. Allows you to display the image at a percentage of its resolution. Use the sub-menu to select the percentage.
  • Rectangle. Allows you to display an area of the image by drawing a rectangle over the area.
  • To Fit. Displays the data area so that it fits within the current window size.
  • Zoom and Pan to Point. Allows you to enter a pixel coordinate for the center of the window and a zoom percentage.

Note: The zoom function depends on the zoom limits set in the options. For example, you cannot zoom in past the maximum zoom in limit. See Windows Workspace Options - Cube for more information.
