Color Composites

Opticks provides three predefined color composites for image display: NIR, SWIR, and True Color.

You can customize these composites. You can also create new composites.

If Opticks is unable to apply the color composite to the image (because the required wavelengths don't exist), it will display an error message. It will try to match what it can.

You can switch color composites as often as you like. You can use the Undo button on the Standard Toolbar to return to the previous display.

Apply a Color Composite to an Image

Option 1

  1. Right-click on the image.
  2. Select the image name.
  3. Select Display As.
  4. From the list, select the color composite.

Option 2

  1. Go to the Session Explorer. Be sure the Windows tab is displayed.
  2. Go to Workspace Windows and expand the image name.
  3. Right-click on the raster layer and select Display As.
  4. From the list, select the color composite.

Option 3

When importing raster data:

  1. On the Import window, select Options.

  1. On the Import Options window, be sure the Data tab is displayed.
  2. Click the Set Display Bands button and select the color composite from the drop-down list.

  1. Click the OK button.
  2. On the Import Window, click the Open button.

Option 4

  1. Go to the Session Explorer. Be sure the Windows tab is displayed.
  2. Right-click on the raster layer.
  3. Select Properties.

  1. In the Properties window, go to the RGB section and select the Presets button.
  2. Select the color composite from the drop-down list.

Customize Color Composites

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Layers to display a list of layers. Click Raster.

The color composites are listed towards the bottom of the Options window. The example below displays all the color composites and their default values.

Note that each color composite includes the minimum wavelength, maximum wavelength, and band number for red, green, and blue, for a total of nine values for each composite. Opticks selects the median value between the minimum and maximum values. The minimum and maximum wavelengths take precedence over the band number. In general, the band number is used as a fallback value if the wavelengths are unavailable or are not known.

To edit a color composite:

  1. Double-click in the cell of the value you want to edit.
  2. Enter the new value.
  3. Press the Enter key on your keyboard to accept the new value.

To create a new color composite:

  1. Click the Add button.
  2. Opticks creates a new color composite called "New Color Composite."
  3. Double-click on New Color Composite and give the color composite a new name.

Be sure all your color composites have unique names. If Opticks encounters several color composites with the same name, it will keep one and delete the others.

  1. Double-click in each cell to enter the values for the new color composite.
  2. Click OK.

The new color composite will now be displayed in the "Display As" list.

To delete a color composite:

  1. Select the color composite you want to delete.
  2. Click the Remove button. No confirmation dialog is displayed.

Important: If you delete any of the default color composites, they cannot be restored. However, you can recreate them. Click here for the default values for the Opticks color composites.
