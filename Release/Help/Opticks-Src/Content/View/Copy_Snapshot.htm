Copy Snapshot

Note: This feature is available in Microsoft Windows only.

Opticks enables you to create a snapshot of the current image and copy it to the Clipboard. You can then use the Clipboard function to paste the image to another application such as Microsoft Word or Microsoft PowerPoint. If you have more than one image displayed, the snapshot is taken of the active window image.

You can either create a snapshot using the defaults set in Windows Workspace Options - All or you can set the image size.

To create a snapshot using the defaults:

Use the keyboard shortcut Ctrl+Shift+C.

OR

  1. On the Session Explorer window, be sure the Windows tab is displayed.
  2. Right-click on the image name.
  3. Select Copy snapshot. (Be sure you select the first "Copy snapshot" entry.)

The image is saved to your Clipboard.

To copy a snapshot by setting the size:

  1. On the Session Explorer window, be sure the Windows tab is displayed.
  2. Right-click on the image name.
  3. Select Copy snapshot.... (Be sure you select the second "Copy snapshot" entry.)

The Copy Snapshot dialog is displayed.

  1. In the Copy Snapshot dialog, enter the Width and Height of the snapshot you want to copy.
  2. To lock the aspect ratio, click the Lock Aspect Ratio button. This button is a toggle.
  3. Click the OK button.

The image is saved to your Clipboard.
