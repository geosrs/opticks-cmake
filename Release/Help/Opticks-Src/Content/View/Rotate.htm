Rotate

The Rotate selections on the View menu allow you to rotate the display.

To select a Rotate option:

On the Opticks Main Application window, go to the View menu. Select Rotate and select the rotate option from the sub-menu.

OR

Click the appropriate button on the Display Toolbar.

Button

What it does

Rotate Left 90°. Rotates the data display counter-clockwise by 90°.

Rotate Right 90°. Rotates the data display clockwise by 90°.

Flip Horizontally. Flips the data display horizontally (from left to right).

Flip Vertically. Flips the data display vertically (from top to bottom).

Rotate By. Rotates the data display by a number of degrees you enter. When you select this option, a dialog box is displayed. This box allows you to enter the rotation degrees from 360 to -360.

Free Rotate. Allows you to freely rotate the data display by moving your mouse over the display.

Reset. Resets the data display to its original orientation.
