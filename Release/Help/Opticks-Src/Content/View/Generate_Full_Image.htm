Generate Full Image

Normally, Opticks prepares portions of an image to display as they're needed. This takes a small, but noticeable amount of time and may cause the smooth pan function to hesitate while Opticks renders a portion of the image.

To avoid this hesitation:

Click the Generate Full Image button on the Display Toolbar.

OR

Go to the View menu and select Generate Full Image.

This forces Opticks to fully render the image at full resolution. Note, however, that the smooth pan speed depends on the speed of your computer and the associated graphics hardware. When smooth panning, Opticks will redraw as fast as the computer is capable of. To specify the number of pixels used for each redraw, see Windows Workspace Options - Cube.

To use the smooth pan, use the Ctrl + Arrow Key. To smooth pan rapidly, use Ctrl + Shift + Arrow Key.
