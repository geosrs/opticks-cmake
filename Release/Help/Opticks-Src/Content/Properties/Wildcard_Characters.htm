Wildcard Characters

Let's say you have data identified as "mydata" and there are 20 numbered entries: mydata1, mydata2, etc.

* matches a single character or a string of characters. So if you enter:

mydata*

the search results will be everything called "mydata." In other words, mydata1 through mydata20.

? matches only a single character. So if you enter:

mydata?

the search results will be only mydata1 through mydata9. The remaining data—mydata10 through mydata20—will not be included in the search results because there are two characters after "mydata" instead of only one character.
