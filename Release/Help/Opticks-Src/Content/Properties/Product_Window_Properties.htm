Product Window Properties

A product window is not associated with a single data set but allows you to create products by arranging annotation objects on a virtual sheet of paper.

The product window view has two annotation layers:

  • Classification Layer
  • Layout Layer

The product window, the layout layer, and the classification layer each have different properties in the Properties dialog box.

Product Window Properties

To open the Product Window properties in the Properties dialog box:

  1. Right-click in the Product Window and select Properties....

OR

Go to the Session Explorer window. Navigate to the Product Window name. Right-click and select Properties....

The Product Properties dialog box is displayed. This dialog box has two tabs: View and Product View.

  1. When you’re finished making the changes:

To apply the new properties and close the dialog box, click the OK button.

To apply the new properties and leave the dialog box open, click the Apply button.

To cancel the changes and close the dialog box, click the Cancel button.

View Tab

You can modify the background color and the origin location. The origin location determines where the product window is displayed in the Opticks Main Application window (upper left or lower left).

Checking the Crosshair checkbox displays a crosshair in the center of the product window.

For more information on choosing colors, see Edit Color.

Classification Tab

On the Classification Tab, you can change the classification for the product. The Classification Tab has two sub-tabs: Top + Bottom and Descriptor.

Top + Bottom Sub-Tab

Use the Top + Bottom Tab to select the markings displayed on the top and bottom of the product window.

Note: The classification "Confidential" is used here for illustration purposes only.

  1. Select the Classification Level from the drop-down list.
  2. Select the markings type from the list on the left side of the tab.
  3. Select the marking from the list of available markings on the right side of the tab.
  4. Your selections are displayed in the Preview box.

Click OK to apply the markings.

To add a marking:

  1. In the Top + Bottom list, select "System" or "Declassification."
  2. Click the Add... button.
  3. In the small Security Markings dialog box, type in the new marking.
  4. Click OK.

This marking will be available for all classification levels.

To reset the markings:

  1. Select the Classification Level from the drop-down list.
  2. Click the Reset button.

Descriptor Sub-Tab

This tab allows you to specify additional classification information that does not appear in the markings in the view.

Note: The classification "Confidential" is used here for illustration purposes only.

  1. Select the Classification Reason from the drop-down list.
  2. Select the Declassification Type from the drop-down list.
  3. To change the Declassification Date, select the month, date, or year to highlight it. Type in the new date information or use the arrow buttons to select new information.
  4. Select the File Downgrade name from the drop-down list.
  5. To change the Downgrade Date, select the month, date, or year to highlight it. Type in the new date information or use the arrow buttons to select new information.
  6. Select the File Control name from the drop-down list.
  7. Use the Description field to enter descriptive text.
  8. Select the Copy Number and the Number of Copies.

Favorites

The Favorites area of the Classification tab allows you to save classification strings.

To save the classification string as it’s displayed in the Preview box, click the Add to favorites button.

To use a favorite for the current classification, select it from the Favorites drop-down list. The Top + Bottom tab will be filled with the contents of the favorite.

To remove a string from the list, select it from the Favorites drop-down list and click the Remove from favorites button.

Product View Tab

This tab enables you to set the color for the product view area.

Classification Layer

The Classification Layer allows you to edit the properties for the classification markings text objects.

To open the Classification Layer:

  1. Go to the Session Explorer window.
  2. Expand the Product Name.
  3. Select Classification and the Text item.
  4. Right-click and select Properties....

You can edit these areas:

  • General. Determines where the text is displayed.
  • Fill
  • Line
  • Text. Use this area to enter the text and set its display.

Layout Layer

The Layout Layer Properties become available after you select an annotation layer for the product window. For more information about working with the annotation layer, see Annotation Layer.

The properties for the Layout Layer are the same as those for the Annotation Layer Properties.
