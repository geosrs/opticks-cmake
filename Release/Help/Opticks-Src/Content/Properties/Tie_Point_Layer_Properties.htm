Tie Point Layer Properties

For each Tie Point Layer, you can change the color and size of the tie point labels. You can also turn the label display on and off. These properties can be edited independently for both the reference and mission scenes.

To open the Tie Point Layer Properties dialog box:

  1. Go to the Session Explorer window and select the Tie Point layer name.

  1. Right click. On the Layer Menu, select Properties....

The Tie Point properties dialog box is displayed.

To change the label size, select the number from the Symbol Size list or type in a new number.

To change the label color, select the color you want to use from the Color drop-down or click the Custom... button to select a custom color. For more information on working with color, see Edit Color.

To turn the label display on or off, check or uncheck the Display Labels checkbox.

To apply the new properties and close the dialog box, click the OK button.

To apply the new properties and leave the dialog box open, click the Apply button.

To cancel the changes and close the dialog box, click the Cancel button.
