Properties

Opticks provides the ability to set image display and other properties for any view, plug-in, or data element. Properties are set after a view, plug-in, or data element is created. Options are the defaults for a view that hasn't been created yet.

To open the Properties dialog box:

  1. On the Opticks Main Application window, go to View and select Properties....

OR

On the Toolbox Toolbar, click the Properties button.

OR

Go to the Session Explorer window and select the item. Right-click and select Properties.

OR

Right-click on the main spatial data view.

  1. The Properties dialog box is displayed.

When a view is selected, this dialog box has three tabs: View, Classification, and Spatial Data View. Click the tab name to activate the tab.

  1. When finished with your selections, click the OK button to apply the selections and close the dialog box.

Click the Apply button to apply the changes but leave the dialog box open.

Click the Cancel button to cancel the selections and close the dialog box.

View Tab

On the View tab, you can set the defaults for the image display area of the Opticks Main Application window.

  1. Use the Background Color drop-down list to set the background color for the image display.
  2. Use the Origin Location drop-down list to select the placement of an image when it is opened.
  3. When the Crosshair checkbox is checked, a crosshair is displayed in the center of the display area.

Classification Tab

On the Classification Tab, you can change the classification for the view, plug-in, or data element. The Classification Tab has two sub-tabs: Top + Bottom and Descriptor.

Top + Bottom Sub-Tab

Use the Top + Bottom Tab to select the markings displayed on the top and bottom of the window.

Note: The classification "Confidential" is used here for illustration purposes only.

  1. Select the Classification Level from the drop-down list.
  2. Select the markings type from the list on the left side of the tab.
  3. Select the marking from the list of available markings on the right side of the tab.
  4. Your selections are displayed in the Preview box.

Click OK to apply the markings to the window.

To add a marking:

  1. In the Top + Bottom list, select "System" or "Declassification."
  2. Click the Add... button.
  3. In the small Security Markings dialog box, type in the new marking.
  4. Click OK.

This marking will be available for all classification levels.

To reset the markings:

  1. Select the Classification Level from the drop-down list.
  2. Click the Reset button.

Descriptor Sub-Tab

This tab allows you to specify additional classification information that does not appear in the markings in the view.

Note: The classification "Confidential" is used here for illustration purposes only.

  1. Select the Classification Reason from the drop-down list.
  2. Select the Declassification Type from the drop-down list.
  3. To change the Declassification Date, select the month, date, or year to highlight it. Type in the new date information or use the arrow buttons to select new information.
  4. Select the File Downgrade name from the drop-down list.
  5. To change the Downgrade Date, select the month, date, or year to highlight it. Type in the new date information or use the arrow buttons to select new information.
  6. Select the File Control name from the drop-down list.
  7. Use the Description field to enter descriptive text.
  8. Select the Copy Number and the Number of Copies.

Favorites

The Favorites area of the Classification tab allows you to save classification strings.

To save the classification string as it’s displayed in the Preview box, click the Add to favorites button.

To use a favorite for the current classification, select it from the Favorites drop-down list. The Top + Bottom tab will be filled with the contents of the favorite.

To remove a string from the list, select it from the Favorites drop-down list and click the Remove from favorites button.

Spatial Data View Tab

On the Spatial Data View tab, you can set the defaults for the spatial data view.

  1. Pan Limit. Controls the limit that a scene can be panned in the view.
  2. When Zooming In. The minimum number of pixels that can be shown.
  3. When Zooming Out. The data should occupy at least this much of the display window.
  4. Smooth pixel edges. When zoomed in, the pixel edges are sharp by default. When you check this checkbox, the edges are smoothed or softened.
