Annotation Layer Properties

The Annotation Layer properties enable you to update properties for all objects in the annotation layer.

Note: The properties for the Layout Layer of the Product window are identical to the Annotation Layer properties.

To open the Annotation Layer Properties dialog box:

  1. Go to the Session Explorer window and select the annotation layer name.

  1. Right click. On the Layer Menu, select Properties....

OR

  1. Go to the Annotation Toolbar and select the Annotation button.
  2. On the Annotation menu, select Properties.

Below is an example of the Properties dialog box for a rectangle. The properties that you can edit vary by object.

Select the object in the list on the left. For more information on each properties type, see Properties for Annotation Objects.

To apply the new properties and close the dialog box, click the OK button.

To apply the new properties and leave the dialog box open, click the Apply button.

To cancel the changes and close the dialog box, click the Cancel button.
