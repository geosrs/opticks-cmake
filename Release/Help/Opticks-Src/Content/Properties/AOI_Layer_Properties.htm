AOI Layer Properties

For each AOI Layer, you can change the shape and color of the pixel markers.

To open the AOI Layer Properties dialog box:

  1. Go to the Session Explorer window and select the AOI layer name.

  1. Right click. On the Layer Menu, select Properties....

The AOI properties dialog box is displayed.

To change the shape of the pixel marker, select the shape you want to use in the Symbol drop-down.

To change the color, select the color you want to use from the Color drop-down. To select a custom color, in the Color drop-down, click the Custom... button. For more information on working with color, see Edit Color.

You can add different object shapes to an AOI layer (similar to an annotation layer). Each object can have its own name. The Shape checkbox toggles the display of the object names.

To apply the new properties and close the dialog box, click the OK button.

To apply the new properties and leave the dialog box open, click the Apply button.

To cancel the changes and close the dialog box, click the Cancel button.
