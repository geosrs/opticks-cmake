Working with Metadata

On the Data Elements Properties window, you can add, edit, delete, and clear the metadata for the file or a component.

Add Metadata

  1. Be sure the Metadata tab is displayed.

  1. On the Metadata tab, click the Modify Values button.

  1. Click OK.

If you don't want this dialog to be displayed again, check the Don't show this again checkbox.

  1. Click the Add Child button (if there is no data) or the Add Sibling button (if there is already data).

The Add Metadata Value dialog box is displayed.

  1. In the Name field, enter the metadata name.
  2. Select the type from the Type list. For more information on the value types, see Value Item Types.
  3. In the Value box, enter the metadata value.
  4. Click OK.

Edit Metadata

  1. Be sure the Metadata tab is displayed.
  2. If necessary, click the Modify Values button. Click the Ignore button or the Yes button.
  3. Click the Edit... button.

The Edit Metadata Value dialog box is displayed.

  1. Edit the metadata value.
  2. Click the OK button.

Working with Filters

You can create, edit, and delete filters that hide metadata attributes. See Metadata Filters.

Delete Metadata

  1. Be sure the Metadata tab is displayed.
  2. If necessary, click the Modify Values button. Click the Ignore button or the Yes button.
  3. Select the metadata from the list. Click the Delete button.

The metadata is deleted. (There is no confirmation box.)

Clear All Metadata

This removes all metadata for the selected element.

  1. Be sure the Metadata tab is displayed.
  2. If necessary, click the Modify Values button. Click the Ignore button or the Yes button.
  3. Click the Clear button.
