Properties for Annotation Objects

Property Definition Applies to
(Annotation Objects) 		Edit Caption
Alpha The alpha value is a transparency index for the image ranging from 0 to 255. If the value is 0, the image is transparent. If the value is 255, the image is opaque. Image Image
Apex The Apex property applies to the triangle object only. It determines the position of the triangle’s apex. Valid values are 0.0 to 1.0. Triangle Apex
Bounding Box All annotation objects are surrounded by a bounding box that identifies the object’s location using screen pixel coordinates. The box is defined using its lower left and upper right coordinates. You can edit these coordinates. Arc, Arrow, Ellipse, Image, Line, Polygon, Polyline, Rectangle, Rounded Rectangle, Text Box, Triangle, Latitude/Longitude, Measurement, North Arrow, East Arrow, View, Scale Bar Bounding Box
Fill Color Allows you to select the color used to fill the object. For more information on selecting colors, see Edit Color. Arc, Ellipse, Image, Polygon, Rectangle, Rounded Rectangle, Text Box, Triangle, View, Scale Bar, Latitude/Longitude, North Arrow, East Arrow Fill
Fill Style A drop-down box that allows you to select the style used to fill the object. Arc, Ellipse, Image, Polygon, Rectangle, Rounded Rectangle, Text Box, Triangle, View, Scale Bar, Latitude/Longitude Fill
Font Allows you to select the text font, size, alignment, and style (bold, italic, underlined). Text Box Text
Hatch Style Allows you to choose the line style for hatch fill. The fill style must be set to “Hatch.” Arc, Ellipse, Image, Polygon, Rectangle, Rounded Rectangle, Text Box, Triangle, View, Scale Bar, Latitude/Longitude Fill
Filename The Image properties provide access to the image object’s file name and alpha value. Image Image
Line Color Allows you to select the line color for an object. For more information on selecting colors, see Edit Color. Arc, Arrow, Ellipse, Image, Line, Polygon, Polyline, Rectangle, Rounded Rectangle, Text Box, Triangle, View, Scale Bar, Latitude/Longitude, North Arrow, East Arrow Line
Line State Toggles the display of the line border by selecting or de-selecting the “Object Border” checkbox. Arc, Ellipse, Image, Polygon, Rectangle, Rounded Rectangle, Text Box, Triangle, View, Scale Bar Line
Line Style Use this drop-down list to choose the style of the line (solid, dashed, dot, etc.) Arc, Arrow, Ellipse, Image, Line, Polygon, Polyline, Rectangle, Rounded Rectangle, Text Box, Triangle, View, Scale Bar, Latitude/Longitude Line
Line Width Use this box to select the width of the line. Arc, Arrow, Ellipse, Image, Line, Polygon, Polyline, Rectangle, Rounded Rectangle, Text Box, Triangle, View, Scale Bar, Latitude/Longitude, North Arrow, East Arrow Line
Rotation The Rotation property allows you to edit the rotation angle of the object from 0 to 360 degrees. Arc, Arrow, Ellipse, Image, Line, Polygon, Polyline, Rectangle, Rounded Rectangle, Text Box, Triangle, North Arrow (if not oriented to North, (East Arrow, Scale Bar Rotation
Text Color Allows you to select the text color. For more information on selecting colors, see Edit Color. Text Box, Latitude/Longitude, North Arrow, East Arrow Text
Text String Allows you to edit the text in the text box. Text Box Text
Unit System Allows you to select the units associated with a scale bar. Scale Bar Scale Bar
Wedge The Wedge properties identify the start and stop angle for an arc object. Valid values are 0 to 360 degrees. Arc Wedge
View Allows you to select an existing view to display in the object. View View
