Scripting Window Properties

To access the Scripting Window Properties:

  1. Go to the Session Explorer window. Expand Dock Window and select Scripting Window.

  1. Right-click and select Properties....

The Scripting Window Properties dialog box is displayed.

  1. To apply the new properties and close the dialog box, click the OK button. 

To apply the new properties and leave the dialog box open, click the Apply button.

To cancel the changes and close the dialog box, click the Cancel button.

General

The window Scroll Buffer identifies how many lines are added to the scrolling window before lines are removed at the top of the buffer. A “line” is a single command entry to a single processing results entry (normal or error). The default number of lines displayed in the scripting window is 512.

Input Text

The Input Text area enables you to select how input text is formatted.

Output Text

To change the color of the Output message or Error message, click the Color... button. The Select color dialog box is displayed. For more information, see Edit Color.

To change the font of the Output message or Error message, click the Font... button. The Select Font dialog box is displayed.
