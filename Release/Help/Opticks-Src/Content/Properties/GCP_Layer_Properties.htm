GCP Layer Properties

For each GCP Layer, you can change the shape, size, and color of the markers. This layer is called the “GCP List” or the “Corner Coordinates.” (“Corner Coordinates” is the name of the GCP list created when a data set is loaded if the data set contains that information.)

To open the GCP Layer Properties dialog box:

  1. Go to the Session Explorer window and select the GCP layer name.

  1. Right click. On the Layer Menu, select Properties....

OR

  1. Open the GCP Editor window.
  2. Click the Properties... button.

The GCP properties dialog box is displayed.

To change the shape, select the shape you want to use in the Symbol grid. The "No" symbol indicates that no symbols will be displayed for the GCP layer.

To change the size of the marker, select the size from the Symbol Size box. The size applies to the size of the symbol in on-screen pixels.

To change the color, select the color you want to use from the Color drop-down list or click the Custom... button to select a custom color. For more information on working with color, see Edit Color.

To apply the new properties and close the dialog box, click the OK button.

To apply the new properties and leave the dialog box open, click the Apply button.

To cancel the changes and close the dialog box, click the Cancel button.
