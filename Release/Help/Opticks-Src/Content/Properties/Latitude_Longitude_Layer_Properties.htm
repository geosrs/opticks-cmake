Latitude/Longitude Layer Properties

The Lat/Long Properties dialog enables users to format the Lat/Long grid displayed in the results layer from georeferencing the current data set.

To open the Lat/Long Layer Properties dialog box:

  1. Go to the Session Explorer window and select the Geo Results layer name.

  1. Right click. On the Layer Menu, select Properties.... The layer property box is displayed.
  2. When you’re finished updating the properties:

To apply the new properties and close the dialog box, click the OK button.

To apply the new properties and leave the dialog box open, click the Apply button.

To cancel the changes and close the dialog box, click the Cancel button.

Coordinates

Select the Coordinate Type:

  • Latitude/Longitude. This is the default. Use the drop-down list to select the format.
  • UTM Coordinates. Universal Transverse Mercator Coordinates.
  • MGRS. Military Grid Reference System.

Use the Font drop-down list to select the font type.

Use the Font Size drop-down list to select the font size.

Gridlines

Use the Gridlines area to edit the Width, Style, and Color of the gridlines.

Spacing

The Spacing area contains the controls for the distance between the gridlines on the layer.

Automatic computes a reasonable grid spacing based on the extents of the data set. When the Autogrid box is checked, the Latitude and Longitude boxes are disabled but show the computed value.

Custom enables you to enter values in the Latitude and Longitude boxes.

If the latitude or longitude spacing is set to 0, the gridlines will not appear.
