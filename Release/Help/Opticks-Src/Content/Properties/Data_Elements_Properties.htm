Data Elements Properties

The Data Elements Properties dialog displays general data, file data, and a list of all metadata for a file or component. The general data and file data are read-only. You can edit the metadata.

If you select a component, only the general data and metadata are displayed in the data elements properties window.

To open the Data Elements Properties window:

  1. Go to the Session Explorer window. Select the Elements tab.

  1. Select the file name or component name and right-click. Select Properties....

The Properties window for the file has three tabs: Data, File, and Metadata.

  1. To apply the new properties and close the dialog box, click the OK button. 

To apply the new properties and leave the dialog box open, click the Apply button.

To cancel the changes and close the dialog box, click the Cancel button.

Data Tab

The Data Tab provides basic import data information such as file name and type. For raster elements, it also provides more detailed information such as the number of rows and columns, pixel size, and display mode. The example below provides detailed information. Note that you can expand the Units field to display more units information.

The information on this tab is display only.

File Tab

The File Tab provides detailed file information. If GCPs are identified, it lists the location for each GCP.

The information on this tab is display only.

Classification Tab

On the Classification Tab, you can change the classification for the data element. The Classification Tab has two sub-tabs: Top + Bottom and Descriptor.

Top + Bottom Sub-Tab

Use the Top + Bottom Tab to select the markings displayed on the top and bottom of the window.

Note: The classification "Confidential" is used here for illustration purposes only.

  1. Select the Classification Level from the drop-down list.
  2. Select the markings type from the list on the left side of the tab.
  3. Select the marking from the list of available markings on the right side of the tab.
  4. Your selections are displayed in the Preview box.

Click OK to apply the markings.

To add a marking:

  1. In the Top + Bottom list, select "System" or "Declassification."
  2. Click the Add... button.
  3. In the small Security Markings dialog box, type in the new marking.
  4. Click OK.

This marking will be available for all classification levels.

To reset the markings:

  1. Select the Classification Level from the drop-down list.
  2. Click the Reset button.

Descriptor Sub-Tab

This tab allows you to specify additional classification information that does not appear in the markings in the view.

Note: The classification "Confidential" is used here for illustration purposes only.

  1. Select the Classification Reason from the drop-down list.
  2. Select the Declassification Type from the drop-down list.
  3. To change the Declassification Date, select the month, date, or year to highlight it. Type in the new date information or use the arrow buttons to select new information.
  4. Select the File Downgrade name from the drop-down list.
  5. To change the Downgrade Date, select the month, date, or year to highlight it. Type in the new date information or use the arrow buttons to select new information.
  6. Select the File Control name from the drop-down list.
  7. Use the Description field to enter descriptive text.
  8. Select the Copy Number and the Number of Copies.

Favorites

The Favorites area of the Classification tab allows you to save classification strings.

To save the classification string as it’s displayed in the Preview box, click the Add to favorites button.

To use a favorite for the current classification, select it from the Favorites drop-down list. The Top + Bottom tab will be filled with the contents of the favorite.

To remove a string from the list, select it from the Favorites drop-down list and click the Remove from favorites button.

Metadata Tab

The Metadata Tab lists any metadata associated with the data element. You can also create, edit, and delete filters that hide metadata attributes. The metadata is the only information you can edit on this window. See Working with Metadata.

Wavelengths Tab

The Wavelengths Tab provides the capability to load or save wavelengths in a file and to assign wavelengths to a loaded data set.

The Wavelengths Tab is only available for raster data sets.

For input, this tool takes a wavelengths file or the wavelengths included in a data cube and writes an output wave file or writes the wavelengths to the current data cube.

The Wavelengths Tab functionality should be executed before running any plug-in that requires wavelength data to run. The Wavelengths Tab also can be used to review the wavelengths of a specific data set.

If the data set already contains all required wavelength information (start, end, and center channel wavelengths), the Wavelengths Tab is not needed.

For more information, see Wavelengths Tab.

ENGRDA

If the loaded data set includes NITF ENGRDA TREs, the Data Elements Properties window includes an ENGRDA tab. The information that's included depends on the data in the data set.

Component Window

The Properties window for a component (such the Corner Coordinates) has three tabs: Data, Classification, and Metadata. Note that the data tab for a component does not contain as much information as the data tab for the whole file.
