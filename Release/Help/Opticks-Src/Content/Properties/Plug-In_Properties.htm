Plug-In Properties

Plug-in properties provide general information about each plug-in such as general information, settings, inputs, and outputs. Plug-in properties cannot be edited.

To view plug-in properties:

  1. On the Session Explorer window, select the Plug-Ins tab.

  1. The Plug-Ins tab contains a list of all plug-ins organized into a directory tree.

The directory tree displays a list of modules (DLLs). When you expand the module, it lists the plug-ins associated with the module. When you a plug-in, it displays the number of running instances of the plug-in.

To display the properties for a module or plug-in, navigate to and select the module or plug-in from the list.

  1. Right-click on the plug-in name and select Properties....

The Plug-In Properties dialog is displayed. It provides general information about the plug-in.

(The Apply button is not operational for modules or plug-ins.)
