Metadata Filters

You can create, edit, and delete filters that hide metadata attributes.

To enable filters, check the Filter checkbox on the Metadata tab.

Create a Filter

  1. Click the Create Filter button.

You can specify a filter name or use the default. For the metadata attributes, you can enter a name and/or value. Use the Enable Wildcarding and Case Sensitive checkboxes to specify partial matches or case sensitive matches. Opticks supports the wildcard characters * and ?. For an explanation of these characters, click here.

  1. Enter the filter parameters you want to use.
  2. Click the OK button.

Edit a Filter

  1. Be sure the Filter box is checked.
  2. Select the Filter from the drop-down list.

  1. Click the Edit Filter button.

  1. Edit the filter parameters.
  2. Click the OK button.

Delete a Filter

Note: When you delete a filter, be sure you select the correct filter name before you click the Delete button. You do not get a confirmation dialog box when deleting filters.

  1. Be sure the Filter box is checked.
  2. Select the Filter from the drop-down list.

  1. Click the Delete Filter button.

The filter is deleted.
