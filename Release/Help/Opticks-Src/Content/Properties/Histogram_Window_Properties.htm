Histogram Window Properties

For each Histogram Window plot, you can modify the width, style, and color of the gridlines and the color of the histogram.

To open the Histogram Window Properties dialog box:

  1. Go to the Session Explorer window. Expand the Dock Windows. Go to the Histogram Window and select the histogram band.

  1. Right-click. In the Histogram Window menu, select Properties.

The Histogram Windows Properties dialog box is displayed. This dialog box has two tabs: the Plot View tab and the Histogram Plot tab.

  1. Use the Plot View tab to edit the style, width, and color of the gridlines. See also Display Gridlines on a Plot.

Use the Histogram Plot tab to set the color of the histogram.

For more information on working with color, see Edit Color.

  1. To apply the new properties and close the dialog box, click the OK button.

To apply the new properties and leave the dialog box open, click the Apply button.

To cancel the changes and close the dialog box, click the Cancel button.
