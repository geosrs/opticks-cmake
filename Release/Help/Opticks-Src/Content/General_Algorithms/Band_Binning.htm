Band Binning

The Band Binning algorithm enables you transform an image by removing and/or combining bands. This can be useful to reduce the dimensionality of the image such as taking the average of the "red" bands and combining them into a single band in the output.

Bad values are removed from the calculation. The algorithm runs much faster when either all bad values are removed from the data prior to processing or when all bad values are uniform across all layers to be averaged. Although the algorithm will process data of any interleave, it will usually work fastest with BIP data. Interleave conversion is possible on import through Import Options. The results will be displayed in a new window and will have the same data type (such as signed 2-byte integers). Integer data types will be truncated when necessary. For example, “(5 + 6)/2” will result in an output of 5.

Band groups can be thought of as a range from the first band to the last band. This means that a bin defined as “from band 2 to band 4” will assign the average of bands 2, 3, and 4 to the corresponding output band.

It is possible to remove bands from the output by not including them in any groups. It is also possible to include a band in more than one group such as “from band 3 to band 6” and “from band 5 to band 7”. A group can contain only one band such as “from band 7 to band 7”. However, it is not possible to have gaps in the bins such as “from band 7 to band 10, excluding band 8”.

The Band Binning algorithm is available through the General Algorithms Toolbar. It is also available on the Wizard Builder in both batch and interactive mode.

To run the Band Binning algorithm:

  1. Go to the General Algorithms Toolbar and select Band Binning.

The Band Binning Window is displayed.

  1. Enter the first and last input band numbers for each bin number or load a binning file. Use the buttons at the bottom of the window to manage the bins or load a file.
  2. When your inputs are entered, click the OK button.

The algorithm runs and the image is displayed in a new window.

  • The values in Bands 3 though 9 will be averaged and placed in the new image as Band 2.

Binning Files

You can save band bins to a .txt file and load it again for later use. If you run the Band Binning algorithm from a wizard in batch mode, the file is required. In interactive mode, the file is optional and is used to initialize the GUI.

The band binning values in this Select Band Bins window:

will look like this if you save them to a .txt file:

1 7

3 9

12 15

You can modify the values directly in the .txt file by opening a text editor and making the changes.
