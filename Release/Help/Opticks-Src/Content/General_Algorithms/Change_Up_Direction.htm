Change Up Direction

Change Up Direction enables you to change the orientation of the image based on an arrow you draw on the image.

  1. Create an annotation layer or activate an existing one.
  2. On the Annotation Toolbar, make sure that Add New Object is an arrow.

  1. On the image, draw an arrow so that it points in the direction that you want to use as the "up" direction.

  1. Go to the General Algorithms Toolbar and select Change Up Direction.

  1. When the Change Up Direction dialog box indicates that the rotation is complete, click the Close button

The image will be reoriented to the direction the arrow is pointing to.
