Flatten Annotation Layer

Flatten Annotation Layer merges the annotation layer with the image. It only works for images in the currently active annotation layer. It ignores other annotation objects.

After you have created the annotation layer that you want to merge:

  1. Go to the General Algorithms Toolbar and select Flatten Annotation Layer.

  1. The following dialog box warns you that flattening the annotation layer will clear the undo history in your current session.

Click Yes to flatten the annotation layer and clear the undo history. Click No to cancel flattening the annotation layer.
