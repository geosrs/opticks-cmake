QT Cluster

The QT Cluster algorithm takes Quality Threshold Cluster points generated externally to Opticks and passed as an AOI and identifies points in a cluster.

  1. Go to the General Algorithms Toolbar and select QT Cluster.

The QT Cluster dialog box is displayed.

  1. The Cluster radius field identifies how large the radius around the cluster will be. Note that, in general, the smaller the radius, the more results you will probably get.
  2. The Result layer name is the name of the layer that's created.
  3. The Result type identifies how the results are displayed.
  4. Click OK to run the algorithm.
