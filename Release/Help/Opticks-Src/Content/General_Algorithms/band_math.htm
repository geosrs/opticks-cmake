Band Math

The Band Math plug-in uses mathematical expressions to compute a single band or, using multiple cubes, a full cube result. New data is created in memory or on click following the processing location of the source data. Corner coordinates are copied to new cubes.

To access the Band Math plug-in:

  1. On the General Algorithms Toolbar, select Band Math.

The Band Math dialog box is displayed.

  1. Select Single Cube or Multi Cube.

When you select Single Cube, the box on the right side of the Band Math dialog box lists the selected cube.

When you select Multi Cube, the list has all open cubes with the same number of rows, columns, and bands as the selected cube.

  1. Build an expression with the operators, numbers, and bands/cubes.

For each pixel in each row and column (and band in multi-cube), the expression will be evaluated using the corresponding pixel for any band or cube number in the expression.

For definitions of each operator button, click here.

  1. Select Degrees or Radians. This determines whether the triginometric operators are performed with degrees or radians.
  2. If single cube, you can check the Overlay Results box or leave it unchecked.

If checked, Band Math places the results in a results matrix as a layer on the original cube.

If unchecked, the results are placed in a new cube with a singe band.

  1. Click the Process button to begin processing.

To clear the entire expression, click the Clear button.

To undo the last action in the expression, click the Undo button.

To cancel all actions and close the dialog box, click the Cancel button.
