Export Product Data

To export product data as a picture:

  1. In the Session Explorer window, select the product name.

Right-click and select Export.

OR

Right-click in the product window and select Export.

The Export dialog box is displayed.

  1. Select or enter the file name. If necessary, select the File type and the Exporter. For more information on the available exporters, see Product Data Exporters.

To view or edit the export parameters, click the Options button.

Available Export Options

AVI Exporter Options

GeoTIFF View Exporter Options

JPEG View Exporter Options

MPEG-1 Exporter Options

Bitmap View Exporter Options

PNG View Exporter Options

  1. Click the Save button.
