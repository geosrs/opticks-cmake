Import Shape File

A shape file is a group of files used to exchange GISGeographic Information System features between application. GIS features are commonly used to represent roads, cities, rivers, lakes, and other geographic information.

The Shape File Importer enables ESRI shape files to be loaded into the current data set.

Opticks imports all shape files as annotations. Opticks can only import shape files that use the WGS84 coordinate system. If the imported shape file uses another coordinate system, Opticks can attempt to import the file but will treat the coordinates as WGS84. However, unless the coordinate system used is very close to WGS84, the import results will probably be unsatisfactory.

Opticks can import the following shape file types:

  • Point
  • Point with Z component
  • Multipoint
  • Multipoint with Z component
  • Polyline
  • Polyline with Z component
  • Polygon
  • Polygon with Z component

If you import shapes with Z components, the Z component is ignored.

If you import point and multipoint shape files, the points are drawn as small empty triangles.

Before You Begin

Before you import a shape file, you must import a data file and georeference it. Only shapes within the data set are imported.

Import a Shape File

  1. Go to the File menu and select Import Data....
  2. In the Import dialog box, navigate to the shape file you want to import.
  3. If desired, click the Options button on the Import dialog box.
  4. Go to the Auto Importer or Shape File Importer tab. Select the Connection tab. The Layer name is the name of the layer that the shape file will be imported into. This layer will be created if necessary. You can edit this field.

  1. Go to the Display tab.

The Display area enables you to customize the display

In the Fields area, the Displayed fieldshows the attributes for a sample shape within the file. If you select a row or rows in the table, the shapes will be labeled with the selected attributes.

The File info area displays general file information. This information cannot be changed.

  1. Go to the Clipping tab.

This tab enables you to specify how the shape will be clipped.

  1. To finish the import, click the OK button.
  2. On the Import dialog, click the Open button.

The shape file is now imported into Opticks.

Import from an ArcSDE Server

You can also load shape data from an ArcSDE Server.

  1. Be sure the Geographic Features toolbar is displayed.
  2. Click the Geographic Features button and select Import geodatabase.
  3. In the import dialog, select shape file on the Connection tab.
  4. See Steps 4 through 6 about for other import information.
  5. Click the OK button.
