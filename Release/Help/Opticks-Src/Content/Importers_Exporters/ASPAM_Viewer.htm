ASPAM Viewer

Opticks provides an ASPAM viewer tool that enables you to view the ASPAM and PAR data after you import it.

To open the ASPAM Viewer:

Be sure the ASPAM Toolbar is displayed. On the ASPAM Toolbar, select ASPAM and select View ASPAM.

The ASPAM Viewer tool is displayed.

Working with the ASPAM Viewer Tool

Paragraph Tabs

Each paragraph is displayed in a different tab. All fields are read-only. The tab is only available if the paragraph is loaded for the current data file. Paragraphs C, E, and K are not currently supported.

Loaded Data Files

This is a list of the currently loaded ASPAM and PAR data files. The currently selected file is displayed in the dialog box. To view a different data file, use the drop-down list to select the file.

Unload Button

Use the Unload button to remove the data file from memory.

Close Button

Use the Close button to close the ASPAM Viewer Tool.

Plotting ASPAM Data

Paragraphs D, F, H, and J contain tabular data that can be plotted. To create a plot:

  1. In the ASPAM Viewer, select the tab for the paragraph you want to plot.
  2. Click the column header of the first column to plot. Hold down the Ctrl key on your keyboard and click any additional columns to plot. The Plot button is now available. Click the Plot button.

The ASPAM Plot Parameter Selection dialog box is displayed.

  1. In the Choose the primary axis box, use the drop-down list to select the column that will be used as the primary axis. Additional columns will be plotted as separate curves on the same plot.
  2. In the Add to plot box, select an existing plot to add the curves to the plot or select New Plot to create a new plot for the curves.
  3. Click the Ok button.

The new plot data is now displayed in the ASPAM window.
