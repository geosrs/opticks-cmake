ENVI Library Importer

The ENVI Library Importer is a signature importer, not a data importer. It is similar to the ENVI Importer but loads one or more signatures (as part of a spectral library) instead of a data set. It recognizes *.sli and *.spl files.

The ENVI Library Importer is available when you load signatures into the spectrum plot as part of the Signature Selector dialog box.

The ENVI header file for the signatures describes the signature data format in the corresponding signature data file.
