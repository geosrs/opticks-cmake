NITF Importer

The NITF Importer enables you to load data in NITF format.

To import data in NITF format:

  1. On the File menu, select Import Data....
  2. On the Import window, go to the Importer field and select NITF Importer from the drop-down list. Select the name of the file you want to import.

  1. The options for importing NITF data are similar to other import files. However, the metadata is generally more extensive for NITF files. It also contains NITF-specific information.

To view and update the metadata, click the Options... button.

In the Import Options window, select the Metadata tab. A NITF file can have multiple images. Each image has its own metadata.
