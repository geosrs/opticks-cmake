Export Shape File

To export a shape file:

  1. Go to the Session Explorer window.
  2. On the Windows tab, right-click on the AOI layer. Select Export.

  1. The Export window is displayed.

In the Exporter field, select Shape File Exporter.

  1. Navigate to the folder where you want to save the shape file. You can save using the default name or enter a new one.
  2. Optional. Click the Options button to select the export options. See Shape File Exporter Options.
  3. Click the Save button.
