ENVI Importer

The ENVI Importer allows you to load spectral data that has an associated header file in ENvironment for Visualizing Images (ENVI) Native format.

The ENVI header file contains information used to read an image data file. The header file is usually created the first time a data file is accessed by ENVI. An ENVI header can also be created using a text editor. For more information about the ENVI header, see ENVI Header Format.

To open a file using the ENVI Importer:

  1. On the Opticks Main Application window, go to the File menu and select Open....

OR

On the Standard Toolbar, click the Open a File button.

OR

On your keyboard, press Ctrl+O.

  1. The Open file dialog box is displayed. In the Importer drop-down list, select ENVI Importer. The File type defaults to ENVI Header Files (*.hdr) when you select the ENVI Importer.

  1. Navigate to the file you want to open.
  1. Click the Open button.
