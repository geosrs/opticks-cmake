Import ASPAM File

The ASPAMAtmospheric Slant Path Analysis Model importer loads ASPAM and PAR files that follow the official ASPAM and PAR specifications as well as files that deviate from the standard in certain well-defined ways.

To use the ASPAM importer:

  1. On the Opticks menu bar, go to the File menu and select Import Data....
  2. On the Import window, go to the Importer field. Select ASPAM Importer from the drop-down list.

  1. Navigate to and select the ASPAM file (.wxdata) you want to import.
  2. Click the Open button.

Each ASPAM file contains a number of "paragraphs" or sections. The import is a success if at least one paragraph is successfully loaded. The import fails if no paragraphs are successfully loaded. Note that paragraphs C, E, and K are not currently supported.

Information about the import is available in the message log.
