Product Data Exporters

The following exporters are available when you export product data:

  • AVI Product Exporter. Allows you to export animation. For export options, see AVI Exporter Options.
  • Bitmap View Exporter. Allows you to export a two-dimensional 24-bit image of the view window in bitmap format (.bmp). For product windows, this means the paper and any background are saved in the image. For export options, see Bitmap View Exporter Options.
  • GeoTIFF View Exporter. Allows you to export a two-dimensional image of the view window. For product windows, this means the paper and any background are saved in the image. For export options, see GeoTIFF View Exporter Options.
  • JPEG View Exporter. Allows you to export a two-dimensional 24-bit image of the view window in JPEGJoint Photographic Experts Group format (.jpeg). For product windows, this means the paper and any background are saved in the image. Compression options are available. For export options, see JPEG View Exporter Options.
  • KML Exporter. Allows you to export products to a format that is recognized by Google Earth. This includes raster-type data and annotation layers. The view is a product type view and must be georeferenced.
  • MPEG-1 Product Exporter. For export options, see MPEG-1 Exporter Options.
  • PNG View Exporter. Allows you to export a two-dimensional image of the view window in PNGPortable Network Graphic format (.PNG). For product windows, this means the paper and any background are saved in the image. For export options, see PNG View Exporter Options.
  • PostScript Exporter. For export options, there are no options available for this exporter.
