JPEG View Exporter Options

The JPEG View Exporter Options enable you to select the image size and compression for exporting a JPEG.

The default setting for JPEG compression is 75. This provides a balance between the quality of the image and the compression ratio.

  1. In the Image Size section, enter the Width and Height of the exported image.
  2. To lock the aspect ratio, click the Lock Aspect Ratio button. This button is a toggle.
  3. Drag the Quality slider to adjust the compression ratio. A higher number indicates higher quality and lower compression. A lower number indicates higher compression and lower quality.

JPEGs use lossy compressionLossy compression reduces the size of a file by permanently eliminating some information, especially redundant information. When the file is uncompressed, only part of the original data remains, although you may not notice the difference. ; data may be lost if the quality value is too low.

JPEGs saved with a quality value of 100 generally don't look much different from JPEGs saved at 75. The difference between 50 and 75, however, is much more significant.

  1. Click the OK button.
