Export Results

The results exporter exports algorithm results. This is separate from exporting a data set. The results can be appended to an existing results file or can replace the contents of an existing results file. You can also create a new file.

To use the results exporter, you must have algorithm results to save. The exported results file (*.rls) is an ASCII text file listing pixels and their results values that pass a user-defined threshold.

To export results:

  1. On the Session Explorer window, right-click on the name of the results element on the Elements tab.
  2. Select the Export command on the context menu.
  3. In the Export window, go to Exporter and select Results Exporter from the drop-down list.
  4. Enter the output File name to navigate to the directory where you want to save the file. You can enter a new file name or select an existing one.
  5. Click the Options... button. The Export Options window is displayed.

Subset Tab

On the Subset tab, you can specify a subset of the results by specifying rows, columns, and bands.

Results Exporter Tab

On the Results Exporter tab, you can specify the results you want to export.

  1. Using the Pass Area and First Threshold fields, select the threshold for pixel information written to the file. The options are:

  • Less Than. All pixels with a results value less than the threshold value are exported.
  • Greater Than. All pixels with a results value greater than the threshold value are exported.
  • Between. All pixels with a results value greater than the first threshold and less than the second threshold are exported. When you select Between from the drop-down list, the Second Threshold field is enabled.
  • Outside. All pixels with a results value less than the first threshold and greater than the second threshold are exported. When you select Outside from the drop-down list, the Second Threshold field is enabled.

  1. If the data set has been georeferenced, you can select a Coordinate Type for the pixel location (latitude/longitude, UTM, or MGRS). If the data set is not georeferenced the pixel row and column coordinate is exported instead.
  2. Check the Save Metadata checkbox to save any metadata associated with the results.
  3. Check the Append to File checkbox to append the results to an existing file.
  4. Click OK.
