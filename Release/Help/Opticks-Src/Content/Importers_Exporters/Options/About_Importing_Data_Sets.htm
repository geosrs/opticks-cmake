About Importing Data Sets

Selecting the Data Sets to Load

A check box appears next to each file and each data set. You can select and de-select which data sets you want to load.

To select all check boxes for all files and data sets, click the Import All button.

To de-select all check boxes for all files and data sets, click the Import None button.

Data Sets That Can and Can't Be Loaded

A green check mark appears next to any data set that can be loaded by the selected importer.

A red X appears next to any data set that cannot be loaded by the selected importer.
