ENVI Exporter Options

The ENVI Exporter creates an ENVI header file for an existing data set, or a new data file with a corresponding ENVI header file.

The ENVI Exporter is available when you export a data set. The data set window must be active. It is not available when a product window is active.

The ENVI Exporter Options window has two tabs. The Subset Tab enables you to select the parameters for exporting a subset. The ENVI Exporter Tab enables you to select whether to export both a data set and a header file or just a header file.

After you make your export selections, click the OK button.

Subset Tab

On the Subset Tab, you can specify a subset that you want to export.

ENVI Exporter Tab

There are two options for the type of ENVI export:

  • A data set and corresponding header file. This is the default option.
  • A header file only. Note that if you select the header file only option, any edits to export a subset are ignored.
