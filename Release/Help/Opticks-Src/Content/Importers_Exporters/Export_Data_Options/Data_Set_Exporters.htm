Data Set Exporters

The following exporters are available when you export a data set:

  • ENVI Exporter. Allows you to save an ENVIENvironment for Visualizing Images header (.hdr) for the current data set, or a new data set file with a corresponding ENVI header file. For information about the export options, see ENVI Exporter Options.
  • GeoTIFF Exporter. Allows you to export the data set in GeoTIFF format with any geocoordinates. (The same format is loaded with the GeoTIFF Importer.) For information about the export options, see GeoTIFF Exporter Options.
  • Ice Exporter. Allows you to export the data set and some metadata in the Opticks standard format (.ice.h5) with definable cube dimensions. This is a format used to natively store a data set. For information about the export options, see Ice Exporter Options.
  • NITF Exporter. Allows you to export the data set in NITF format. For information about the export options, see NITF Exporter Options.
  • Wavelength Text Exporter. Allows you to save wavelengths in a simple ASCII text file format with no associated data (e.g., units).
  • Wavelength Metadata Exporter. Allows you to save wavelengths in an XML file format with associated data (e.g., units).
