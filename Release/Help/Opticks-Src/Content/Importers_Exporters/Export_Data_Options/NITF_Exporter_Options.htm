NITF Exporter Options

The NITF Exporter Options window has two tabs. The Subset Tab enables you to select the parameters for exporting a subset. The Classification Tab enables you to set classification options.

After you make your export selections, click the OK button.

Subset Tab

On the Subset Tab, you can specify a subset that you want to export.

Classification Tab

The NITF Export Classification tab is based on the MIL-STD-2500C classfication standards. The classification information is required if you are exporting a classified file on a classified system. Note that these fields include text tips if you need additional information.
