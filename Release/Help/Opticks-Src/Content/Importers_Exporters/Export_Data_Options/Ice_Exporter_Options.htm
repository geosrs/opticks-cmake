Ice Exporter Options

On the Ice Exporter Options window, you can specify a subset that you want to export and the compression.

Subset Tab

Ice Exporter Tab

Use the Ice Exporter Tab to set the compression and chunk size options.

Compression Options

The compression options determine if and how an image is compressed. Compressing an image does not remove data. The options are:

  • None.
  • GZIP. A zipped type of compression.
  • Shuffle+GZIP. Applies the HDF5 shuffle, which rearranges the data, and then compresses it.

The GZIP Compression Level affects the speed of compression/decompression.

Chunk Size Options

The chunk size splits the image into chunks. The chunk size has some impact on the load speed. If the image is chunked, the compression is applied to each chunk.
