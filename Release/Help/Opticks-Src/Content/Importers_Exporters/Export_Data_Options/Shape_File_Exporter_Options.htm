Shape File Exporter Options

The Shape File Exporter Options enable you to refine how you want to save the shape file.

Add Feature

To add a feature:

  1. On the Shape File dialog box, click the Add Feature button.

The Add Feature dialog box is displayed.

  1. Highlight the feature you want to add.
  2. Click OK.

Remove Feature

To remove a feature:

  1. On the Shape File dialog box, highlight the feature you want to remove in the Features list.
  2. Click the Remove Feature button. The feature is removed from the list.

Clear Features

To remove all features from the Features list:

On the Shape File dialog box, click the Clear Features button.

Add Field

Fields are added to all features. When you add a field, a new column appears in the Features list in the Shape File dialog box.

To add a field:

  1. On the Shape File dialog box, click the Add Field button.

The Add Field dialog box is displayed.

  1. Enter the field name in the Name field. Note that field names cannot be longer than 11 characters. They must be alphanumeric. You can include underscores but not spaces.
  2. Select the field type from the Type drop-down list.
  3. Click OK.

Remove Field

To remove a field:

  1. On the Shape File dialog box, click the Remove Field button.

The Remove Field dialog box is displayed.

  1. Select field name you want to remove from the Field Name drop-down list.
  2. Click OK.
