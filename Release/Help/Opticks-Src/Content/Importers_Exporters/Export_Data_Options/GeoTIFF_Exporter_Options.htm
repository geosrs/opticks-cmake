GeoTIFF Exporter Options

The GeoTIFF Exporter Options window has two tabs. The Subset Tab enables you to select the parameters for exporting a subset. The GeoTIFF Exporter Tab enables you to set the compression options.

After you make your export selects, click the OK button.

Subset Tab

On the Subset Tab, you can specify a subset that you want to export.

GeoTIFF Exporter Tab

Use the GeoTIFF Exporter Tab to set the compression options.

  1. In the Rows Per Strip field, enter the number of rows per strip.
  2. Check the Pack Bits checkbox to turn the compression on.
  3. Check the Save Settings checkbox to save the compression options.
