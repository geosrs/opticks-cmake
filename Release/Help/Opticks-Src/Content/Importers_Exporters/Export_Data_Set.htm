Export Data Set

To export a data set:

  1. Go to the Session Explorer window and select the element to export.
  2. Right-click on the element name and select Export.
  1. The Export dialog box is displayed.

Select or enter the file name. If necessary, select the File type and the Exporter. For more information on the available exporters, see Data Set Exporters.

To set the export options, click the Options button.

Available Export Options

ENVI Exporter Options

GeoTIFF Exporter Options

Ice Exporter Options

NITF Exporter Options

  1. Click the Save button.
