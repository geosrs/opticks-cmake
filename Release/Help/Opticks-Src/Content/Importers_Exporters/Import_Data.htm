Import Data

To import a data file in Opticks:

  1. On the Opticks Main Application window, go to the File menu and select Import Data....

OR

On the Standard Toolbar, click the Import Data button.

OR

On your keyboard, press the keyboard shortcut for Import Data. The default is Ctrl+O but you can change it.

  1. The Import dialog box is displayed. Navigate to and select the file you want to import.

To select multiple files, hold down the Shift or Ctrl key on your keyboard and select the files you want to import.

  1. If desired, select the file importer from the Importer drop-down list. For more information on importers, see Importers.

If desired, click the Options button to view and edit the loading parameters. See Import Data Options.

  1. Click the Open button.

To open a recently imported file:

Go to the File menu and select from the list of the most recently imported files.

If you hold down the Shift key on your keyboard when you select a file from the list of recently imported files, the Import Data Options dialog box is displayed for the file.
