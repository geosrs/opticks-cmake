Message Log Window

The Message Log Window displays information about actions performed in Opticks. These actions include: importing and exporting data, running algorithms and other plug-ins, manipulating AOIs, GCPs, and data set metadata, and executing wizards.

Actions that perform multiple steps (such as the steps defined in a wizard) are logged as Steps that may contain sub-steps. Errors, warnings, and general informational messages are logged as Messages that do not contain sub-steps or sub-messages.

The Message Log Window displays a message ID, the type of message (Message or Step), a brief summary of the message or step, the result (success or failure) for steps, and the date and time the message was generated.

To view sub-steps, click the plus sign (+) to the left of the message ID.

To open the Message Log Window:

On the Opticks Main Application window, go to the Tools menu and select Message Log Window.

OR

On the Toolbox Toolbar, click the Message Log button.

OR

Go to the Session Explorer window. Navigate to the Message Log Window and select it. Right-click and select Show.

To close the Message Log Window:

Click the Close button in the upper left corner of the Message Log Window.

OR

Click the Message Log button on the Toolbox toolbar.

OR

Go to the Tools menu and select Message Log Window.
