Background Plug-In Window

The Background Plug-In window provides information about plug-ins running as background processes and allows you to stop the plug-in and to dismiss its progress items.

This window contains a list box. When a background plug-in starts, a progress object appears in the list box. This object is similar to the progress dialog box. It contains the name of the plug-in, a status message, and a progress bar that indicates how much work the plug-in has completed. When the plug-in completes, the status item remains in the list but is grayed-out.

To open the Background Plug-In Window:

Go to the Tools menu and select Background PlugIns Window.

OR

Right-click in the toolbar area of the Opticks Main Application window and select Background Plug-In Window from the drop-down list.

To stop a plug-in:

  1. Select the process item(s) in the Background Plug-In Window. If any of these items points to a running plug-in, the Abort button is enabled.
  2. Click the Abort button to stop the selected plug-ins.

To dismiss selected, completed progress items:

  1. Select the process item(s) in the Background Plug-In Window. If any of these items has completed or been aborted, the Dismiss button is enabled.
  2. Click the Dismiss button to dismiss the process items.
