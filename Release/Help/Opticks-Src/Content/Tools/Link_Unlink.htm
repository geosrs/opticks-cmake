Link/Unlink

The Link/Unlink option allows you to link aspects of two data sets. An example of this is looking at the same view taken at two different times. Link/Unlink allow you to link display aspects such as pan, zoom, and rotate. It also allows you to duplicate the annotation, AOI, and GCP layers.

To link data windows:

  1. On the Opticks Main Application window, go to the Tools menu and select Link/Unlink....

OR

On the Toolbox Toolbar, click the Link/Unlink button.

  1. The first Link/Unlink dialog box is displayed. Select the source data set from the list of available data sets. Click Next>.

  1. Select the Layers checkbox to duplicate the annotation, AOI, or GCP layer. Each layer is listed separately under the Layers item. You must check the individual layer checkbox for each layer you want to copy.

Select the View checkbox to link the displays so you can pan, zoom, and rotate both displays.

Select one of the link types:

  • Automatic
  • Mirror
  • Latitude/Longitude enables you to link the images by latitude/longitude so that you can pan by lat/long instead of by pixels.
  • Unlink

Select the Two-Way checkbox to link the source view to the destination view and the destination view to the source view at the same time.

Click Next>.

  1. In the third dialog box, select the destination data set from the list of available data sets. Click Finish.

  1. An information box displays the message that the zoom and pan limits are disabled when you link views. To continue linking the views, click the OK button.
