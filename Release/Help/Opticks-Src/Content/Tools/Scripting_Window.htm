Scripting Window

The scripting window is similar to a DOS prompt window and provides an interface that allows you to enter specific commands to perform interactive processing. Each command line contains a prompt with the name of the current interpretive language. A plug-in is required to recognize the scripting language.

After you enter a command and press Enter, the process is performed and any results are added to the scripting window.

To open the scripting window:

On the Opticks Main Application Window, go to the Tools menu and select Scripting Window.

OR

On the Toolbox Toolbar, select the Scripting Window button.

OR

Go to the Session Explorer window. Expand Dock Windows. Select Scripting Window. Right-click and select Show.
