Image Chipping Window

The Image Chipping Window allows you to create a chip, or subset, of the current image. After you select the chip image, you can create a new window with the chip image or you can export it to a file.

The Image Chipping Window has zoom and pan capabilities.

If you create an image chip without first georeferencing the original data set, the new window or file will not contain the Corner Coordinates GCP list. A warning message is displayed.

To create an Image Chipping Window:

  1. Be sure a spatial data window is currently active.
  2. On the Opticks Main Application window, go to the Tools menu and select Image Chipping Window.

OR

On the Toolbox Toolbar, click the Image Chipping Window button.

The Create Image Chip dialog box is displayed.

  1. Use your mouse to draw a rectangle to indicate the exact area of the chip. The Pixel field displays the cursor location (in pixel coordinates).

  1. The default Chip Size is “Custom” and is based on the area you selected. You can also select a chip size from the drop-down list. The sizes indicate the number of rows and columns in the chip.

When you hover your mouse in this area, a tool tip is displayed showing the start row, the stop row, the start column, stop column, and the row and column skip factors.

You can also select the specific rows, columns, and bands you want to use for the subset chip. Click the Advanced... button.

In the Select Image Subset dialog box, you can:
  • Select the Start Row, End Row, and Skip Factor.
  • Select the Start Column, End Column, and Skip Factor.
  • Select the Bands and customize the selection by clicking the Custom Selection... button. This button opens the Band Selection Options dialog box.

Select the Subset or Bad Bands File radio button to customize the subset selection.

If you select Subset, select the Start Band and End Band from the drop-down lists. Type or use the arrow buttons to select the Skip Factor.

If you select the Bad Bands File, browse to and select the file you want to use.
  1. Select the Chipping Mode. The default is “Create new window.” “Export to file” allows you export the chip selection to a file.
  2. When you select a custom chip area with the mouse or a default chip size, the view is zoomed in to increase the displayed area of the chip.

The Zoom to Fit button returns the view to the full extents in case you want to select a different area that is outside the current view area.

When the view is zoomed in on a chip area, you can also pan the chip around the scene by holding down the Shift key and clicking and dragging the left mouse button..

You can also zoom in and out with the mouse wheel. The view tries to keep the chip area in the center of the view. When you pan or zoom, the chip area may become off-center. Use the Center View button to return the chip area to the center of the view. It may also modify the zoom to ensure that the entire chip area is displayed.

  1. Click OK. If you chose “Export to file,” an Export dialog box is displayed. Enter the location and file name of the export file in this dialog box.
