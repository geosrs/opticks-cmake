Save the Histogram Window Data

The Histogram Window data is saved with the extension .hst. The data is in two columns of ASCII text. The first column is the bin center and the second column is the count for that bin.

The window is saved by band (gray, red, green, or blue) or by layer (threshold, layer).

To save the Histogram Window:

  1. With your mouse over the Histogram Window, right-click to display the plot menu. Select Save....

OR

Go to the Session Explorer window. Expand the Dock Windows tree. Go to the Histogram Window. Right-click on the plot you want to save. The Histogram menu is displayed. Select Save.... 

OR

On your keyboard, press the keyboard shortcut for Save. The default is Ctrl+S but you can change it.

  1. The Save As dialog box is displayed. Select the name and location of the file and click the Save button.
