Reset Stretch

When you adjust the stretch on the histogram window, you can reset the stretch to the original.

Reset single channel or all channels using menu:

  1. Right-click on the histogram window.
  2. Select Reset Stretch.
  3. Select This Channel to reset only the currently selected channel.

Select All Channels to reset the stretch on all channels.

Reset all visible layers using reset button 

To reset all visible layers, click on the Reset button on the Display Toolbar.

Note: If you have run an algorithm that changed the data, the reset results may be slightly different from the original.
