Print the Histogram Window

If you're printing from the Histogram Window, be sure the plot you want to print is the active plot. To make the plot active, click in the plot tab.

If you're printing from the Session Explorer window, the plot you print does not have to be the active plot.

To print the Histogram Window:

  1. With your mouse over the Histogram Window, right click to display the Histogram menu. Select Print....

OR

Go to the Session Explorer window. Expand the Dock Windows tree. Go to the Histogram Window. Right-click on the plot you want to print. The Histogram menu is displayed. Select Print....

OR

On your keyboard, press the keyboard shortcut for Print. The default is Ctrl+P but you can change it.

  1. The Print dialog box is displayed. Select the printer and options and click the Print button.
