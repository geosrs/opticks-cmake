Band Bad Values

Band Bad Values allows you to see the bad values currently set for the selected band on the Histogram plot and to change the values.

  1. Go to the Session Explorer window. Expand the Dock Windows tree. Go to the Histogram Window. Right-click on the layer you want to use (it does not have to be the active layer). The Histogram menu is displayed. Select Bad Values....

OR

With your mouse over the Histogram Window, right-click to display the Histogram menu. Select Bad Values....

  1. The Band Bad Values dialog box is displayed. Enter the values in the Bad Values field. For a list of bad values, separate the entries with commas. Example: 1,2,5.

  1. To apply the bad values to all bands, check the “Set Bad Values for all Bands in Layer” box. This means that if you set the bad values for the red band, the same bad values will be set for the blue and green bands as well.
  2. Click OK.
