Display Gridlines on a Plot

Follow these steps to toggle the gridline display on a Histogram Plot.

To toggle the gridline display:

  1. Go to the Session Explorer window. Expand the Dock Windows tree. Go to the Histogram Window. Right-click on the layer you want to use (it does not have to be the active layer). The Histogram menu is displayed. Select Gridlines.

OR

With your mouse over the Histogram Window display, right-click to display the selection menu. Select Gridlines.

  1. From the Gridlines sub-menu select the gridlines you want to display or remove from the plot. The options are: Major Horizontal, Major Vertical, Minor Horizontal, and Minor Vertical.

Turning off the major gridlines automatically turns off the minor gridlines as well.

Note that the gridline display applies to the current plot only.

To change the color, style, and width of the gridlines, see Histogram Window Properties.
