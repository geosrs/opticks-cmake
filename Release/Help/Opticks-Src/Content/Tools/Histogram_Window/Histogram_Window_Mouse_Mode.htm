Histogram Window Mouse Mode

There are two ways to access the histogram window mouse mode functions. You can access each function directly from the Histogram menu or you can access them from the Mouse Mode Toolbar on the Histogram plot. The mouse modes are

  • Pan. Allows you to pan around the plot.
  • Zoom. Allows you to select and zoom in on a rectangular area over the histogram.
  • Adjust Stretch/Threshold. This mouse mode is called "Adjust Stretch" for raster layer plots and "Adjust Threshold" for threshold layer plots. For histograms of raster data, this mouse mode enables you to graphically edit the contrast stretch or display threshold of the data. This is the default.
  • Annotation. Displays an Annotation Toolbar in the plot window. Use this toolbar to add any annotation object to the plot.

To access from the Histogram menu:

  1. Go to the Session Explorer window. Expand the Dock Windows tree.
  2. Go to the Histogram Window. Right-click on the layer you want to use (it does not have to be the active plot). The Histogram menu is displayed. Select Mouse Mode.

OR

Be sure your mouse is over the Histogram Window and right-click. The Histogram menu is displayed. Select Mouse Mode.

OR

Display the Mouse Mode Toolbar in the plot and click on the button of the mouse mode you want to use.

To access from the Mouse Mode Toolbar:

  1. Go to the Session Explorer window. Expand the Dock Windows tree.
  2. Go to the Histogram Window. Right-click on the layer you want to use (it does not have to be the active plot). The Histogram menu is displayed. Select Toolbars, Mouse Mode.

The Mouse Mode Toolbar is now displayed on the Histogram plot.
