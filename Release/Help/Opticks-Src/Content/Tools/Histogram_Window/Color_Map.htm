Color Map

Color Map allows you to change the shading color of the current raster layer by loading or creating a color map file. It works in grayscale mode only.

To load a color map:

  1. On the Session Explorer window, navigate to the Histogram Window. Select the grayscale layer. Right-click to display the Plot menu.

OR

On the Histogram Window, right-click to display the Plot menu.

  1. Select Color Map.
  2. In the drop-down menu, select Load Color Map... or select one of the pre-loaded Color Maps.
  3. The Select ColorMap File dialog box is displayed. Navigate to the .clu file you want to load and click the Open button. The active raster layer will update its display to match the selected color map. Its corresponding contrast stretch region in the Histogram plot will also update to match.

To save a color map:

  1. On the Session Explorer window, navigate to the Histogram Window. Select the grayscale layer. Right-click to display the Plot menu. 

OR

On the Histogram Window, right-click to display the Plot menu.

  1. Select Color Map.
  2. In the drop-down menu, select Save Color Map....
  3. The Save Color Map File dialog box is displayed. Select or type in the .clu file name you want to save and click the Save button.

To create or edit a color map:

See Create/Edit Color Map.
