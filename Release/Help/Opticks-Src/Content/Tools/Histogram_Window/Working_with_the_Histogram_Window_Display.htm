Working with the Histogram Window Display

Opticks provides you with a variety of ways to customize the display on a Histogram Window.

To access these functions:

Be sure your mouse is over the histogram window and right-click. The Plot menu is displayed.

OR

Go to the Session Explorer window. Expand the Dock Windows tree. Go to the Histogram Window. Right-click on the layer you want to use (it does not have to be the active layer). The Histogram menu is displayed.

This table describes the Plot menu options including the selections that allow you to manage the Histogram Window display.

Menu Item

What it does

See also

Save...

Saves the histogram data.

Save the Histogram Window Data

Print...

Prints the histogram window.

Print the Histogram Window

Axis Scale Enables you to switch between a linear scale and a logarithmic scale for one or both axes. You can also link the axis scale to the contrast stretch type.  

Gridlines

Enables you to toggle the gridlines on the display.

Display Gridlines on a Plot

Mouse Mode

Defines the actions when you move or click the mouse.

Histogram Window Mouse Mode

Toolbars Toggles the display of the Annotation and Mouse Mode Toolbars on the plot. Annotation Toolbar and Histogram Window Mouse Mode
Legend Toggles the display of the legend on the histogram window.  
Statistics Toggles the display of the statistics on the histogram window. Statistics
Refresh Statistics Refreshes the statistics display on the histogram window.  

Custom Zoom

Enables you to enter customized zoom values.

Custom Zoom

Rescale Axes

Restores the plot axes to the original scale.

 

Bad Values...

Allows you to view and edit the bad values omitted from the display.

Band Bad Values

Set Spatial Sampling... Enables you to set the step size that is used when computing statistics. Set Spatial Sampling
Auto Zoom If you set a custom zoom level and the histogram data changes, the auto zoom adjusts so you can see the entire histogram.  
Synchronize Auto Zoom If the raster layer is in RGB mode, synchronizer auto zoom sets the same auto zoom for each band.  
Color Map Enables you to change the shading color of the current raster layer by loading or creating a color map file. It works in grayscale mode only. Color Map

Stretch Type

Enables you to set the stretch type for the current tab display.

Stretch Type

Stretch Units

The stretch units determines the meaning of the numbers displayed in the “stretch values” field. Stretch units are applied by band.

Stretch Units

Stretch Favorites

For a raster layer histogram, you can save custom contrast stretch configurations. Select the Stretch Favorites menu item and select Add Stretch to Favorites. You can also delete items from favorites. In addition, Opticks saves the following favorites by default:

  • Histogram Equalization
  • Linear 0-100 Percentile
  • Linear 2-98 Percentile
  • Linear 5-95 Percentile
  
Stretch Reset When you adjust the stretch on the histogram window, you can reset the stretch to the original. Reset Stretch
Displayed Element Enables you to select the raster data that is displayed. Different data can be displayed in each of the R, G, and B channels.  
Displayed Band Enables you to select which band is currently displayed. Displayed Band
Copy Name Copies the histogram name to the clipboard. This command appears only when you click on a plot in the Session Explorer.  
Export Enables you to export the selected histogram plot. This option is available from the Session Explorer only.  

Properties

Enables you to modify the width, style, and color of the gridlines. Also allows you to change the display color of the histogram plot. This option is available only from the Session Explorer.

Histogram Window Properties
