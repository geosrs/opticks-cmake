Stretch Type

Each tab of the Histogram Window has a shaded area that reflects the current contrast stretch for the layer or the current threshold. The color of the shaded area matches the color of the layer or the color of the pixel markers for the threshold layers. You can stretch this area by clicking and dragging the edge of the shaded area.

You can also select the stretch type from the Session Explorer window.

  1. On the Histogram Window, select the tab for the layer you want to apply the new stretch type to. With your mouse over the Histogram Window, right-click to display the Histogram menu. Select Stretch Type.

OR

Go to the Session Explorer window. Select the layer you want to apply the stretch to. Right-click. From the Layer menu, select Stretch Type.

  1. From the sub-menu, select the stretch type.
    • Linear. The stretch is linear from the lower to the upper stretch value. The value halfway from the lower to the upper value is shaded at 50% brightness.
    • Exponential. The contrast stretch is exponential in shape from the lower to the upper value. This suppresses the darker (weaker) features in the image and accentuates the brighter ones.
    • Logarithmic. The contrast stretch is exponential in shape from the lower to the upper value. This accentuates the darker (weaker) features in the image.
    • Histogram Equalization. The contrast stretch is an equalized histogram.

The stretch type applies to the currently selected layer only.
