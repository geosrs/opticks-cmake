Set Spatial Sampling

Set Spatial Sampling allows you to set the step size that is used when computing statistics on the Histogram Window. Note that this factor is applied to all bands of the Histogram Window but does not affect other layers with plots in the window.

  1. With your mouse over the Histogram Window, right-click to display the plot menu. Select Set Spatial Sampling....

OR

Go to the Session Explorer window. Expand the Dock Windows tree. Go to the Histogram Window. Right-click on the layer you want to use (it does not have to be the active layer). The Histogram menu is displayed. Select Set Spatial Sampling....

  1. The Statistics Resolution dialog box is displayed. Enter the step size you want to use.

  1. Click OK.
