AOI Statistics

To view statistics on an AOI:

  1. Create an AOI.
  2. Right-click in the image. Select the raster layer and select Calculate Statistics on Subset.

The Select a subset dialog box is displayed.

  1. The Plot name field identifies the name of the plot that will be created on the tab that's created on histogram window. You can change this name.

In the Select spatial subset field, select the spatial subset from the drop-down list. In the Select band subset, select the bands. The statistics are run on this subset.

  1. Click the OK button.

The histogram window is displayed.

The statistics are displayed on a new tab in the histogram window.

Refresh Statistics

If you update an AOI, you can refresh the statistics.

  1. Right-click on the histogram.
  2. From the menu, select Refresh Statistics.
