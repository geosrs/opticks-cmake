Displayed Band

This option allows you to select the band displayed in the Histogram Window.

  1. Go to the Session Explorer window. Expand the Dock Windows tree. Go to the Histogram Window. Right-click on the layer you want to use (it does not have to be the active layer). The Histogram menu is displayed. Select Displayed Band....

OR

Select the tab (gray, red, green, or blue) on the Histogram Window. With your mouse over the Histogram Window, right-click to display the Histogram menu. Select Displayed Band.

  1. Select the band number from the drop-down list.

OR

When the histogram plot has the keyboard focus, press the keyboard shortcut for Next Band or Previous Band to move to the next or previous displayed bands. (See Keyboard Shortcuts.)

The image display and the histogram tab now reflect the newly selected band.
