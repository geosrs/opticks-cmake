Custom Zoom

This option enables you to customize the zoom on the histogram.

  1. Go to the Session Explorer window. Expand the Dock Windows tree. Go to the Histogram Window. Right-click on the layer you want to use (it does not have to be the active layer). The Histogram menu is displayed. Select Custom Zoom.

  1. The Custom Zoom dialog box is displayed. Enter the minimum and maximum values for the X coordinates and the Y coordinates.

  1. Click OK.
