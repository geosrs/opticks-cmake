Histogram Window

The Histogram Window displays the number pixels in the image display that have a specific intensity or brightness. The number of pixels is displayed on the y-axis and the intensity is displayed on the x-axis.

To display the Histogram Window:

On the Opticks Main Application window, go to the Tools menu and select Histogram Window.

OR

On the Toolbox Toolbar, click the Histogram Window button.

Each band has a layer with gray, red, green, and blue component. You can display each component in the Histogram Window by selecting the component tab at the bottom of the window. Raster and threshold layers are also displayed.

The histogram has a shaded area that reflects the current contrast stretch for the layer or the current threshold. The color of the shaded area matches the color of the layer. You can stretch this area by clicking and dragging the edge of the shaded area. The appropriate mouse mode must be active. You can also pan and zoom the plot. See also Stretch Type.

Use the Brightness Toolbar to control the brightness and contrast of the layer.
