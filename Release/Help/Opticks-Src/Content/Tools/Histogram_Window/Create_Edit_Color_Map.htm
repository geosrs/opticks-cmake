Create/Edit Color Map

Opticks provides the ability to create a new color map or edit an existing one.

Note: The image must be in grayscale.

  1. On the Session Explorer window, navigate to the Histogram Window. Select the grayscale layer. Right-click to display the Plot menu. 

OR

On the Histogram Window, right-click to display the Plot menu.

  1. Select Color Map.
  2. In the drop-down menu, select Create/Edit Color Map.... The Colormap Editor dialog box is displayed.

  1. The Control-Colors box enables you select the number of colors used in the color map. You can select up 20 colors.

The Gradient Steps box enables you to select the number of gradient steps used.

For each color, click on the color button (far right) to edit the color used. (See Edit Color for more information on editing colors.)

Use the slide bar next to each color to assign how the colors are distributed. You can also click the Distribute box to distribute the colors evenly across the scale.

See the example below:

  1. To apply the colors to the image and leave the dialog box open, click the Apply button.

To apply the colors and close the dialog box, click the OK button.

To save the color map, click the Save button. The Save Colormap dialog box opens. Navigate to the folder where you want to save the colormap. Enter the colormap's name. Colormaps are saved with the extension .cgr.

To load a different color map, click the Load button.

To close the dialog box, click the Close button.
