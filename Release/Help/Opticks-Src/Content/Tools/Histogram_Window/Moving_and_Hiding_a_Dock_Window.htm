Moving and Hiding a Dock Window

Follow these steps to move or hide the following window types:

  • Histogram Window
  • Message Log Window
  • Scripting Window
  • Background Plug-In Window
  • Session Explorer Window

Move the Dock Window from the Opticks Display

To free the window from the Opticks display and move it anywhere on your monitor display:

  1. Right-click your mouse in the gray area below the window’s display.
  2. The Window Menu is displayed. Select Undock.

OR

  1. Go to the Session Explorer Window and select the window name.
  2. Right-click your mouse.
  3. The Window Menu is displayed. Select Undock.

You can now move the window.

Move the Dock Window Back to the Opticks Display

To move the window back to its previous position in the Opticks display:

  1. Right-click your mouse in the gray area below the window’s display.
  2. The Window Menu is displayed. Select Dock.

OR 

  1. Go to the Session Explorer Window and select the window name.
  2. Right-click your mouse.
  3. The Window Menu is displayed. Select Dock.

The window is now back in its original position on the Opticks display.

Hide the Dock Window

This option allows you to hide the window from view but recover it by selecting the window type from the Tool menu or the Toolbox Toolbar.

  1. Right-click your mouse in the gray area below the window’s display.
  2. The Window Menu is displayed. Select Hide.
