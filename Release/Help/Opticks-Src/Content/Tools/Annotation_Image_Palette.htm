Annotation Image Palette

The annotation image palette enables you to easily place image chips in an annotation layer.

To activate the Annotation Image Palette:

Go to Tools and select Annotation Image Palette.

OR

Right click in the toolbar area and select Annotation Image Palette.

The palette is automatically docked at the bottom of the Opticks window.

The Annotation Toolbar is automatically activated. There are several options specific to the Annotation Image Palette.

From the Annotation Image Palette, you can drag and drop an image onto the image you have loaded in Opticks.

To move or resize the image, select the Move Object button on the Annotation Toolbar.

To rotate the image, select the Rotate Object button on the Annotation Toolbar.

You can also merge this annotation layer with the underlying data. See Flatten Annotation Layer.
