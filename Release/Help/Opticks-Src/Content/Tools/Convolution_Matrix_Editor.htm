Convolution Matrix Editor

To activate the Convolution Matrix Editor:

Go to Tools and select Convolution Matrix Editor.

OR

Right click in the toolbar area and select Convolution Matrix Editor.

The editor can be docked in the left panel over the Session Explorer.

Use the vertical and horizontal sliders to display cells to select the size of the matrix.

If the locked button is activated , using one of the sliders moves both sliders the same number of cells.

If the unlocked button is activated , the sliders can be moved independently of one another.

By default, the matrix is divided by the number of squares in the matrix. You can also enter a value in the Divisor field to further divide the value in addition to the default.

Use the Reset button to return the matrix fields to their default.

Use the Apply button to apply the settings to the image.

Note: When convolving an image, the edge extend method used extends the values on the edge of the image.

Save the Settings

To save the settings for future use:

  1. Click in the Presets field.
  2. Enter a name for the settings.
  3. Click the Save button.

Load a Saved Setting

To load a saved setting:

  1. Click the down button in the Presets field to display the list of saved settings.
  2. Select the name of the saved setting from the list.
  3. Click the Load button.

Remove a Saved Setting

To remove a saved setting:

  1. Click the down button in the Presets field to display the list of saved settings.
  2. Select the name of the saved setting from the list.
  3. Click the Remove button.
