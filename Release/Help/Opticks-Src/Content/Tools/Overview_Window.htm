Overview Window

The Overview Window is a small window that displays an overview of the currently loaded data set.

To display the Overview Window:

  1. On the Opticks Main Application window, go to the Tools menu and select Overview Window.

OR

On the Toolbox Toolbar, click the Overview Window button.

  1. A selection rectangle appears in the window depicting the area currently displayed in the spatial data window.

Left-click and drag to select a new display area rectangle.

Shift+left-click with the mouse or use the arrow keys to pan the display area around the view.

The Image Trail

When you pan the image, Opticks leaves a colored "snail trail" of what you've looked at.

You can reset the trail and change the color, opacity, and zoom threshold for this trail. You can also save a snapshot of the Overview Window to the Clipboard.

To reset the trail:

Right-click in the Overview Window and select Clear the Trail.

To change the trail color:

Right-click in the Overview Window and select Change Trail color.... For more information on selecting colors, see Edit Color.

To change the opacity:

  1. Right-click in the Overview Window and select Change Trail opacity....
  2. The Set Overview Trail Opacity dialog box is displayed. The opacity determines how opaque the trail color is on the Overview Window. Zero percent is completely transparent. One hundred percent is completely opaque.

Enter the level you want and click OK.

To change the zoom threshold:

  1. Right-click in the Overview Window and select Change zoom threshold....
  2. The Set Trail Zoom Threshold dialog box is displayed. Opticks won't draw the trail until you reach this threshold.

Enter the zoom threshold percentage and click OK.

To change the options for this trail, see Windows Overview Options.
