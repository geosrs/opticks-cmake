Merge AOIs

The Merge AOIs function allows you to merge two or more AOIs.

To merge AOIs:

  1. On the AOI Toolbar, click the Merge AOIs button.

The AOI Merge dialog box is displayed.

  1. To select the AOIs you want to merge, hold down the Ctrl or Shift key and select the AOIs from the list.
  2. Select the Merge Method (combination or common pixels).

A “combination” or “union” merge contains all pixels in either of the source AOIs.

A “common pixels” or “intersection” merge contains only pixels present in both source AOIs.

  1. For the Output AOI Name, you can select an existing AOI from the drop-down list or you can type in a new name.
  2. Click OK.
