AOI Pixel Selection Tool

The AOI Pixel Selection Tool allows you to select the shape of the AOI.

To use the AOI Pixel Selection tool:

On the AOI Toolbar, select the drop-down list for the AOI Pixel Selection Tool.

Shape

What it does

Selects a single pixel.

Creates a rectangle with grab points at each corner.

Creates a polygon with grab points at the corners. To create the polygon, click the left mouse button to start and to create a new grab point. To end, double-click the left mouse button.

Creates a line with two endpoints.

Creates a horizontal line.

Creates a vertical line.

Creates a polyline with grab points at the corners and a variable number of vertices that have grab points. To create the line, click the left mouse button to start and to create a new grab point. To end, double-click the left mouse button.

Creates a horizontal line the width of the data cube.

Creates a vertical line the height of the data cube.
