AOI

An Area of Interest (AOI) is a group of selected pixels on a data set. It has two main uses:

  • As an input for an algorithm. These pixels are known as pixels of interest (POI). POIs allow you to use information from within a scene as input to an algorithm.
  • As a region of the scene over which to run an algorithm. This area is known as a region of interest (ROI). ROIs allow you to eliminate unnecessary processing by running an algorithm over a portion of a scene. This reduces the total time for processing.

To work with AOIs, you must create at least one AOI layer. To activate an existing layer, see Activate a Layer.

To create an AOI layer:

  1. On the Session Explorer Window, select the name of the image you want to add the AOI layer to. Right-click with your mouse.
  2. Select New Layer. On the sub-menu, select AOI. Opticks creates a new layer and gives it a default name, beginning with "AOI 1," then "AOI 2," and so on. If the AOI Toolbar is not displayed, creating a new layer automatically displays the AOI Toolbar.
  3. On the AOI Toolbar, be sure the AOI Draw Mode button is activated.
  4. Select the pixels. Opticks automatically inserts the label “AOI 1” on the image.

Use the Display Toolbar to zoom in and out on selected pixels.

When creating AOIs, you can use mouse panning mode to move around the image. See Mouse Panning Mode.
