Starting the Quick-Look Data Fusion Tool

To start using the Data Fusion Plug-in you must first load a primary data set and a secondary data set. 

  • You must select two different data sets to run the Data Fusion tool properly.
  • We recommend that you select the higher resolution data set as the Secondary Data set. The algorithm will use the higher resolution to produce a better product when fusing a data set with visual EO data. A warning notice will display if the Primary Data set has a higher resolution than the Secondary Data set.

To start the Plug-In:

  1. Be sure the General Algorithms Toolbar is displayed. Click the General Algorithms button and select Data Fusion to start the Quick-Look Data Fusion Plug-In.
  2. The Quick-Look Data Fusion dialog box is displayed.

  1. Select one data set from the Primary Data Set list and one data set from the Secondary Data Set list.
  1. (Optional) Opticks may provide you with the option of bypassing the step of creating tie points. To bypass this step, go to the Algorithm Options box and select the Bypass Tie Point Step checkbox.

Important: By selecting the Bypass Tie Point Step checkbox, you agree that the georeferencing provided by the two data sets is accurate enough to bypass the Tie Point step.

  1. (Optional) If you want to bypass the tie point step each time you run the Quick Look Data Fusion Plug-in, check the Save Settings for future sessions checkbox. This checkbox applies only to bypassing the tie point step.
  2. Click the Next> button.

If you selected the Bypass Tie Point Step checkbox, go to Copying Layers.

If you did not select the Bypass Tie Point Step checkbox, go to Creating Tie Points.
