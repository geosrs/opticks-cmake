Manipulating Fused Views

After executing the data fusion algorithm, the two image chips are rescaled and reoriented, placed in a side-by-side view and/or a flicker/blend view. The flicker tools enable you to manipulate the flicker/blend view so that you can control the transparency, x/y translation of the secondary image chip, automated/manual flicker, and flicker rate.

Manipulating Flicker/Blend View

If you have not run the flicker tools plug-in from the wizard, perform one of the following actions to view the flicker controls:

  • Select data fusion from the Tools menu and choose Flicker Controls.
  • Right click the menu area and select Flicker Controls.

Note: The flicker controls window automatically docks at the bottom of the screen. It can be moved.

To use the automated flicker feature:

  1. Set desired flicker rate.
  2. Click Start. You can adjust the rate during flicker.
  3. Click Stop to end flicker.
  4. (Optional) The smoothing feature is a default setting. You can disable smoothing by clicking the properties dialog and de-selecting smoothing.

To use the manual flicker feature:

  1. Set the flicker rate to 0 and press the Stop button.
  2. Click the Toggle button.

To use the transparency slide:

  1. Click the button on the transparency slide to modify the transparency from transparent to opaque.

To use the x,y sliders:

  1. Click the button on either the x or y slide to move the secondary image away from the primary image.

Manipulating Side-by-Side View

To use the side-by-side view:

  1. Click one of the two side-by-side images.
  2. Use the following key commands:
  • Z / Shift+Z ==> Zoom In / Zoom Out
  • Shift+Left / Shift+Right ==> Rotate Left / Rotate Right
  • Arrow Keys ==> Pan

Notes:

  • If you change views, flickering will automatically stop and the Flicker Tools window may become disabled. To restart the flicker tools, select the fused flicker view.
  • If you close the flicker tools window the selected image will continue to flicker.
  • The side-by-side view is only available when the Data Fusion wizard is open. If the wizard is closed, the side-by-side view also closes.
