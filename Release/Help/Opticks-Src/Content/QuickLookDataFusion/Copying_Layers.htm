Copying Layers

The Quick-Look Data Fusion Plug-In gives you the option of transferring raster layers and annotation layers (used with Opticks algorithms) to the fused product.

You can select raster layers with or without color maps applied. The raster layers will be clipped as needed to the Region of Interest identified in the next step.

If you select an annotation layer (or layers), all annotation objects on the annotation layer will be copied.

If you don't want to select any layers, simply click the Next> button.

To select the raster and annotation layers:

  1. In the Available Layers for Fusion list, select the layers you want to transfer. Use the Shift or Ctrl key to select multiple layers.

  1. Click the Next button to select a Region of Interest (ROI) and Output Views.
