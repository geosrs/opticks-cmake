Creating Tie Points

When you select the data sets, you have the option of creating tie points or bypassing the tie point step.

A tie point is the same feature on two different images. You can use multiple tie points to correct inconsistencies in the geolocation data. More than one tie point may be used, but only one is necessary.

To create a tie point:

  1. Use the Zoom Rectangle button on the Display Toolbar to navigate to a pixel on the primary image and a pixel on the secondary image. If you know the latitude and longitude you can georeference the scenes and use the coordinate locator to go directly to the coordinate.
  1. Click the center of each pixel. Each pixel will be automatically added to the appropriate list.

Note: You can use the GCP Editor to make changes to the GCP. The Data Fusion wizard will automatically accept the changes.

  1. Select two GCPs, one from each list, and click the Derive button to create a tie point. The GCP markers will disappear.

Note: If you remove a tie point, the GCPs will again be visible on the selected pixels.

  1. (Optional) Select a tie point and the desired zoom level in the Zoom % list box and click the Verify button to verify that the selected tie point is correct. The tool will zoom and pan to the specified tie point. If the cross-hair is visible, the tool will zoom and pan with the points marked by the cross-hair.
  2. Click the Next> button to go to the next step Copying Layers.
