Selecting a ROI and Output Views

After you select raster and annotation layers, you can select a region of interest (ROI) and view a fused version of the ROI using Flicker/Blend or Side-by-Side views.

To select a ROI and output views:

  1. Use your mouse and the Opticks zoom tools to zoom to the desired location in the image.
  2. Go to the Session Explorer window and activate the AOI layer.
  3. Using the tools on the AOI Toolbar, create a ROI on the Primary Data set labeled Fusion Region. The rectangle tool is selected by default when this page is displayed.
  4. Select one or both output Available Views: Flicker/Blend or Side-by-Side.
  5. (Optional) Select the Run Flicker Tools when done check box to run the flicker tools that manipulate the fused flicker view. To access the flicker tools, go to Tools, Data Fusion, Flicker Controls.
  6. (Optional) If you're using the flicker view, you can copy the color maps from the primary and/or the secondary image to their respective chips. Select the Primary Chip check box and/or the Secondary Chip check box. The color map is not copied for the side-by-side view.
  7. (Optional) If you are working on a computer that may not have enough memory to run the algorithm in memory, you can check the Optimize for memory checkbox. The computer will run slower but you will be able to run the algorithm without restarting Opticks or the computer.
  8. Click the Finish button if you want to run the data fusion algorithm and close the wizard as soon as the algorithm is completed. Click the Execute button to run the data fusion algorithm. This gives you the option to run on a different ROI on the same two images without restarting the wizard.
