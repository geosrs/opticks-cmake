Wavelengths Tab Algorithm Information

The wavelength is the distance between adjacent peaks or valleys measured in the direction of propagation in a harmonic wave.

The wavelength and the frequency f are related by

v = f

where v is the phase velocity of the waves.

The Full Width Half Maximum (FWHM) values are needed to calculate the Minimum and Maximum Channel Wavelengths. In most hyperspectral collections, the true FWHM values come from the sensor. If the input data does not contain sensor provided FWHM values, they can be estimated by the Wavelength Editor.

The Wavelength Tab uses the distance between N + 1 and N center wavelengths to approximate the width between channels, where N is equal to the current band center channel. The FWHM constant enables you to provide a scaling factor to manipulate the minimum and maximum channel wavelengths until the proper bandwidth is achieved. If the FWHM Constant is set to 1.0, then the bandwidths are not scaled. The default is 1.0.

The formula for Band N is:

1. Value = ((Band N+1 center wavelength – Band N center wavelength) * FWHM constant) / 2

2. Band N Minimum channel wavelength = Band N center wavelength – Value

3. Band N Maximum channel wavelength = Band N center wavelength + Value
