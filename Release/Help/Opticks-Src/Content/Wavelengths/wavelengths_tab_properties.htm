Wavelengths Tab: Properties Window

The Wavelengths Tab on the Properties Window provides the capability to load or save wavelengths in a file and to assign wavelengths to a loaded data set.

The Wavelengths Tab is only available for raster data sets.

For input, this tool takes a wavelengths file or the wavelengths included in a data cube and writes an output wave file or writes the wavelengths to the current data cube.

The Wavelengths Tab functionality should be executed before running any plug-in that requires wavelength data to run. The Wavelengths Tab also can be used to review the wavelengths of a specific dataset.

If the dataset already contains all required wavelength information (start, end, and center channel wavelengths), the Wavelengths Tab is not needed.

Using the Wavelengths Tab

The Wavelengths Tab appears in the Import Options dialog box and the Data Elements Properties dialog box. It cannot be displayed independently from a raster element.

When you open the Wavelengths Tab, you can open it with a data cube already open. If you open it with the data cube already open, you can also open a wavelength file associated with the data cube and output any changes to the wavelengths directly back to the opened data cube.

To assign wavelengths to a loaded data set:

  1. Go to the Session Explorer and select the Elements tab.
  2. Select the raster element of the desired data set.
  3. Right-click and select Properties. The Wavelengths Tab appears on the properties dialog box. See also Data Elements Properties.

The From Element button allows you to assign the wavelengths of another data set to this data set. The data sets must have the same number of bands.

To input the wavelengths from a wavelength file:

  1. On the Wavelengths tab of the Properties window, click the Load button.
  2. Navigate to and open the wavelength file. Wavelength files have the extension .wav, .wave, or .wmd. The wavelengths are automatically loaded into the wavelengths table.

The Units field displays the loaded wavelengths in various units.

  1. If you choose to multiple the wavelength values by a specified factor, click the Scale button. This activates a text entry box. Type in the desired factor and then select OK. The wavelength values are updated with the new values.

If the collected data does not contain a Full Width Half Max value, one can be calculated based on a specified constant to derive the Minimum and Maximum Channel Wavelengths. To compute the minimum and maximum channel wavelengths, set the FWHM constant by clicking the FWHM button, button, type in a factor in the FWHM Constant text field (the default value is 1.0), and select the OK button. The minimum and maximum channel wavelength columns are filled in with the computed values.

To output the wavelengths to a new wavelength file:

  1. Click the Save... button.
  2. Navigate to the desired directory and either enter a new filename or select a current wavelength file. If you choose to overwrite a current file, a warning dialog is displayed asking if you want to overwrite the file. The new wavelength file will have the extension .wav, .wave, or .wmd based on the selected wavelength exporter.

When you are ready to write the wavelength values to the data set, click the OK button.
