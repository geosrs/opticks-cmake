Derive a Product

Derive Product creates a new product window and loads the default template file specified in Options.

All view objects are initialized to display the active spatial data view.

If no default template file is specified, the new product window is created with a single view object based on the active spatial data view. If a spatial data view is not active, the command is disabled.

To derive a product:

On the Layout Menu, select Derive Product.
