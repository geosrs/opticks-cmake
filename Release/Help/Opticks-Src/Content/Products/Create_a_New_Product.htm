Create a New Product

Create a New Product creates a new product window and automatically loads a default template. The default template is specified in Options.

On the Layout menu, select New Product. A new product window is created using the default template.

You can specify a template file or no template file. If you don’t specify a template, Opticks creates an empty product in the new product window.

To specify a template:

  1. In Opticks, go to the Tools menu and select Options.
  2. In the Options window, go to Session and select File Locations.

  1. Go to the Default Location table (on the right) and double-click the Product Template row.

  1. You can now type in the location of the template or click the browse button (on the far right) to navigate to the template location.

If you use the default template file, view objects created from the template are not initialized to display a particular view.

To work with the product window, be sure the Layout Layer is activated.
