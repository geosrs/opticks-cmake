Edit Selected View

The Edit Selected View command allows you to edit view and layer properties of a single selected spatial data view object in the Product Layout layer. You can zoom and pan the view object, rotate the view, and edit the layer properties. You must have one, and only one, view object selected.

  1. Select the spatial data view object.
  2. On the Layout menu, select Edit Selected View.
