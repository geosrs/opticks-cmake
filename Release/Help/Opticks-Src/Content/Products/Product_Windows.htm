Product Windows

After you’ve completed your analysis of an image, you can save the image as a product.

A product window is not associated with a single data set but allows you to create a product by arranging annotation objects on a virtual sheet of paper.

A product has a Layout layer and a Classification layer.

The Layout layer allows you to add annotation objects to the product. The Annotation Layer must be active.

The Classification layer allows you to edit the properties of the classification markings on the virtual paper.
