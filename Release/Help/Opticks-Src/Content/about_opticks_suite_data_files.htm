About Opticks Suite Data Files

The following information applies to common Opticks formats. It does not include plug-in-specific formats.

On Windows 32-bit, the size limit for image data is about 500 megabytes. Any file larger than 500 megabytes must be processed on disk or might not be able to load at all.

On Solaris and Windows 64-bit, the size limit for image data is over 3.0 gigabytes.

Importers

Opticks Suite can load the following types of image data:

  • ENVI Standard
  • GeoTIFF
  • SIO
  • Ice/HDF5
  • DTED
  • Generic
  • NITF

Exporters

Opticks Suite can export the following types of image data:

  • Bitmap (BMP)
  • GeoTIFF
  • JPEG
  • PNG
  • NITF
  • Ice/HDF5
  • ENVI Standard

Other Standard Input/Output Files

  • CGM (*.cgm) (Read/Write)
  • Shape File (Read/Write)

Opticks-Specific Input/Output Files

  • Wizard (*.wiz) (Read/Write)
  • Batch (*.xml) (Read/Write)
  • Template (*.spg) (Read/Write)
  • Annotation (*.ano) (Read/Write)
  • Ground Control Points (*.gcp) (Read/Write)
  • Tie Points (*.tpl) (Read/Write)
  • AOI (*.aoi) (Read/Write)
  • Session (*.session) (Read/Write)
