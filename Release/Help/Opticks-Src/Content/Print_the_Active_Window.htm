Print the Active Window

When printing, be sure the window you want to print is the active window. To make the window active, click in the window.

To print the current view of the active window:

  1. Go to the File menu and select Print....

OR

Press Ctrl + P.

OR

On the Standard Toolbar, select the Print button.

  1. If you selected Print from the File menu or Ctrl + P, the Print dialog box is displayed. Select the printer and options and click the Print button.

If you selected the Print button on the Standard Toolbar, the data is sent directly to the printer.
