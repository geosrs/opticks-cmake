Location Units

On the Measurement Toolbar, use the Location Units drop-down box to label the measurement endpoints. This effects all the measurements currently in Opticks.

No Unit. Displays pixel coordinates.

Decimal degrees. Displays latitude and longitude using decimal degrees.

Decimal minutes. Displays latitude and longitude using degrees and decimal minutes.

Degrees, minutes, seconds. Displays latitude and longitude using degrees, minutes, and decimal seconds.

Universal Transverse Mercator. Displays using UTM.

Military Grid Reference System. Displays using MGRS.
