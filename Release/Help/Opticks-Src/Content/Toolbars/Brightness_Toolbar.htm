Brightness Toolbar

Button/Slider/Menu

What it does

Controls the brightness of the image. Values range from 0 to 100.

Controls the contrast of the image. Values range from 0 to 100.

Use this drop-down menu to select the layer and/or color component the brightness and contrast values are applied to.

Use this drop-down menu to select the layer that the color component applies to.
Reapplies the default contrast and brightness values.
