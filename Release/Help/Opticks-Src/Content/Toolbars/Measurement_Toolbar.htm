Measurement Toolbar

Use the Measurement Toolbar to measure distances and bearing between two points on a data set. The data set must be georeferenced to use any units other than pixel values.

Button

What it does

 

Move Measurement Object.

(left end of toolbar) Measurement Draw Mode. When selected, creates a new measurement when you click within the data set. When unselected, moves an existing measurement.
Show/Hide Layer. Toggles the display of the Measurement Layer.
Show/Hide Bearing Label. Toggles the display of the bearing label next to the measurement object.
Show/Hide Distance Label. Toggles the display of distance next to the measurement object.

Show/Hide End Point Labels. Toggles the display of the end point labels of the measurement object.

Location Units. Use this drop-down list to label the measurement endpoints. This effects all the measurements currently in Opticks. See Location Units.

Distance Units. Use this drop-down list to select the units used to label distances. This effects all the measurements currently in Opticks. See Distance Units.

(right end of toolbar)

Activates Measurement mouse mode. Only one edit mode can be active at a time.
