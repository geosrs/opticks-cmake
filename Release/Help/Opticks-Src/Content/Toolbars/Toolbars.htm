Toolbars

The toolbars provide access to some of the same functionality as the Opticks drop-down menus. They also provide access to specialized functionality. For example, the Annotation Toolbar functionality provides you with the means to create and modify objects, lines, and text on the image’s annotation layer.

Toolbars can be toggled on and off.

Some toolbar functionality may not be available until you load an image or activate the layer that the toolbar is associated with. The following toolbars can only be accessed when the layer is active: AOI, Annotation, Geo, and Tie Point. (See Activate a Layer for more information.)

Displaying the Toolbars

There are two ways to display the toolbars:

  • Go to the View drop-down menu and select Toolbars. The list of toolbars is displayed. Active toolbars have a checkmark in front of them. Select the toolbar name to toggle it on or off.

OR

  • Using your mouse, move the pointer to the toolbar area of the Opticks Main Application window. Right-click the mouse. The list of windows and toolbars is displayed. Active toolbars have a checkmark in front of them. Select the toolbar name to toggle it on or off.
