Tie Point Toolbar

Be sure the Tie Point layer is active. If the Tie Point Toolbar was not already displayed, it is automatically displayed when you activate the layer.

The Tie Point Toolbar functions apply to the active image only.

Button

What it does

Tie Point Marker Color. Allows you to change the tie point marker color. See Edit Color.

Labels Enabled. This toggles the display of the tie point pixel marker labels. (When you turn the display off, the pixels are still displayed but not labeled.)

Tie Point Editor. Allows you to edit the Tie Point layer. See Tie Point Editor.
