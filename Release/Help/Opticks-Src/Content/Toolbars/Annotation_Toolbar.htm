Annotation Toolbar

The Annotation Toolbar is available if the current image includes an annotation layer and if the annotation layer is active. See Create a New Layer and Activate a Layer.

You can also open an Annotation Toolbar on the Histogram Window. See Histogram Window Mouse Mode.

For more information on using this toolbar, see Working with Annotation Objects and Working with Annotation Text Boxes.
