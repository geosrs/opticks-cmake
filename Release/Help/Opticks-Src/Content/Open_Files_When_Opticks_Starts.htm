Open Files When Opticks Starts

You can open data files when you start the Opticks application. You can open multiple data files.

Open Opticks from a Command Line

When you open Opticks from a command line, you can also enter the file names for the files you want to open with Opticks. If the file is not in the current directory, you must enter the full path name.

Example:

C:\> Opticks.exe cube1.sio cube2.sio c:\data\map3.tiff

Select Files from the File System Explorer

Open your file system explorer and drag the file name (or names) onto the Opticks program execute icon on your desktop. Opticks will open and import the files.

Setting the Properties

When you open data files while starting Opticks, Opticks calls the Auto Sensor Data Importer to open the file. The Import Data Options dialog box opens, allowing you to customize the import parameters.

If the specified file is not a valid data file, click Cancel to move to the next file.
