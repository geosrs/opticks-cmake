Edit Color

Throughout the Opticks application, you can customize the color used in display elements such as text, pixel markers, and shapes.

To select a standard color:

  1. Access the color palette. (It may be displayed as a drop-down or it may be displayed within a dialog box.) The color palette displays a grid of standard colors.

  1. Click the color you want to use.

To create a custom color:

  1. Click the Custom... button located below or next to the color palette.
  2. The Select color dialog box is displayed. Use this box to select a color from the display or to enter the color value.

  1. If desired, you can add the color to your custom colors by clicking the Add to Custom Colors button.
  2. When finished, click OK.
