Export Options

Export options enable you to specify some options for exporting GeoTIFF, JPEG, and movie products.

To open the Export options: 

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Export to display a list of layers.
  3. Select the Export type: BMP, GeoTIFF, Ice, JPEG, Movie Product, or PNG.
  4. After making the appropriate updates, click OK to accept the selected options.

Click Cancel to cancel the changes.

GeoTIFF

Under Compression Options, enter the Rows Per Strip.

Check the Pack Bits checkbox to turn the compression on.

Ice

Under Compression Options, if you want to set the compression level, select GZIP or Shuffle + GZIP from the drop-down list. The default is no compression ("None").

Use the slider to set the GZIP Compression Level.

Under Chunk Size Options, select the Chunk size from the drop-down list or type in the number. The chunk size is measured in megabytes.

JPEG

For JPEG exports, you can select the percent of quality you want to maintain. Use the slider to make your selection.

Movie Product

Opticks provides numerous options for fine-tuning your movie product export.

The plus and minus boxes in front of the section names enable you to expand and collapse the options list boxes.
