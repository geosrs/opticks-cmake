Windows Workspace Options–Cube

The Windows Workspace–Cube options enable you to set the display options for cube images displayed in Opticks.

To open the Windows Workspace All options:  

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Windows to display a list of layers. Click the plus sign (+) next to Workspace. Click Cube.
  3. Update the options.
  4. Click OK to accept the selected options.

Click Cancel to cancel the changes.

Dynamic Inset

These options determine the size of the dynamic inset window (right-mouse + Ctrl). These options take immediate effect.

  • Size. This is the size in screen pixels.
  • Zoom Factor. This number is used to determine the relative zoom.
  • Zoom Factor Is.
  • Absolute. 100% X Zoom Factor.
  • Relative. The current zoom multiplied by the zoom factor.

Pan Speed

The Pan Speed allows you to control how fast the smooth-pan moves the image. The pan speed is in pixels.

Use Ctrl + Shirt + Arrow Key to Fast pan.

Use Ctrl + Arrow Key to Slow pan.

These options take immediate effect.

Note: Certain types of overlays (especially AOI layers, threshold layers, and pseudocolor layers) can be slow to render and may limit the maximum smooth-panning speed. See Generate Full Image.

Mouse Pan Sensitivity determines how quickly the mouse pan moves over the image. The higher the number, the faster the pan.

Other Options

  • Geo Coordinate Tool Tip. Displays geo coordinate tool tip information at the cursor if the geo information is available. The format is based on the Lat/Lon layer. The information is the same as that show in the status bar.
  • Confirm non-undoable layer delete. When turned off, you won't receive a confirmation dialog box when you delete a layer that cannot be undone.
  • Activate Layer When Histogram Selected. When selecting a histogram plot, the corresponding layer will be brought to the front.
  • Show Pixel Coordinates When Zoomed In. This checkbox controls what the numbers mean when you zoom in on an image. When it's not checked, the number represents the raw cube value. When it is checked, the pixel coordinates are displayed.

Default Panning and Zooming Limits

  • Pan Limit. Sets the limits for panning.

"No Limit" means that there are no boundary limits on panning. "Limit to Cube Extents" requires that the crosshair be in the boundary cube. "Limit to Extents of All Layers" limits panning to the extents of any layers in the image.

  • When Zooming In. You can't zoom in any closer than this limit.
  • When Zooming Out. You can't zoom out any further than this limit. This depends on the size of the window.
