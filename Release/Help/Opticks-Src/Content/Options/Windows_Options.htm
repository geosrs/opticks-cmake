Windows Options

The following windows options are available in Opticks:

  • Overview. Determines the way the image trail is displayed in the Overview Window and the zoom threshold for the trail.
  • Status Bar. Determines what fields are displayed in the status bar of the displayed image.
  • Workspace–All. Enables you to set the display options for images displayed in Opticks.
  • Workspace–Cube. Enables you to set the display options for cube images displayed in Opticks.
