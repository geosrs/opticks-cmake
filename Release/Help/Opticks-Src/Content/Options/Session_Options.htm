Session Options

The Session Options enable you to set the defaults for sessions in Opticks.

To open the Session Options:

  1. On the Opticks Main Application window, go to Tools and select Options.
  2. The Options dialog box is displayed. The panel on the left enables you to navigate to the available options. There are four session option types:
  • File Locations.
  • General.
  • Keyboard Shortcuts. Opticks provides both default shortcuts and the option of creating your own keyboard shortcuts.
  • Save.
  1. After updating the session options, click OK to save your changes.

Click Cancel to cancel the changes.

File Locations

The File Locations options set the default locations for Opticks files.

The Default Product Template specifies the template that is automatically applied when creating a new product or deriving a new product on the Layout menu. If the value is empty (the default), then no product template is applied.

The Text Editor file location specifies the external text editor that is invoked when editing the vector <string> wizard Value item. The Arguments field lists any arguments that would be passed into the editor if invoking it from the command line. For example, if an editor allowed files to be opened in tabs within a single window or separately in multiple windows, this setting can be specified in the Arguments field. The values that can be added here are solely based on the specified external editor.

To change a file location:

  1. Double-click on the file location in the Location column. The browse button is displayed.
  2. Enter the location string or click the browse button.
  3. If you click the browse button, the Browse For Folder dialog box is displayed.

Navigate to the folder and click the OK button. You can also create a new folder.

General

When you go to the Opticks Main Menu and select File, the most recently used (MRU) files are listed at the bottom of the menu. The default for the number of entries is 4. In the Most Recently Used (MRU) Files field, you can change the number of files displayed. The maximum number is 16.

Opticks sets aside an area of memory for saving the most recent Undo/Redo actions. You can change the size of this area in the Buffer Size field.

Some processes can run in parallel. The Multi-threading option determines the number that can run in parallel at any time and depends on the number of CPUs attached to the computer you're using. You can change the number of Worker Threads based only on your number of CPUs. If this number is set to "1," the processes won't run in parallel, regardless of the number of CPUs.

When the Progress Dialog/Automatically close on process completion checkbox is checked, all successful progress dialogs close automatically without requiring you to select a confirmation button.

The Mouse Wheel Zoom Direction determines the direction of the mouse zoom. When Alternate mouse wheel zoom direction is checked, moving your mouse wheel towards you zooms in on the image. Moving the mouse wheel away from you zooms out. This is the default.

When Alternate mouse wheel zoom direction is not checked, moving your mouse wheel toward you zooms out. Moving the mouse wheel away from you zooms in. This is the mode used by many commercial applications such as Google Earth.

Save

You can set the following options for saving a session:

  • Save On Close. This option determines whether the session is saved when you close Opticks.
  • Auto-Save. When enabled, this option automatically saves the current session based on the selected interval. The defaults are that Auto-Save is enabled and saves the current session every 15 minutes.
