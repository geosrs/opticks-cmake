Optional Messages Options

The Optional Messages Options allow you to enable and disable all messages that are displayed in suppressible dialog boxes.

To open the Optional Messages options: 

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Optional Messages. Click Opticks.
  3. Under Display Optional Messages, check the checkboxes for any optional messages you want to enable. Uncheck the checkboxes for any optional messages you want to disable.
  1. Click OK to accept the selected options.

Click Cancel to cancel the changes.
