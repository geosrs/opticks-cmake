Keyboard Shortcuts

Opticks provides a number of shortcuts using your computer keyboard or a combination of your mouse and your keyboard.

You can also create your own shortcut keys. See Creating Shortcut Keys.

To view the keyboard shortcuts currently assigned to your version of Opticks:

  1. On the Opticks Main Application window, go to the Tools menu and select Options.
  2. In the left panel, go to Session and select Keyboard Shortcuts.

The panel on the right displays a navigation tree of the functions available for shortcuts and the currently assigned shortcuts. Use the and to expand and collapse the navigation tree.

For the default keyboard shortcuts, see Default Keyboard Shortcuts.

You can change the defaults using the instructions in Creating Shortcut Keys.
