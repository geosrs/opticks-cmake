Pseudocolor Layer Options

The Pseudocolor Layer Options dialog enables you to set the default marker properties for the pseudocolor layer.

To open the Pseudocolor layer options:  

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Layers to display a list of layers. Click Pseudocolor.
  3. In the Default Marker Properties, use the drop-down list to select the Symbol type you want to use.
  4. Click OK to accept the selected options.

Click Cancel to cancel the changes.
