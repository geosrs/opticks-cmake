Options

Opticks provides the ability to set image display and other options.

Note that Opticks plug-ins may also include additional options. The options described here are the standard Opticks options and do not include plug-ins.

The plus and minus boxes in front of the section names enable you to expand and collapse the options list boxes.

To access the view options:

  1. On the Opticks Main Application window, go to Tools and select Options.
  2. The Options dialog box is displayed. The panel on the left enables you to navigate to the available options. Here are the general option types:
    • Animation Options. Enable you to set the defaults for the animation cycle and the frame speeds on the Animation Toolbar.
    • Export Options. Enable you to select the export options for GeoTIFF, JPEG, and movie product exports.
    • Import Options. Enable you to set the import options for georeferencing a raster data set.
    • Layers Options. Enable you to set the default properties for new display layers.
    • Optional Messages Options. Allow you to enable and disable all messages that are displayed in suppressible dialog boxes.
    • Session Options. Enable you to set the defaults for sessions in Opticks.
    • Windows Options. Enable you to set the defaults for overview, status bar, and workspace windows display.

  1. When finished, to apply the options you select, click the OK button. The new values are automatically saved in the user configuration file.

To cancel the changes, click the Cancel button.
