Creating Shortcut Keys

As an example, we will create a shortcut key for creating an animation. When you create a shortcut key, be sure that you select a keystroke combination that is not already being used.

For our example, we will use the Alt key plus the A key.

  1. On the Opticks Main Application Window, go to the Tools menu and select Options.
  2. The Options dialog is displayed. Navigate to Session, Keyboard Shortcuts.

The Current Shortcuts table provides a list of functions that are available for shortcut keys.

  1. In the Current Shortcuts list, click on the plus sign (+) next to View to expand the list of View functions.

  1. Click the Shortcut column next to the function.
  2. On your keyboard, press the keys you want to use as your shortcut.

Using the example, press the "Alt" key and, still pressing it down, press the "A" key. Opticks records your keystrokes. [Opticks automatically inserts the plus sign (+) in between the keystrokes.]

  1. Click the OK button.
