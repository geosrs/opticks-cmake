Measurement Layer Options

The Measurement Layer Options enable you to set the default options for the Measurement Toolbar.

To open the Measurement Layer options:

  1. On the Opticks Main Application Window, to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Layers to display a list of layers. Click Measurement.
  1. Select the option properties you want to use. There are three general groups of option properties:
  • Default Display Properties. These properties determine how the measurement is labeled, including the precision, the location units, and the distance units.
  • Default Line Properties. These properties determine the measurement line width, style, and color.
  • Default Text Properties. These properties determine the measurement text font type, size, and color.

4. When finished, click OK to accept the selected options.

Click Cancel to cancel the changes.
