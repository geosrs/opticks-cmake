GCP Layer Options

The GCP Layer Options enable you to set the default options for creating GCPs.

You can select the symbol type, size, and color used for GCPs.

To open the GCP layer options: 

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Layers to display a list of layers. Click Gcp.
  3. Use the drop-down list to select the Symbol type you want to use.

To change the label size, select the number from the Symbol Size list or type in a new number.

Use the drop-down list to select the Color of the marker. For more information, see Edit Color.

  1. Click OK to accept the selected options.

Click Cancel to cancel the changes.
