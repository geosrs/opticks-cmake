Import Options

Import options enable you to specify some import options for georeferencing a raster data set.

To open the Import options: 

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Import. Select Raster Element.
  3. To enable the georeference options, check the Automatically georeference on import checkbox.

Select either Use plug-in determined by the importer or Use best available plug-in.

If you select Use best available plug-in, you can select your preferred plug-ins. Select the plug-in from the Available Plug-Ins list and click the Add-> button to add it to the Preferred Plug-Ins list. The plug-ins are used in the order in which they appear in the Preferred Plug-Ins list. If the data can be georeferenced with the first plug-in in the list, the first plug-in is used. If the first one can't be used, the second one is used. And so on down the list.

To remove a plug-in from the Preferred Plug-Ins list, select the plug-in and click the <-Remove button.

Check the Display latitude/longitude layer checkbox to show this layer.

  1. After making the appropriate updates, click OK to accept the selected options.

Click Cancel to cancel the changes.
