Tie Point Layer Options

The Tie Point Layer Options enable you to set the default options for creating tie points.

To open the Tie Point layer options:

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Layers to display a list of layers. Click Tie Point.
  3. To change the label size, select the number from the Symbol Size list or type in a new number.

Use the drop-down list to select the Color of the marker. For more information, see Edit Color.

The Auto Color option selects a new default color for each layer that is added to the view. This allows you to quickly distinguish selected pixels on multiple tie point layers without needing to change each layer color individually.

If you select auto color, the default color option for the layer is ignored.

To show labels on the Tie Point Layer, check the Show Labels checkbox. Uncheck the box to hide the labels.

  1. Click OK to accept the selected options.

Click Cancel to cancel the changes.
