Annotation Image Palette Options

The Annotation Image Palette Options enable you to set the path for annotation images and the initial width for loading them.

  1. Go to Tools, Options.
  2. In the Options window, go to the options list on the left. Expand the Windows tree and select Annotation Image Palette.

  1. To add a palette path, right-click in the Palette Path box and click Add.

OR

On your keyboard, press the Insert button.

  1. Click the browse button in the Palette Path box. (If it's not displayed, click in the area just below the banner "Palette Path." This should activate the button.)

  1. In the Browse For Folder dialog box, navigate to and select the folder where the palette files are located. To create a new folder, click the Make New Folder button.

  1. Click the OK button.
  1. By default, the palette is loaded using its full width. To adjust the size, click in the Full Size checkbox. This activates the Initial Image Width box. Change the number.

  1. Click the OK button.

Remove a Palette Path

  1. Go to Tools, Options.
  2. In the Options window, go to the options list on the left and select Annotation Image Palette.
  3. In the list under Palette Paths, right-click on the path you want to remove. Select Remove.

OR

Select the path you want to remove. On your keyboard, press the Delete key.

  1. Click the OK button.
