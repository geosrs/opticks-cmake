Windows Status Bar Options

The Windows Status Bar Options determine what fields are displayed in the status bar of the displayed image. This status is located in the lower left corner of the Opticks Main Application window when you are working with an image.

To open the Windows Status Bar options:  

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Windows to display a list of layers. Click Status Bar.
  3. Update the Displayed Fields checkboxes.
  4. Click OK to accept the selected options.

Click Cancel to cancel the changes.
