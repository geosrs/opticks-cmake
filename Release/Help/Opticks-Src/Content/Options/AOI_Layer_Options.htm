AOI Layer Options

The AOI Layer Options enable you to set the default options for creating AOIs.

To open the AOI layer options:

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Layers to display a list of layers. Click Aoi.
  3. Use the drop-down list to select the Symbol type you want to use.

Use the drop-down list to select the Color of the marker. For more information, see Edit Color.

The Auto Color option selects a new default color for each layer that is added to the view. This allows you to quickly distinguish selected pixels on multiple AOI layers without needing to change each layer color individually.

If you select auto color, the default color option for the layer is ignored.

  1. Click OK to accept the selected options.

Click Cancel to cancel the changes.
