Threshold Layer Options

Use the Threshold Layer Options to set the default options for the threshold layer.

To open the Threshold layer options:  

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Layers to display a list of layers. Click Threshold.
  3. Enter the default marker properties and default pass area options you want to use.
  4. Click OK to accept the selected options.

Click Cancel to cancel the changes.

Default Marker Properties

Use the drop-down list to select the Symbol type you want to use.

Use the drop-down list to select the Color of the marker. For more information, see Edit Color.

The Auto Color option selects a new default color for each layer that is added to the view. This allows you to quickly distinguish selected pixels on multiple layers without needing to change each layer color individually.

If you select auto color, the default color option for the layer is ignored.

Default Pass Area

The Default Pass Area sets the options for the threshold.

The Pass Area determines the pass area based on the threshold values.

  • Less Than. All pixels with a results value less than the first value. The second value is ignored.
  • Greater Than. All pixels with a results value greater than the first value. The second value is ignored
  • Between. All pixels with a results value greater than the first value and less than the second value.
  • Outside. All pixels with a results value less than the first value and greater than the second value.

The Pass Area Units determine the meaning of the first and second values.

  • Raw. The numbers reflect the same units as the threshold data.
  • Percentage. The stretch values are represented in terms of their position relative to the maximum and minimum raw values in the data set.
  • Percentile. The stretch values are represented in terms of their percentile within the data set. For example, a stretch value that is higher than half the data values in the cube and lower than the other half is represented as “50.” A value that is higher than 75% of the data and lower than the other 25% is represented as “75.”
  • Standard Deviation. The values are represented in terms of how many standard deviations they are from the average data value in the image. A negative value indicates a value that is less than the average and a positive value indicates a value greater than the average.

The First Value and Second Value set the thresholds.
