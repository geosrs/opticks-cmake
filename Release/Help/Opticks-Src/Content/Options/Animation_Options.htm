Animation Options

The Animation Options enable you to set the defaults for the animation cycle and the frame speeds on the Animation Toolbar.

The animation cycle determines the behavior of the movie player after the final frame is displayed. The frame speeds are a measure of how fast the animation player moves through the frames of the animation.

To open the Annotation options window:

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click Animation.

The Animation options are displayed.

To enable the animation to drop frames:

Check the Can Drop Frames checkbox.

To change the default animation cycle:

Go to the Default Animation Cycle field. Click the down arrow button and select a new cycle. For more information on the available animation cycles, see Animation Cycle.

To edit the default frame speeds:

The default frame speeds are the speeds that are displayed in the frame speed drop-down list on the Animation Toolbar.

To add a new frame speed:

  1. Click the Add button.
  2. The cursor is activated at the end of the list. Type in the new speed and press the Enter key on your keyboard.

To remove a new frame speed:

  1. Select the speed from the list of default values.
  2. Click the Remove button.
