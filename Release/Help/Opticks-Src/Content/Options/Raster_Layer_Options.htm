Raster Layer Options

Use the Raster Layer Options to set the default options for the raster layer.

To open the Raster layer options:

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Layers to display a list of layers. Click Raster.
  3. Select the option properties you want to use. There are four general groups of option properties: default image, default RGB stretch, default grayscale stretch, and color composites.
  4. When finished, click OK to accept the selected options.

Click Cancel to cancel the changes.

Default Image Properties

  • Complex Component. This option controls how data is generated for complex components. The options are magnitude, phase, in phase (real data), and quadrature (imaginary data).
  • Enable Dynamic Texture Generation. This enables you to generate an image in real time. You can only use this option if your graphics card supports it. You should select it only if you know you need it.
  • Fast Contrast.
  • Background Tile Generation. This option enables you to generate textures in the background. This option is only useful if you can allow it to run for long periods without doing anything else on your PC.

Default RGB Stretch

These options apply to the stretch values of RGB histograms.

  • Lower and Upper Values. The stretch values are the range of values the stretch is mapped over. You can enter the Lower and Upper values for each band. The meaning is dependent on the stretch units.
  • Units. Use the drop-down list to select the stretch units. The stretch units determines the meaning of the numbers displayed in the “stretch values” field. Stretch units are applied by band (for RGB there is one for red, one for green, and one for blue). 
  • Raw. The numbers reflect the same units as the raster data. For example, if the data cube is in reflectance scaled from 0–10,000, the stretch values will correspond directly to these 0-10,000 values.
  • Percentage. The stretch values are represented in terms of their position relative to the maximum and minimum raw values in the data set.
  • Percentile. The stretch values are represented in terms of their percentile within the data set. For example, a stretch value that is higher than half the data values in the cube and lower than the other half is represented as “50.” A value that is higher than 75% of the data and lower than the other 25% is represented as “75.”
  • Standard Deviation. The stretch values are represented in terms of how many standard deviations they are from the average data value in the cube. A negative value indicates a stretch value that is less than the average and a positive value indicates a stretch value greater than the average.
  • Type Use the drop-down list to select the stretch type. This applies to the entire display.

  • Linear. The stretch is linear from the lower stretch value to the upper one. A value halfway from the lower to the upper (at the median of the two values) is shaded at 50% brightness.
  • Exponential. The contrast stretch is exponential in shape from the lower to upper value. This has the effect of suppressing weaker/darker features in the image and accentuating the brighter ones.
  • Logarithmic. The contrast stretch is logarithmic in shape from the lower to the upper value. This has the effect of accentuating the weaker/darker features of the image.
  • Histogram Equalization. Sets the raster layer to an equalized histogram contrast stretch. Every display value (0-255) has an equal number of pixels.

  • Favorites. Select, add, or remove favorites based on selected values.

Color Composites

Opticks provides three predefined color composites for image display: NIR, SWIR, and True Color.

You can customize these composites. You can also create new composites. For more information, see Color Composites.
