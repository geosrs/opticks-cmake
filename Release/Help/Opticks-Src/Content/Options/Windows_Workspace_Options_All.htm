Windows Workspace Options–All

The Windows Workspace–All options enable you to set the display options for images displayed in Opticks.

To open the Windows Workspace All options:  

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Windows to display a list of layers. Click the plus sign (+) next to Workspace. Click All.
  3. Update the options.
  4. Click OK to accept the selected options.

Click Cancel to cancel the changes.

Default Size

Use the Default Size box to set the size of the window upon opening.

  • Fixed Size. Sets the window to a fixed size. The default is 450 pixels by 250 pixels.
  • Maximized. Maximizes the window in the Opticks Main Application window area.
  • Percentage. Sets the window as a percentage of the Opticks Main Application window. (This does not include the toolbar and dock window space.)

Default Zoom

Use the Default Zoom area to set the initial zoom mode.

  • Zoom To Fit. The view is initially zoomed so the view extents are fully contained in the window.
  • Percentage. The view is initially zoomed to the given percentage. A zoom level of 100% means that one pixel in the data set is displayed in one pixel on the screen.

Copy Snapshot

When the Use View Resolution checkbox is checked, the default export uses the view resolution of the currently active view

When this checkbox is not checked, you can set the defaults. Enter the Width and Height in pixels.

In the Copy Method box, select Fixed Size or Bounding Box.

Fixed Size exports the image using the sizes in the Width and Height fields.

Bounding Box exports the largest image possible if fit into the bounding box with the aspect ratio of the view being exported.

Other Default Properties

Use the Other Default Properties area to set the background color and origin location.

  • Background Color. Sets the background color of the image window. For more information on setting colors, see Edit Color.
  • Origin Location. Use this drop-down list to specify the location of the 0, 0 coordinate in the window. Currently Upper Left and Lower Left are available.
  • Confirm Close. When this check box is checked, you must confirm closing the image window. If this box isn't checked, you don't have to confirm closing an image window.
