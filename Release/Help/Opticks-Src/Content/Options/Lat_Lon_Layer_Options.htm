Lat/Lon Layer Options

The Lat/Lon Layer Options enable you to set the default properties for displaying the latitude and longitude layer on an image.

To open the Lat/Lon layer options: 

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Layers to display a list of layers. Click Lat/Lon.
  3. Use the drop-down lists to select the options you want to use.

Default Label Properties enable you to select the DMS (degrees, minutes, seconds) format as well as the font type and size.

Default Gridline Properties enable you to select the gridline style, width, and color.

  1. Click OK to accept the selected options.

Click Cancel to cancel the changes.
