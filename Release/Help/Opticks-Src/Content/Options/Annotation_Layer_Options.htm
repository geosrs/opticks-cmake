Annotation Layer Options

The Annotation Layer Options enable you to set the default options for creating annotations.

To open the Annotation Layer options:

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Layers to display a list of layers. Click Annotation.
  3. Select the option properties you want to use. There are four general groups of option properties:
  • Default Common Properties. These properties determine how the annotation object looks including the line width, style, and color, the fill style and color, hatch style, opacity, and whether the object is filled and has a border.
  • Default Text Properties. These properties determine the annotation text font type, size, and color.
  • Default Arc Properties. These properties determine the start and stop angles for annotation arcs as well as the arc region.
  • Default Triangle Properties. The triangle properties enable you to set the default apex for annotation triangles

4. When finished, click OK to accept the selected options.

Click Cancel to cancel the changes.
