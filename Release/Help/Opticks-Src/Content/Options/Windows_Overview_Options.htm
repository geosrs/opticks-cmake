Windows Overview Options

The Overview Options determine the way the image trail is displayed in the Overview Window and the zoom threshold for the trail.

When you open the Overview Window and pan an image, Opticks leaves a colored "snail trail" of what you've looked at.

The options for this trail enable you to set the color, opacity, and zoom threshold for this trail.

To open the Windows Overview options:  

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Windows to display a list of layers. Click Overview.
  3. Update the Default Trail Properties.

The Color is the color of the trail. For more information on changing the color, see Edit Color. 

The Opacity determines how opaque the trail color is on the Overview Window. Zero percent is completely transparent. One hundred percent is completely opaque.

Opticks won't draw the trail until you reach the Zoom Threshold.

  1. Click OK to accept the selected options.

Click Cancel to cancel the changes.
