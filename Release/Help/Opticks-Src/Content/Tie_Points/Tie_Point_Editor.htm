Tie Point Editor

The Tie Point Editor displays information about the tie points in a tie point layer. It allows you to add new tie points, edit the values for existing tie points, and delete individual tie points.

Note: You can leave the Tie Point Editor open while you work in Opticks.

You can use the arrow keys and the Page Up and Page Down keys on your keyboard or the scroll bar to navigate through the table of pixels.

To open the Tie Point Editor:

On the Tie Point Toolbar, click the Tie Point Editor button.

OR

On the Tools menu, select Tie Point Editor....

The Tie Point Editor window is displayed. This window has a table with the following information for each identified pixel:

  • The X and Y coordinates of the reference (or primary) scene.
  • The offset of the X and Y coordinates of the mission scene.
  • The confidence and phi values. These are two integers or an algorithm to associate data with a tie point.

 

To edit tie point values:

  1. Click in the cell you want to edit.
  2. Type in the new value.

To find a tie point:

  1. Click the Go To... button.
  2. In the Go To Row dialog box, enter the row number of the tie point.
  3. Click OK.

To add new tie points:

  1. Click the Add button.
  2. Opticks adds a new row to the end of the table. Enter the values of the new tie point.

To delete existing tie points:

  1. Click a cell in the row of the tie point you want to delete.
  2. Click the Delete button. This deletes the whole tie point.
