Delete Tie Points

When the Tie Point toolbar is active, you can delete rectangular regions of tie points from the reference (or primary) scene. The deletions are reflected in the mission scene. You cannot delete tie points directly from the mission scene.

To delete a tie point region:

  1. Be sure the Tie Point Toolbar is active.
  2. Use your left mouse button to select a rectangular region where you want to delete the tie points. When you release the mouse button, all the points in the selected region are deleted.
