Working with the Opticks Main Application Window

When you open the Opticks application, the Main Application window is displayed. This window provides access to all Opticks functionality, including plug-ins. You can customize your display of this window by displaying and hiding toolbars, moving the toolbars, and by docking and undocking toolbars and some windows.

The Session Explorer is a dockable window that enables you to manage your Opticks session and provides access to some of Opticks' functionality. For more information about using the Session Explorer, see Session Explorer.

Note: Opticks saves the toolbar and dock window positions when you close Opticks and they will be in the same positions when you open Opticks again.

Drop-Down Menus

The drop-down menus provide commands for Opticks functionality. Opticks has a standard set of menus but plug-ins and wizards may also provide menus that are displayed in the Opticks Main Application Window.

Menu commands may also be available on the Opticks toolbars. Some commands also have associated shortcut keys.

Drop-down menus can also appear on a toolbar.

Toolbar Area

The toolbars provide access to some of the same functionality as the Opticks drop-down menus. They also provide access to specialized functionality. For example, the Annotation functionality provides you with the means to create and modify objects, lines, and text on the image’s annotation layer.

Toolbars can be toggled on and off.

There are two ways to access the toolbars:

  • Go to the View drop-down menu and select Toolbars. The list of toolbars is displayed. Active toolbars have a checkmark in front of them. Select the toolbar name to toggle it on or off.

OR

  • Using your mouse, move the pointer to the toolbar area of the Opticks window. Right-click the mouse. The list of windows and toolbars is displayed. Active toolbars have a checkmark in front of them. Select the toolbar name to toggle it on or off.

You can move toolbars around the Toolbar Area or dock them along any side of the main application window. Toolbars can also float anywhere on the desktop, even outside the main application window.

To move a toolbar:

  1. Click and hold your left mouse button on the toolbar handle.

  1. Continue holding the mouse button and move the toolbar to the desired location.
  1. Release the mouse button.

For more information on toolbars, see Toolbars.

Image Display Area

The Image Display Area is the main workspace for Opticks. Image windows and product windows are displayed in this area. These windows can be moved around the Image Display Area and can be minimized and maximized.

Other windows such as the spectrum window and histogram window (as well as toolbars) are also displayed in this area, although they can be undocked and moved anywhere on your computer screen display.

Shortcut Keys and Mouse Actions

Shortcut keys and mouse actions provide keyboard and mouse shortcuts that you can use throughout the Opticks application. See Keyboard Shortcuts and Mouse Actions.

Note that you can also assign your own shortcut keys. These shortcuts are saved in Opticks. See Creating Shortcut Keys.

Plug-Ins

Opticks supports a variety of plug-ins. Note that plug-ins may add toolbar buttons.
