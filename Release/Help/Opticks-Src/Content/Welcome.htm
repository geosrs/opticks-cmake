Welcome to Opticks.

Opticks is a collection of software tools that enables analysts to develop intelligence products. Opticks generates these products by analyzing signatures and data sets collected from specific technical sensors. Opticks presents data sets and signatures to the analyst using images and plots. The analyst then uses these views to manipulate the data and run algorithms to generate intelligence products.

Opticks provides a core application suite for manipulating data and running algorithms. It also provides access to a number of plug-ins.

To get started using the Opticks software, see Working with the Main Application Window or the online Tutorials provided for the Opticks Suite.
