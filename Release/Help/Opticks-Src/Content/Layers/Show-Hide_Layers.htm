Show/Hide Layers

To show or hide a layer:

  1. In the Session Explorer window, navigate to the layer name.
  2. Select the checkbox in front of the layer name. When this box is checked, the layer is displayed. When this box is unchecked, the layer is hidden.

OR

Right-click on the layer name and select Displayed. This toggles the layer display on and off. If the layer is already displayed, the Displayed menu item has a check in front of it.

When the layer is hidden, the name is displayed in the Session Explorer window in italics.
