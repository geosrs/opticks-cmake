Working with Layers

Use the Session Explorer window to work with the layers of the displayed image. Right-click on the layer you want to use to access the Layer Menu.

The Layer Information Menu provides the following options:

  • Displayed. Toggles the layer display.
  • Activate. Activates the layer. Only displayed if the layer is not the active layer. See Activate a Layer.
  • Copy. Enables you to create a new layer that is a copy of the selected layer.
  • Convert.... Enables you to convert the layer to a different layer type. Creates a new layer and deletes the existing layer. The underlying data element that the layer displays is not modified or deleted. Currently the raster, threshold, and pseudocolor layers can be converted between the three types.
  • Delete. Deletes the layer.
  • Rename.... Enables you to rename the layer.
  • Export. Enables you to export the layer.
  • Properties. Displays the layer properties dialog box. See Properties.
