Create a New Layer

To create a new layer in an image:

  1. Go to the Session Explorer window. Select the name of the image you want to add the layer to.
  2. Right-click on the image name. On the Image Menu, select New Layer. On the New Layer sub-menu, select the type of layer you want to create. You can create the following types of new layers: Annotation, AOI, GCP List, and Tie Point. Opticks automatically assigns a layer name, although you can rename the layer.

The new layer is now displayed in the Session Explorer window. The name of the active layer is in bold.
