Activate a Layer

To activate a layer:

  1. Go to the Session Explorer window. Select the name of the layer you want to activate.

  1. Right-click. On the Layer Menu, select Activate. (This option is displayed ONLY if the layer you select is not the active layer.) The layer name is now displayed in bold. This indicates that it is the active layer. The associated toolbar is also activated.
