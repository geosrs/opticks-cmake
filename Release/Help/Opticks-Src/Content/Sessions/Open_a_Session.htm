Open a Session

Follow these steps to open a previously saved session.

Note: You cannot open a session from a previous version of Opticks, or from a different platform on the same version of Opticks. In other words, a session saved on 32-bit Windows will not load on 64-bit Windows.

To open a session:

  1. On the Opticks Main Application window, go to the File menu and select Open Session....
  2. The Open Session File dialog box is displayed. Navigate to the directory where the session file is saved. Select the file name and click the Open button.

The existing session is closed (as when you create a new session) and the session you selected is loaded into Opticks. All windows and data are restored to the state they were in when the session was saved.
