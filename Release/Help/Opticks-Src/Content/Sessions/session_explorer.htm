Session Explorer

The Session Explorer is a window that enables you to manage your Opticks session and provides access to some of Opticks' functionality.

To open the Session Explorer:

  1. On the Opticks Main Application Window, go to the Tools menu.
  2. Select Session Explorer.

OR

  1. Go to the Toolbox Toolbar and click the Session Explorer button.

The Session Explorer window is displayed. This window has four tabs: Windows, Animations, Elements, and Plug-ins. Click on the tab name at the bottom of the Session Explorer window to activate the tab.

Use the tree structure to expand or collapse the information displayed in this window.

Use the check boxes to toggle on and off the items available such as image layers or toolbars. Items that are checked are currently displayed.

Text that is displayed in bold in this window indicates the active window, layer, band, etc.

Windows Tab

The Windows Tab has three general areas: Workspace Windows, Dock Windows, and Toolbars.

The Workspace Windows Area lists the images currently loaded into Opticks and the layers associated with the image. Note that you can change the layer display using drag-and-drop in the Session Explorer. This is the only area of the Session Explorer that allows drag-and-drop.

The Dock Windows Area enables you to work with the dock windows that are available in Opticks. By right-clicking on the window name in the Session Explorer, you can display and undock the selected window. Some windows have additional functionality available using the right-click menu.

The Toolbars Area lists the toolbars that are available for your version of Opticks.

Animations Tab

After you create an animation, the animations tab enables you to play and delete it.

Elements Tab

The Elements Tab lists all data elements currently loaded into the Opticks session.

Plug-Ins Tab

The Plug-Ins Tab provides a list of the Opticks plug-ins currently loaded for your version of Opticks. The first level of the tree structure lists general areas of plug-ins. Expand the tree structure to view the names of individual plug-ins.
