Save a Session

Follow these steps to save your current session. Saving a session saves all your work as well as the placement of all Opticks windows at the time that the session is saved.

Note: By default, Opticks automatically saves the current session every 15 minutes. You can change this interval in your session options.

To save the current session using the existing session name:

  1. On the Opticks Main Application window, go to the File menu and select Save Session.
  2. Opticks saves the session using the current name, either the default name or the name you already assigned. The default name is Opticks.session.
  3. After the session is saved, the Save Session dialog box is displayed. Click the Close button.

 To save the current session using a different session name:

  1. On the Opticks Main Application window, go to the File menu and select Save Session As....
  2. The Save Session As... dialog box is displayed. Navigate to the directory where you want to save the session. Opticks gives the session a default file name, but you can change it. The extension is ".session."

  1. Click the Save button.
  2. After the session is saved, the Save Session dialog box is displayed. Click the Close button.
