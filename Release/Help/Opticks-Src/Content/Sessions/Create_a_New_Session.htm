Create a New Session

Follow these steps to stop the current session and create a new one.

To create a new session:

  1. On the Opticks Main Application window, go to the File menu and select New Session.
  2. The Close Session dialog box is displayed. If you choose Yes, the Save Session As... dialog box is displayed so you can save the current session. See Save a Session for more information.

If you choose No, the current session is not saved and any open image windows are closed.
