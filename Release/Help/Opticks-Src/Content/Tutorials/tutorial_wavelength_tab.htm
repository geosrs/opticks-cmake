Tutorial: Using the Wavelengths Tab and Wavelength Importer

Learning Objective: When finished with this tutorial, you will be familiar with using the Wavelengths Tab to import wavelength data and to generate start and end wavelengths.

In this tutorial, you will perform the following:

  • Begin importing a hyperspectral data cube.
  • Import wavelength data into the Wavelengths Tab.
  • Calculate the start and end wavelengths based on the center wavelengths and the Full Width at Half Max (FWHM).
  • Export the wavelengths to a file.
  • Complete the hyperspectral data cube import.
  • View the wavelengths of the imported dataset.

Lesson: The Wavelengths Tab allows you to import wavelength data from a loaded dataset or from a wavelength file. It allows you to calculate the start and end wavelengths when given the center wavelengths. You will save the newly calculated wavelength information and associate them against the hyperspectral dataset.

Open File

  1. On the Opticks menu bar, select File, Import Data....
  2. In the Import window, go to the Importer field and select the ENVI Importer. Navigate to and select the sample data file f970620t01p02_r03_sc01.a.hdr and select Options.
  3. Click on the Wavelengths Tab.

Notice the current wavelength data is limited to the center wavelength.

  1. Click the FWHM button to calculate the start and end wavelengths data.
  2. Keep 1.00 in the FWHM Constant field and click the OK button.

The start and end wavelengths are calculated based on the FWHM values in the sensor's metadata, or estimates the FWHM based in the input constant value. For more information on the FWHM Constant and calculation, see Wavelengths Tab.

Notice the start and end wavelengths have now been calculated and populated in the table. You are now ready to export the data into a file for future use.

  1. Click the Save button.
  2. In the Save Wavelengths Window, navigate to the directory where you want to save the file enter a filename that you can associate with the cube for later use on import. Click the Save button.

  1. A dialog box displays the message "Wavelength export completed successfully." Click OK.
  2. You are now finished with the wavelengths, so click the OK button. On the Import window, click Open to import the data set.

You have now successfully calculated all the wavelength data needed to improve processing and exploitation results.

Wavelength Properties

  1. On the Session Explorer, select the Elements Tab. Right-click on f970620t01p02_r03_sc01.a.bip and select Properties.
  2. Select the Wavelength tab. The start, center, and end wavelengths set during import are displayed.

Summary/Review: You should now be able to import a hyperspectral dataset using the Wavelengths tab to associate wavelengths with the data set.
