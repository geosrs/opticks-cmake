Tutorial: Product Mode in Opticks

Learning Objective: When finished with this tutorial, you will be familiar with creating and deriving a product in Opticks.

In this tutorial, you will perform product window functions in the following sequence:

  • Derive a product
  • Edit a derived product
  • Create a new product
  • Add items to a blank product canvas

Lesson: A Product Window is not associated with a single data set but allows you to create products by arranging annotation objects on a virtual sheet of paper.

A product has two layers: a Layout layer and a Classification layer. The Layout layer allows you to add annotation objects to the product. The Annotation layer must be active before annotation objects can be added. The Classification layer allows you to edit the properties of the classification markings on the virtual paper.

Opticks provides two methods for creating new products: derived product or new product. To create a derive product, the data file must be loaded into Opticks. A product will be derived from the view that is displayed in the currently active window. Any annotations, GCPs, AOIs, and other additional layers will be displayed in the derived product. To create a product completely from scratch, you start with a blank Product canvas and add whatever needs to be seen on the product. This product is custom made by you. Whichever method you choose, the product should be created after the analysis is complete.

These two methods are accessible from the Layout menu on the main Opticks toolbar. A lesson on each method will be given below in the tutorial.

Derive a Product

  1. On the Opticks menu bar, select File, Import Data.... Navigate to and select the file pyramids_psh_2feb2002_8bitgeotiff.tif. Make sure the Importer is either the Auto Importer or the GeoTIFF Importer. Click the Open button.

  1. Go to the Session Explorer. Right-click on pyramids_psh_2feb2002_8bitgeotiff.tif. Select New Layer, Annotation.

On the Annotation Toolbar, click on the New Annotation Object button and select Text Box.

Click in the display window. The Enter Text dialog box is displayed.

Type in “Exercise” and click the OK button. The text will appear with white squares around it.

  1. On the Annotation Toolbar, go to the font size drop-down box and select 48.

On the Annotation Toolbar, select the Annotation Text Color button  and change the text color to yellow.

Derive Product

  1. On the Opticks menu bar, go to Layout and select Derive Product.
  2. Go to the Display Toolbar and click the Zoom to Fit button. Notice that the annotation made from the original data file is displayed in the product.

Edit Product

  1. In the Session Explorer, expand Product 1/Layout/View 1/pyramids_psh_2feb2002_8bitgeotiff.tif/Annotation 1/Text.
  2. Right-click on the Text 1 object under Annotation 1 in the Session Explorer and select Properties.
  3. Scroll down to the Text area in the properties window. Click on the yellow box labeled “Color” under the “Text” portion of the properties dialog and change the color to red.

Click OK to close the dialog and apply the change.

  1. Go to the Opticks menu bar and select Window, Tile.

  1. Close both the image window and the product window.

Create a New Product

  1. Go to the Opticks menu bar and Select File, Import Data.... Navigate to and select the file pyramids_psh_2feb2002_8bitgeotiff.tif. Make sure the Importer is either the Auto Importer or the GeoTIFF Importer. Click the Open button.
  2. Select Layout, New Product from the Opticks menu bar. A blank Product canvas is displayed.
  1. To view the blank canvas and the image side-by-side, select Window, Tile. Both items will appear in the data display window.

Add Items to Blank Product Canvas

To produce a customized product, you can populate the blank Product canvas with annotations, cube views, images, plots, etc. You can add items to the canvas using the Annotation Toolbar or by copying/pasting from other currently loaded products. You can only add Cube Views from cubes currently loaded into Opticks.

  1. To add a Cube View to the new product, select the Layer Edit Mode button  on the Toolbox Toolbar.

On the Annotation Toolbar, go to Add New Object and select the View button.

This allows you to draw a new Cube View on the Product canvas. Place the cursor (paint brush) in the blank image and click and drag the view area to the size you want.

When you release the mouse button, the Select View Image dialog box is displayed.

  1. In the drop-down list, select the pyramids_psh_2feb2002_8bitgeotiff.tif file.

You can add several Cube Views to a Product. However, you need to follow this procedure for each Cube View created on a Product.

  1. Click the OK button. The image will appear in the View Cube. Click on the white squares and drag to enlarge the image.

  1. On the Annotation Toolbar, click on the New Annotation Object button and select Text Box.

Click the paint brush cursor in New Product window. An Enter text dialog box will appear. Type in "Exercise" and click the OK button. The text will appear in the New Product window.

  1. Click on File, Close or select the small “x” in the upper right corner to exit.

Summary/Review: You should now be able to create and derive products in Opticks by performing various product window functions.
