Tutorial: Run a Wizard in Batch Mode

Learning Objective: When finished with this tutorial, you will be familiar with running a wizard in batch mode. Note that you will use the wizard you created and saved in Tutorial: Wizard Builder.

In this tutorial, you will perform these functions:

  • Reload a previously saved wizard
  • Delete and add Desktop Item/Plug-In Item/Value items
  • Connect new algorithms and processes
  • Change Value Item numbers
  • Save the new wizard
  • Run the wizard in Batch Mode

Lesson: In addition to wizards created in the Wizard Builder, Opticks supports Batch Processing using batch wizards. A batch wizard contains regular wizards and provides expanded batch processing capabilities including the following:

  • Multiple wizard files can be executed in sequence.
  • Wizard files can be repeated multiple times consecutively with different input values on each wizard run.
  • Wizards can be run completely in batch mode, without using the Opticks Main Application window.
  • Wizards can be run interactively with little or no user input, but you can still see and manipulate results.

Toolbars used in this tutorial:

Wizard Builder Toolbar

Reload the Wizard

  1. From the Opticks menu bar, select Tools, Wizard Builder. The Wizard Builder Window opens.

  1. On the Wizard Builder Window go to the File menu and select Open.
  1. The Open dialog box is displayed. Navigate to MyWizard.wiz, the wizard you created and saved in the Wizard Builder tutorial (Tutorial: Wizard Builder). Select MyWizard.wiz and click the Open button.
  1. MyWizard is displayed in the Wizard Builder window.

  1. On the Wizard Builder window, click the go to Execution Mode and select Batch from the drop-down list. This enables you to run the wizard completely in batch mode without interacting with Opticks.

  1. In the Wizard section of the Wizard Item Palette dock window, select the Create Export File Descriptor item. Click and drag the item onto the display area

Add Second Plug-In Item

  1. In the Exporter section of the Wizard Item Palette dock window, select the GeoTIFF Exporter item. Click and drag the item onto the display area
  1. Arrange all the items so you can clearly see them.

Change Value Item Number

The value items must have a higher execution number (lower number) than the plug-ins and desktop items they’re connected to. An item with an execution number of 1 is executed first and so on. Because value items only have a single output, they must be executed before the item(s) they are connected to. This means the value items will always have a lower numeric number than the connected item(s).

  1. To change the order of the value item displayResults from 4 to 5, select the item and click the up arrow on the Wizard Builder Toolbar. This also changes the value of the useDegrees item from 5 to 4.
  2. Make sure the wizard items are assigned the following order numbers:

1. Value-input

2. Import Data Set

3. Value-expression

4. Value-display

5. Value-useDegrees

6. Band Math

7. Value-output

8. Create Export File Descriptor

9. GeoTIFF Exporter

Save Wizard

  1. To save the wizard, go to the File menu on the Wizard Builder Window and select Save As. (Use "Save As" rather than "Save" so you don't overwrite the wizard that was created in Tutorial 9.)
  2. Save the file to the Wizards directory. Name the file BandMath.wiz. When the wizard file is saved in full Batch Mode, a .batchwizfile is also saved.

If you want the new wizard to appear in the previously created MyWizard menu structure, you will also have to save the file to the default location, which is the temp directory.

Run Wizard in Batch Mode through Command Prompt

  1. Close the Wizard Builder. Close Opticks by selecting File, Exit in the Opticks Main Application window. When the dialog is displayed that asks if you want to save the session, click No.
  2. Windows:

On Windows, go to the Start menu and select Run….

Type cmd in the Run window.

Click the OK button.

A command prompt window is displayed.

Solaris:

On Solaris, open a terminal window.

Navigate to the Opticks installation directory. On Windows, the default location is C:\Program Files\Opticks\4.0. On Solaris, the default location is /opt/Opticks/4.0.

Press the Enter key on your keyboard.

  1. At the Windows command prompt, type:

bin\opticksbatch /input:wizards\bandmath.batchwiz

On Solaris, type:

OpticksBatch -input:wizards/bandmath.batchwiz

Press the Enter key on your keyboard.

  1. The batch processing will start without a user interface and write out the GeoTIFF file to the specified folder (…/SampleData/GeoTIFF).

Summary: You should now be familiar with the procedures for using a previously saved wizard to perform batch processing in Opticks.
