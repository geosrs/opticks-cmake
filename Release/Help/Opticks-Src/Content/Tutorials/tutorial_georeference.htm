Tutorial: Georeference, Measurement Tool, Band Math, Linking Displays, Layer Information/Annotations, Export JPEG Product

Learning Objective : When finished with this tutorial, you will be able to perform manual georeferencing, measure a distance using the measurement toolbar, execute the band math algorithm, link two images, annotate the image, and export it as a JPEG product.

In this tutorial, you’ll manipulate and export data using the following functions:

  • Open a file
  • Georeference the image
  • Use the Measurement Toolbar
  • Perform band math
  • Link displays
  • Manipulate layer information/annotations
  • Export JPEG product

Lesson: The georeference process enables you to assign latitude and longitude coordinates to each data pixel based on selected ground control points (GCPs). GCPs match a point on the data set to be georeferenced with a latitude/longitude coordinate.

The Measurement Tool can be used both before and after georeferencing. If the tool is used before georeferencing, distances can only be displayed in unitless pixel lengths where one unit is equivalent to the length on a single pixel. Once georeferenced, distances will be automatically calculated and displayed in the specified units.

Band math enables you to compute various mathematical functions for single and multiple bands or a complete data cube. When performed as a band math function, a single output band is created. When performed as cube math functions, an entirely new data cube is formed. This lesson uses a standard ratio called the Normalized Difference Vegetation Index (NDVI) that is used to classify vegetation for spectral data within a scene. This ratio relies on vegetation’s low reflectance in the visible region and high reflectance in the near-IR region to separate it from other materials.

The Link function enables you to link two displays of the same spatial extent so that changes in the zoom level or pan location in one image will also affect the other.

The Session Explorer Window enables you to turn displayed layers on and off, change the order of displayed layers, delete layers, and derive or convert layers into new layers.

Annotation is an additional layer overlaying the data set and processing results layers. Objects created by using the annotation tools can be saved as an object and reused for annotation of other scenes. Examples of annotation objects include legend keys, scale bars, and possible target callouts.

The JPEG Exporter enables you to export a product as a JPEG file.

Turn Off "Automatically georeference on import" Option

Opticks has an import option that automatically georeferences data sets on import. The first thing you'll do is turn off that option.

  1. On the Opticks Main Application Window, go to the Tools menu and select Options....
  2. In the structure tree on the left side of the Options window, click the plus sign (+) next to Import. Select Raster Element.
  3. Uncheck the Automatically georeference on import checkbox.

Import Data

  1. On the Opticks menu bar, go to the File menu and select Import Data....
  2. Navigate to and select the p014r033_7t10000923_z18_spectral.re.ice.h5 data set.
  3. Click Open.
  4. Resize the data set window so it fills most of the display area.
  5. On the Display Toolbar, click the Zoom to Fit button.

Georeference Data

There are two ways to manually georeference data.

Manual Georeference Option 1

  1. Activate the Geo Toolbar by right-clicking in the toolbar area and selecting Geo.
  1. On the Geo Toolbar, select the Geo button. Select Georeference from the drop-down menu.

  1. The Georeference dialog box is displayed.

Be sure the Georeference Plug-In selected is GCP Georeference.

In the GCP List drop-down, be sure that the Corner Coordinates option is selected. These are the coordinates that automatically load from the metadata contained in the header for this Landsat data set. Notice that only three Ground Control Points are required to perform a georeference. There are five coordinates in this particular image, which will ensure a more accurate georeferencing. (More points don’t always ensure better accuracy if the selected points are inaccurate.)

In the Georeference dialog box, go to the Coordinate Type field and select Latitude/Longitude from the drop-down menu. The other options for Coordinate Type include UTM and MGRS, two other common coordinate systems. (See also More About Georeferencing.)

  1. Select OK to start the Georeferencing process. This starts the georeferencing calculations. (If the Georeference dialog box is displayed, click Close.)

Latitude/Longitude grid is now displayed over the dataset. If necessary, adjust the zoom percentage. Notice the coordinates are displayed on the status bar at the bottom right of the window as the mouse is moved from pixel to pixel.

Run Band Math

  1. To activate the General Algorithms Toolbar, go to the Opticks menu bar and select View, Toolbars, General Algorithms.
  2. On the General Algorithms Toolbar, select Band Math.

The Band Math dialog box is displayed.

Using the six bands displayed in the Bands section of the Band Math dialog, you can perform a variety of mathematical operations. For example, the formula for calculating the Normalized Difference Vegetation Index (NDVI) for a Landsat 5 scene is as follows: 

(NIR-Red)/(NIR+Red)

OR

(b4-b3)/(b4+b3).

  1. Click the parentheses button.

Double-click Band 4.

(b4) should now appear in the script line at the bottom of the Band Math dialog.

  1. Click the subtraction button.  

Double-click Band 3.

The script line should now read (b4-b3).

  1. Click the exit parentheses button. 
  2. Click the division button.
  3. Click the parentheses button.

Double-click Band 4.

  1. Click the addition button.  

Double-click Band 3.

  1. Be sure the formula is correct. It should now read:

(b4-b3)/(b4+b3)

  1. Click the Process button.
  2. Opticks will indicate if the algorithm has run successfully. After it has successfully run, select the Close button.

A new window displays the results. If necessary, go to Windows, Tile Vertically so you can see both windows.

The resulting cube has pixels scaled between -1 and 1, where the pixels with values closer to 1 (the brighter pixels) indicate highly vegetative areas.

Note that the band math algorithm removed the georeferencing, but you can add georeferencing back to the band-math-generated image.

  1. Select the new image. Click Geo, Georeference on the Geo Toolbar.

In the Georeference dialog box, select Corner Coordinates from the GCP List drop-down and click OK. In the completion dialog box, click Close.

The latitude/longitude pairings appear in both images.

Linking Displays

  1. To place the two images side-by-side, close the Session Explorer window. On the Opticks menu bar go to Windows and select Tile.
  2. Highlight each data window and click the Zoom To Fit button  on the Display Toolbar so both images are maximized within their windows.
  3. Click the Link/Unlink button  on the Toolbox Toolbar.
  1. In the Step 1 Link/Unlink dialog box, select the p014r033_7t19990923_z18_spectral.ice.h5 file as the Source Data Set. Click the Next> button.

  1. In the Step 2 Link/Unlink dialog box, go to the Available Links box. Select Views and place a check in the box next to the Source Data Set’s path name. This identifies which cube is to be linked to another data set.
  2. Set the Link Type to Automatic. This will use Mirror (pixel-based) linking if the scenes aren't georeferenced or Latitiude/Longitude linking if the scenes are georeferenced.

In the Linking box, place a checkmark in the Two-Way box. This updates changes made to either data set (the Source data set or the Destination data set) in both windows.

Click the Next> button.

  1. In the Step 3 Link/Unlink dialog box, choose the Band Math Results data set as the Destination Data Set. Click the Finish button.

  1. In the p014r033_7t19990923_z18_spectral.ice.h5 data set, zoom in on the water inlet near the center of the image. Notice how the Band Math Results Data Set zooms to the same location. Zoom out of the Band Math Results Data Set and the other cube will also update.

Annotating the Vegetation

You can control much of Opticks' functionality in the Session Explorer. You can toggle each layer on and off, derive or convert one layer type to another, delete or rename each layer, and access each layer's properties dialog.

  1. On the Opticks menu bar, go to Tools and select Session Explorer.
  2. Select the Windows tab in the Session Explorer. Expand Workspace Windows and then expand the item under it with the same name as the band-math results.
  3. Right-click on the item corresponding to the band-math results window and select New Layer, Annotation. This adds a new annotation layer and displays and activates the Annotation Toolbar.

  1. Maximize the Band Math Results Window. Pan and zoom to the inlet in the upper right area of the image.
  1. Select the New Annotation Object drop-down on the Annotation Toolbar and select the arrow.

  1. Click on the image and drag from start to point. Try to position the arrow so it is pointing at a highly vegetative area (a group of bright pixels).

  1. The arrow should have a dotted rectangular box around it. Color this line green, using the Annotation Line Color button  on the Annotation Toolbar.

Make the line width thicker using the Annotation Line Width button.

  1. Go to the Add New Object button and select Text Box.

Use you mouse to select an area on the image to create this text box. When you release the mouse, the enter text dialog box is displayed.

Type the name Vegetation into the text box and click OK. The text box should appear on the image.

  1. Change the color of the text to green (to match the arrow) using the Annotation Text Color button.
  2. Change the size of the text to 72 and move it so that it's near the arrow.

Export Data

To export a product showing the Latitude/Longitude grid, clean up the image by removing the Ground Control Points and the crosshairs from view. You will do this using the Session Explorer Window.

  1. Hide the GCP points by right-clicking on the Corner layer and unchecking Displayed.
  2. Right-click on the Window item (in bold) and toggle off the crosshair.
  1. To finish the image before you export it, arrange the image in the middle of the screen by clicking the Zoom to Fit button  on the Display Toolbar. This centers the image in the screen.

Click the Zoom Out button  once so the lat/long grid is not covered by the classification markings. Note that the annotation text box will be very small in this view.

  1. Right-click on the Window item and select Export.
  2. In the Exporter panel at the bottom, change AVI Product Exporter to JPEG View Exporter. Enter the filename MySample-Geo.jpg in the File Name box and click the Save button.
  3. Close all files and views.

Summary/Review: You should now be able to perform georeferencing, execute the band math algorithm to find materials within a data image, annotate the image, and export it as a JPEG product by using many of the common core tools in Opticks.
