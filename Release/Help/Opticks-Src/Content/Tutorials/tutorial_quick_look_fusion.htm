Tutorial: Quick-Look Data Fusion Plug-In

Learning Objective: When finished with this tutorial, you will be able to examine construction changes that have taken place over a site during an 18-month period using the Quick-Look Data Fusion Plug-In.

In this tutorial, you’ll perform the following data fusion functions:

  • Start the Quick-Look Data Fusion Plug-In
  • Load files
  • Use Tie Points
  • Designate Area of Interests
  • Manipulate fused views

Lesson: The Opticks Quick-Look Data Fusion Plug-In is a tool used to fuse data from various sensors even if they were taken at different angles, skewed, rotated, or translated.

Start

  1. Import the following data sets from the sample files: ir_bushehr_06jun02_ps.tif and ir_bushehr_28feb04_ps.tif.
  2. Select the image ir_bushehr_06jun02_ps.tif. On the Display Toolbar, click the Zoom to Fit button.
  3. On the Geo Toolbar, select Georeference. In the Georeference dialog box, accept the defaults and click the OK button.
  4. Repeat Steps 2 and 3 for the image ir_bushehr_28feb04_ps.tif.
  5. Go to the Window menu and select Tile Vertically.
  6. Zoom in so that the round building and the rectangular building set on a diagonal are easily visible.

  1. Be sure the General Algorithms Toolbar is displayed. Click the General Algorithms button and select Data Fusion to start the Quick-Look Data Fusion Plug-In.

The Quick-Look Data Fusion wizard is displayed.

  1. In the Quick-Look Data Fusion wizard, both the data sets you've already loaded are displayed.

  1. Highlight the ir_bushehr_06jun02_ps.tif data set in the Primary Data Set list and the ir_bushehr_28feb04_ps.tif data set in the Secondary Data Set list.

Note: We recommend that you select the higher resolution data set as the Secondary Data set. The algorithm will use the higher resolution data set to produce a better product when fusing with the other data set. A warning notice is displayed if the Primary Data set has a higher resolution than the Secondary Data set and the higher resolution data is then degraded to the resolution of the other data set.

  1. (Optional.) Opticks provides you with the option of bypassing the Create Tie Points step. To bypass this step, go to the Algorithm Options box and select the Bypass Tie Point Step checkbox.

Creating a tie point increases Data Fusion accuracy. We recommend that you bypass this step only if you are certain that the georeferencing information in the file is highly accurate (typically RPC georeferencing or higher order GCP georeferencing) and/or you have no context in which to find a tie point.

You can also bypass the tie point step every time you run the Quick-Look Data Fusion Plug-In by checking the Save Settings for future sessions checkbox. This checkbox applies only to the tie point step.

For this tutorial, leave both the Bypass Tie Point Step checkbox and the Save Settings checkbox unchecked.

  1. Click the Next> button. The Quick-Look Data Fusion wizard now displays the window for creating tie point. You must create at least one tie point.

Create a Tie Point

After the data sets are imported, you must create at least one tie point. A tie point represents the same location on two different images. You can use multiple tie points to correct inconsistencies in the geolocation between the data sets, but you need only one tie point to run the tool.

You might want to tile the images.

  2. Select the upper corner of the rectangular building in both images as shown below.

  1. Each pixel is automatically added to the appropriate list.

Note: You can also use the GCP Editor to make changes to the GCP. The Data Fusion wizard automatically accepts the changes.

  1. Go back to the Data Fusion dialog box.

Highlight the two GCPs and click the Derive button to create a tie point.

  1. The GCP markers disappear and a Tie Point appears.

  1. In the Quick-Look Data Fusion dialog box, select the Tie Point. (Be sure it's highlighted.)

  1. The next window enables you to select the Annotation Layers and AOI Layers that are available for fusion. We are not fusing any layers so just click the Next> button.

Select an AOI and Output Views

After you create a tie point, you can define an AOI and view a fused version of the AOI using either the Flicker/Blend or Side-by-Side views.

  1. Move the dialog box aside again. If the Session Explorer isn't already open, go to Tools, Session Explorer to open the Session Explorer window.
  2. Zoom out on the primary image to somewhere between 60% and 70%.
  3. Right-click on the primary image name. Select New Layer, AOI to create a new area of interest.
  4. Go to the AOI Toolbar. Under the AOI Pixel Selection Tool, select the rectangle.

  1. In the primary image, select a rectangular region to fuse. The region should be an area that both scenes contain.

  1. Go back to the dialog box. Click the Next> button.
  2. On the primary image, draw a second rectangle that's slightly larger than the first.

  1. Select the output view: Flicker/Blend.
  2. Select the Run Flicker Tools when done check box to run the flicker tools that manipulate the fused flicker view.

  1. Click the Execute button to run the data fusion algorithm without closing the wizard. This gives you the option to run the algorithm on a different AOI on the same two images without restarting the wizard.
  2. Click the Finish button to run the data fusion algorithm and close the wizard.

Note: If both of the data sets are not already georeferenced, there will be a prompt before the algorithm begins. Also, if the Primary Data Set has a higher resolution input, a warning is displayed.

Flicker/Blend View

After you run the data fusion algorithm, a flicker/blend view is displayed.

The flicker tools enable you to manipulate the flicker/blend view to control the transparency, x/y translation of the secondary image chip, automated/manual flicker, and flicker rate.

If the flicker tools aren't automatically displayed, go to the Tools menu and select Flicker Controls.

Note: The flicker controls window automatically docks at the bottom of the screen, but can be moved or undocked.

To use the automated flicker feature:

  1. Set desired flicker rate (frames per second).
  2. Click the Start button. The rate can be adjusted during flicker.
  3. Click the Stop button to end flicker.

To use the manual flicker feature:

  1. Set the flicker rate to 0; click the Stop button.
  2. Make sure the Secondary Data Set is not completely transparent.
  3. Click the Toggle button to toggle between images.

To use the transparency slide:

Click the button on the transparency slide to modify the transparency from transparent to opaque. This is helpful when trying to use the slide bars to line up the two images.

To use the x, y sliders:

Click the slider bar on either the x or y slide make adjustments to the secondary image in relation to the primary image. You can make finer adjustments to the image by placing the cursor over the slider bars and using the center scroll button.

Manipulating Side-by-Side View

To use the side-by-side view:

  1. Close the flicker window and the controls. Re-run the Quick-Look Data Fusion Plug-In. Select the primary and secondary data cubes, then add GCPs to each image and create a tie point. This time, use the side-by-side option.
  2. Use the basic image manipulation tools to zoom, pan and rotate as desired. Both windows zoom to the same area if done correctly.

Note: The side-by-side view is only available when the Quick-Look Data Fusion wizard is open.

  1. Close all files.
