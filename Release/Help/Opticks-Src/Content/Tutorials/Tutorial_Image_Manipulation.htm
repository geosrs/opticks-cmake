Tutorial: Image Manipulation and Export

Learning Objective: When finished with this tutorial, you will be familiar with data importing, basic image manipulation, and data exporting.

In this tutorial, you’ll manipulate and export data in the following sequence:

  • Open a file
  • Image manipulation: pan, zoom, flip, rotate, reset
  • Histogram
  • Brightness/contrast
  • Export a data cube in ICE Exporter
  • Image Chipping Window
  • Close the file

Lesson: Opticks is not designed to be an imagery tool, but it does have some basic image manipulation functions. Within Opticks, you can rotate, zoom in and out, and pan around the data display. 

The Histogram Window displays a graph of the number of pixels in the image display for each intensity brightness levels. The intensity is displayed on the x-axis and the number of pixels of each intensity is displayed on the y-axis. The Histogram Window enables you to manipulate data and to modify display characteristics such as contrast and algorithm thresholds.

Brightness/Contrast enables you to manipulate the brightness and contrast for individual bands or an entire RGB display.

The ICE Exporter enables you to select the parameters for data format/scaling, cropping, and sampling, and to produce a corresponding ICE-formatted data cube. A ICE data set is created whenever the data is saved. The format contains header information obtained from the original sensor, data, and data statistics and optimizations calculated. Original sensor data cannot be overwritten, so Opticks uses the ICE Format to save processing results.

The Image Chipping Window enables you to create an image that is a subset of a larger image.

Import Data

  1. On the Opticks menu bar, select File, Import Data.

  1. Navigate to and select the file f970620t01p02_r03_sc01.a.bip.
  2. Be sure either the Auto Importer or the GeoTIFF Importer is selected from the Importer drop-down. Click the Open button.

  1. When the import is complete, click the Close button on the Importer dialog box.
  2. Drag the lower right corner of the image so it fills the work area.

Image Manipulation

  1. On the Opticks menu bar, select View, Zoom, Rectangle.

OR

Click the Zoom to Rectangle button  on the Display Toolbar.

Using the left mouse button, click and drag an area in the data window that is smaller than the spatial area of the data. Release the mouse button. This zooms the display to the selected area.

  1. Open a separate overview window by going to the Opticks menu bar and selecting Tools, Overview Window.

A rectangle appears in the overview window outlining the zoomed-in area displayed in the data window. You can see the location of the zoomed-in window within the full-size image.

Note: Some images have ground control points (GCPs) that allow for image mapping to geo-spatial coordinates.

  1. Close the overview window by clicking the Close control “X” in the upper right corner of the window.

To restore the normal view of the data cube, elect the Zoom to Fit button  on the Display Toolbar.

OR

On the Opticks menu bar, select View, Zoom To Fit.

The total spatial area of the data is sized to fit within the current size of the data window. Resize the data window to see how Zoom to Fit automatically increases or decreases the zoom level to fit the window size.

  1. On the Display Toolbar, select the Zoom In button  and Zoom Out button  to change zoom percentage incrementally. This affects the zoom of the active image.

Note: The mouse scroll wheel and middle button can also control zoom.

  1. On the Display Toolbar, select the Zoom In on Mouse button  and Zoom Out on Mouse button  to zoom in and out based on the location of the mouse pointer. Activate the zoom by moving the mouse pointer to the location you want to enlarge and clicking the left mouse button. 
  2. On the Display Toolbar, select the Pan button .

OR

On the Opticks menu bar, select View, Pan.

Hold down the left mouse button and drag to move the image within the data display.

  1. On the Display Toolbar, select the Flip Vertically button.  This flips the scene 180 degrees on its horizontal-axis.
  2. On the Display Toolbar, select the Free Rotate button  and move the mouse around the screen while holding down the left mouse button. This rotates the scene around its center.

Note: All of the rotation tools can also be accessed through the View, Rotate menu on the Opticks menu bar.

  1. On the Display Toolbar, select the Reset button  to reset the image to the original data cube orientation.

Histogram Window

A histogram is a two-dimensional representation of data. It shows the distribution of pixel intensities. The x-axis represents the data value of a pixel in that band. The y-axis represents the number of pixels with a data value of the corresponding x-axis value.

  1. To open the Histogram Window, go to the Opticks menu bar and select Tools, Histogram Window.

OR

Click the Histogram Window button on the Toolbox Toolbar.

  1. The Histogram Window is displayed.

By default, the Histogram Window shows a histogram of the band that is displayed in the data window. If the data window is in RGB mode, the Histogram Window shows a histogram in either the red, green, or blue band. If the data window is in grayscale mode, the Histogram Window shows a histogram of the currently displayed grayscale band.

Each tab on the bottom of the Histogram Window represents a different layer or band. The active layer has a white tab; inactive layers have gray tabs. To activate a different layer, click on the tab.

The red overlay on the Histogram Window represents what is being displayed in the data window. The graphic above illustrates that only pixels with a value of between approximately 8 (left edge) and 19 (right edge) are being displayed in the red band on the data window. Any pixels not meeting those criteria will be displayed but are set to the maximum or minimum value. If this were the blue band, the overlay would be blue. If it were the green band, the overlay would be green, and if it were the grayscale band, the overlay would be gray.

  1. To adjust the overlay size, click near the edge of one side of the overlay and drag the edge. The side of the overlay moves with the mouse. Notice the change in the display.

The purpose of moving the overlay is to adjust the contrast of the data display. Generally, making the overlay very large will decrease the contrast, while making it small will increase the contrast. Experiment by changing the overlay size in the Histogram Window, observing the data window. Use the different tabs (Red, Green, and Blue).

  1. Right-click in the Histogram Window to display the Histogram Window menu. This menu enables you to manipulate the data within the Histogram Window.

The Save function on the Histogram Window menu saves the current histogram to an ASCII text file with the extension hst. This allows the histogram to be imported into other tools such as spreadsheets for more detailed data analysis.

  1. The Mouse Mode function on the Histogram Window menu enables you to control the functionality of the mouse within the Histogram Window. There are four mouse functions available: Set Value, Zoom, Pan, and Annotation. The Set Value function (the default setting) changes the overlay for the spectral band in the Histogram Window.

On the Histogram Window menu, select Mouse Mode, Zoom to zoom in on a section of the histogram. Click slightly in the middle of the histogram color overlay. Hold the left mouse button down while moving the mouse to the right of the overlay. Release the mouse button to see a zoomed-in histogram.

  1. On the Histogram Window menu, select Mouse Mode, Pan. The Pan function allows you to change the displayed region of a zoomed histogram. Click and hold the left mouse button moving the mouse left, right, up, or down to view other areas of the zoomed histogram.
  2. To reset the histogram, on the Histogram Window menu, select Rescale Axes.
  3. Close the Histogram Window (click the X in the upper right corner).

Note: Mouse Mode does not automatically change back to Set Value after Rescale Axes has been chosen. Zoom mode will still be in effect.

Brightness and Contrast Toolbar

Using the Brightness Toolbar is similar to adjusting the overlay in the Histogram Window. The toolbar allows you to adjust the brightness and contrast of a selected band incrementally. 

  1. To display the Brightness Toolbar, go to the Opticks menu bar and select View, Toolbars, Brightness.
  2. Adjust the Brightness slider on the left of the toolbar and notice the changes in the displayed data.

  1. Adjust the Contrast slider and observe the changes in the data.

  1. Open the Histogram Window again by clicking the Histogram Window button  on the Toolbox Toolbar. Adjust the brightness and contrast sliders again and observe the changes in the overlay in the Histogram Window. Experiment with adjusting the brightness and contrast sliders as well as the overlay in the Histogram Window to get a feel for their relationship.
  2. From the Brightness Toolbar, you can also select which displayed band is affected by the slider adjustments. Click the drop-down menu on the right of the toolbar and select Band 1-Gray to change the data to grayscale. This particular image will go black in grayscale. Go back to RGB by selecting the red, blue, or green band. Experiment with the brightness and contrast.

  1. The brightness/contrast adjustments affect only the band selected. Select RGB from the drop-down menu to adjust all three RGB bands at once.

Click the Reset button  on the Brightness Toolbar to reset all changes to their default values.

  1. Adjust the Brightness down to about 32 and the Contrast up to about 69. Doing so makes the vegetative areas appear more green. The graphic below shows a before (top) and after (bottom) view (assuming a true color composite image was loaded and displayed with R3G2B1).

Data Export

  1. On the Opticks menu bar, select File, Export.

The Export dialog is displayed.

  1. Leave the default file name and exporter.

  1. Choose a location for the file.
  2. Click the Save button.

Image Chipping Window – Export

  1. On the Opticks menu bar, select Tools, Image Chipping Window.

OR

Click the Image Chipping Window button  on the Toolbox Toolbar.

  1. The Create Image Chip dialog box is displayed.

  1. Select the Export to file radio button. With the mouse cursor on the image, hold down the left mouse button and drag. A white box appears around the area you selected.
  1. Release the left mouse button. The selected area will appear in the Create Image Chip display window. Under Chipping Mode, select Export to file.

  1. Click OK. A message window may appear stating the created chip will not contain a GCP List. Click the Continue button.

  1. An Export window is displayed. For this tutorial, keep the default settings. Click the Save button.

  1. An ICE Format Exporter message dialog box appears stating that ICE export is complete. Click the Close button.

Close File

  1. On the Opticks menu bar, click on File, Close.
  2. In the Confirm Close dialog box, click Yes.

Reload Chipped Image

  1. On the Opticks menu bar, select File, Import Data. Select the f970620t01p02_r03_sc01.re.ice.h5 file. Click the Open button.
  1. The chipped image will load in the display window.

  1. Close all files.

Summary/Review: You should now be able to import, manipulate, and export an image using many of the common core tools in Opticks including pan, zoom, histogram, and spectrum window tools.
