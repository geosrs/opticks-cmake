Tutorials

Welcome to the Opticks Core tutorials! These tutorials focus on the Opticks Studio.

The tutorials provide you with example scenarios that contain common Opticks core functionality.

When finished with these tutorials, you will be able to perform several general tasks, designated as Learning Objective, and several specific tasks, listed as Samples of Behavior, in each scenario.

Intended audience

Learning level

The intended audience of these tutorials is anyone who uses the Opticks software, particularly those who have not used it before. This includes intelligence analysts, field users/operators, and their respective supervisors.

Mission criticality

The Opticks core is the basis for using any of the plug-ins. Knowing the purpose of a particular plug-in does not necessarily enable you to use the plug-in, unless you have the knowledge of using the core functionality. So knowing the core functionality is critical to any mission that uses the Opticks software.

Requirements

Prerequisites

You should have fundamental knowledge of operating a PC or Solaris workstation. You should also have had general training on processing analysis and interpretation of data, as well as training for the function you are expected to perform (job training).

Plug-ins, Toolbars used

Following the exercises in the tutorials requires no separate plug-ins, only those that are considered part of the core functionality and are installed during the core Opticks installation.
