Tutorial: Wizard Builder

Learning Objective: When finished with this tutorial, you will be familiar with using Wizard Builder to perform automated and semi-automated processing in Opticks.

In this tutorial, you will perform these functions:

  • Start Wizard Builder
  • Add wizard items
  • Connect algorithms and processes
  • Change Value Item number
  • Add menu location
  • Save wizard

Lesson: The Wizard Builder allows you to link algorithms and processes together to perform batch processing in Opticks. This gives you the ability to automate, either partially or fully, repetitive tasks. The Wizard Builder provides an easy-to-use graphical interface that can be learned quickly.

One thing to remember when using Wizard Builder is that it works on the concept of data flow instead of process flow. This means that the steps in the Wizard Builder are not organized according to order of operations, but according to what state the data is in.

There are several types of items that can be inserted into a wizard. "Wizard" items obtain values from you when executed or handle the inputs and outputs of Opticks. Plug-in items contain the algorithms or processes that are executed in Opticks. Value items are simple interfaces that can become part of an algorithm or process. This value can be a Boolean (true or false), string, or number.

Each of these item types appears as a block within the Wizard Builder window. The items are connected together with visible lines and assigned an order number. The algorithms and processes are executed according to the order number assigned them, regardless of how they are connected with the visible lines.

Toolbars used in this tutorial:

Wizard Builder Toolbar

To access the toolbar, go to the View drop-down menu in the Wizard Builder and select Toolbars. The list of toolbars is displayed. Active toolbars have a checkmark in front of them. Select the toolbar name to toggle it on or off.

Start

  1. From the Opticks menu bar, select Tools, Wizard Builder.... The Wizard Builder window is displayed.

Add a Wizard Item

  1. In the Wizard section of the Wizard Item Palette doc window, select the Import Data Set item. Click and drag the item onto the display area.

The wizard item is displayed at the drop location in the Wizard Builder display area.

Add Plug-in Item

  1. In the Algorithm section of the Wizard Item Palette doc window, select Band Math. Click and drag the item onto the display area.

The plug-in item is displayed at the drop location in the Wizard Builder display area.

  1. Be sure the Band Math icon is still selected. Change the Execution Mode to Batch in the Wizard Item Properties dock window.

The selected graphic is outlined in red. This indicates that the item will run without user interaction. The graphic also expands because more information is needed to run it in batch mode.

Add Value Item

  1. In the Value section of the Wizard Item Palette dock window, click and drag the "string" item onto the display area. The Wizard Value Item dialog box is displayed.

  1. In the Name field, enter: expression
  2. In the Type list, be sure that "string" is selected.
  3. In the Value field, enter (b4-b3)/(b4+b3).

  1. Click the OK button.

The Value-expression item is displayed at the drop location in the Wizard Builder display area.

  1. In the Value section of the Wizard Item Palette dock window, click and drag the "bool" item onto the display area. The Wizard Value Item dialog box is displayed .
  2. In the Name field, enter: display
  1. In the Type list, be sure that "bool" is selected.

About Wizard Items

Here is a brief explanation of the meaning of the wizard item icons. If you want a more complete explanation, see Wizard Items.

The number at the top of each icon represents the order the algorithms or processes will be run.

The colored dots on the left and right side of each icon represent the inputs and outputs of each algorithm or process. If you hold the mouse pointer over the top of the dot, the name and type of the input or output will be displayed. The type of output must match the type of input for each connected pair of algorithms or processes. The Wizard Builder will not allow two unlike types to be connected.

Some input and output values support multiple types. These dots are represented with a down arrow next to the dot. Clicking the down arrow cycles the current type through all of its valid types. When you hold the mouse pointer over the top of the dot, the current type and a list of valid types are displayed along with the original type required by the item.

For example, if you wanted to attach the Data Set output of the Import Data Set item to the Data Element input on the Band Math plug-in, the Wizard Builder would not let you because the types are Data Element and Raster Element, and don't match exactly. However, the Band Math input supports multiple types, including Data Element. So, to connect these nodes, simply click the arrow next to the Band Math node until its type matches the type of the Import Data Set node.

To connect inputs and outputs, select the output by clicking your (left) mouse button and dragging the mouse to the input. Release the mouse button.

Change Value Item Number

  1. Select the Value-expression item.
  2. Click the Down button on the Wizard Builder Toolbar. Notice that execution number changes to 1.
  3. Click the Up button. The execution number returns to 2.

The value items must have a earlier execution number (lower number) than all of the connected plug-ins and desktop items. An item with an execution number of 1 is executed first and so on. Since value items only have a single output they must be executed before the item(s) they are connected to. This means the value items will always have a lower numeric number than the connected item(s).

  1. Click the Up button again twice. The execution number does not change; the Wizard Builder will not allow the number to be 4.

Make sure items are numbered as follows:

1. Import Data Set

2. Value-expression

3. Value-display

4. Band Math

Add Menu Location

You can indicate the name and location of created Wizards by using the Menu Location field in the Wizard Properties dock window. The name of the Wizard appears as it was typed in the Menu location box.

The way the name is formatted dictates where the Wizards will appear in the Opticks menu structure:

  • Bracket characters ([,]) specify a toolbar.
  • A slash (/) indicates submenu levels.
  • An ampersand (&) indicates an existing menu location.

The toolbar name must come first in the string. A slash immediately following the toolbar name specifies that the submenus and command should be added to the default toolbar menu, which has the same name as the toolbar. If a slash does not follow the toolbar name, the menus and command are added directly to the toolbar. If the string does not include a toolbar name, the menus and command are added to the main menu bar. The string cannot end with a slash, and the name after the last slash indicates the command name.

Below are a four examples of menu locations and how to type them in the Menu Location box.

Example 1: Adding the Wizard to the end of the top level menu.

Type MyWizard in the Menu Location box.

Example 2: Adding the Wizard to a new toolbar.

Type [Wizards]/MyWizard in the Menu Location box.

Example 3: Adding to an existing menu location.

Type &Help/MyWizard in the Menu Location box.

Example 4: Adding to an existing toolbar.

Type [Geo]/MyWizard in the Menu Location box.

Note: We recommend that you save all custom wizards on a custom toolbar, as shown in Example 2 above.

Note that the Wizard location does not go into effect until you save the wizard and update the wizard list. (On the Opticks window, go to Tools, Update Wizard List.)

Save Wizard

  1. Save the wizard by selecting File, Save. Save the file (MyWizard.wiz) to the default location. If you do not save the file to the default location, Opticks will not be able to add the wizard to your Opticks menu structure.
  2. Test the wizard by clicking the Execute button in the Wizard Properties dock window. The wizard runs automatically. As part of running, the wizard will open the Opticks import window. Select the AST_LIB_00305042006183231_05082006132132_TIR_Swath.tif file.

  1. Close the Wizard Builder by selecting File, Close. The display window will show two data images.

  1. Add the wizard to a menu location by selecting Tools, Update Wizard List from the main Opticks menu bar.
  2. Close all files.
