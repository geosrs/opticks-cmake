GCP Editor

A Ground Control Point (GCP) can be entered using a keyboard, mouse, or a combination of both.

You can then use the GCP Editor to view and edit the GCP data.

To open the GCP editor:

On the Opticks Main Application window, go to the Tools menu and select GCP Editor.

OR

On the Geo Toolbar, click the GCP Editor button.

The GCP Editor dialog box is displayed.

To edit GCP data:

  1. Highlight the row with the GCP data you want to edit.
  2. Click inside the cell to activate it.
  3. Make the changes.
  4. Click the Apply button.

To add a new GCP:

  1. Click the New button.
  2. A new row is added to the table. Edit the data by highlighting the row and then clicking in the cell you want to edit.
  3. Click the Apply button.

To delete a GCP:

  1. Highlight the row of the GCP you want to delete.
  2. Click the Delete button.
  3. The GCP row is deleted and the numbers are updated if necessary.
  4. Click the Apply button.

To change the Coordinate Format:

Use the Coordinate Format radio buttons to select the format. Available formats are:

  • Latitude/Longitude. This is the default.
  • UTM Coordinates. Universal Transverse Mercator Coordinates.
  • MGRS. Military Grid Reference System.

When you change the coordinate format, the table automatically switches to a display for the chosen format.

To view a different GCP list:

Select the GCP List from the GCP List drop-down.

To view the GCP properties:

Click the Properties... button. See GCP Layer Properties.

To use Auto Apply:

When the Auto Apply checkbox is checked, you don’t have to press the Apply button to apply changes before closing the window.
