Georeference

The georeference process allows you to assign coordinates to each spectral data pixel based on one of several methods. Opticks supplies two methods: Ground Control Points (GCPs) and Rapid Positioning Coefficients (RPCs). RPC is used by some types of NITF data. No additional options are needed. Other methods may be supplied by plug-ins. They can be used through this interface.

GCPs match a point on the data set to be georeferenced with a latitude/longitude, Universal Transverse Mercator (UTM), or Military Grid Reference System (MGRS) coordinate. You can enter the coordinate for the GCP manually or select it (using your mouse) from a previously georeferenced data set or map.

The georeference algorithm calculates a coordinate grid with GCPs entered into a mathematical polynomial fit equation. This method tends to stretch the coordinates to fit the GCPs, possibly introducing some level of error. This error is known as the root mean square (RMS) error and it defines the level of accuracy for the georeference calculations. The RMS error can be viewed with the GCP Editor after georeferencing.

After the georeference transformation is complete, the new coordinates are interpolated across the spectral data pixels. This means that every mouse cursor position over the data set will correspond to a precise coordinate value. As you move the mouse over a pixel, the new coordinates are displayed in the status bar at the bottom of the Opticks Main Application window. These coordinates are in latitude/longitude, UTM, or MGRS. Georeference must be executed each time the coordinate type is changed.

To assign coordinates to data set pixels:

  1. Create GCPs using the GCP Editor.
  2. On the Geo Toolbar, click the Geo button to display the drop-down menu. Select Georeference.

The Georeference dialog box is displayed. The list of Georeference Plug-Ins contains all georeferencing plug-ins that can be used on the current data set.

  1. Select the Georeference Plug-In from the list of plug-ins. The contents of this list depends on what plug-ins are installed and what plug-ins can operate on the current data set. The plug-in that you select determines the other data that you need to enter.

If you select the GCP plug-in, continue with Steps 4 through 6.

If you select the IGM plug-in, you may need to select a file. If IGM is already loaded, you don't need to select a file.

If you select the RPC plug-in, no further input is needed. Click OK.

If you select another plug-in, see the help for the plug-in you select.

  1. If desired, enter a new name for the results file in the Results Name field.
  2. Use the drop-down list to select the Coordinate Type.
  3. Click OK.

When you move your mouse over the pixels, the new coordinates for the destination data are displayed in the status bar at the bottom of the Opticks Main Application window.
