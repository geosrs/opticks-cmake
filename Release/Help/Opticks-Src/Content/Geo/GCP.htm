GCP

A Ground Control Point (GCP) is a set of two coordinates: a pixel coordinate and either a latitude/longitude, UTMUniversal Transverse Mercator, or MGRSMilitary Grid Reference System coordinate. The pixel coordinate is flagged on the data window. (The default marker is a green “X” but that can be changed using the GCP Layer Properties.)

The pixel coordinate is located on the data set to be georeferenced. The latitude/longitude, UTM, or MGRS coordinate is located on a control data set or map. Either coordinate can be inserted manually using the GCP Editor.
