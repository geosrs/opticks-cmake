Change the Scale Bar Unit System

When you create a scale bar in an annotation layer, by default, the scale bar units are measured in meters if the image has been georeferenced and in pixels if the image has not been georeferenced. You can change the units.

To change the units:

  1. After you create a scale bar annotation object, go to the Session Explorer. Right click on the annotation layer and select Properties.
  2. In the Annotation Properties window, go to Unit System. In the Unit System drop-down list, select the new measure units. The options are meters/kilometers, feet/kilofeet, and feet/miles.
