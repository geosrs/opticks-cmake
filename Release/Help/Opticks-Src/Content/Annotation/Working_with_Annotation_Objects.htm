Working with Annotation Objects

When working with annotation objects, be sure the Annotation layer is active. See Activate a Layer.

To create an Annotation Object:

  1. On the Annotation Toolbar, use the New Annotation Object drop-down list to select the type of object. For more information about the available objects, see Annotation Objects.
  2. Use your mouse to draw the box that bounds the object.

To resize the object:

  1. Select the object box. Sizing handles (small boxes) appear at the corners and on each side of the bounding box.
  2. Hold your mouse over the handle you want to use for resizing. Hold down the (left) mouse button and move the handle to the new position.
  3. Release the mouse button.

To move the text box:

  1. Select the object box.
  2. Move your mouse inside the object box. Hold down the (left) mouse button and move the object to the new position.
  3. Release the mouse button.

To change the line size:

  1. Select the object.
  2. On the Annotation Toolbar, use the annotation line width list to select the line width.

To change the line color:

  1. Select the object.
  2. On the Annotation Toolbar, use the Annotation Line Color box to select the new color. For more information on working with color, see Edit Color.

To change the object fill color:

  1. Select the object.
  2. On the Annotation Toolbar, use the Annotation Fill Color box to select the new color. For more information on working with color, see Edit Color.
