Special Formatting for Annotation Text

You can use special formatting to display metadata values or classification markings in annotation text boxes.

Metadata Fields

Metadata fields can be included by specifying a metadata field using $M(specification) where specification is:

  • A metadata path that will be loaded from a default location
  • //dataset_name//path where dataset_name is the name of the element
  • //[view_object_name]//path where view_object_name is the name of a view object in a product view

Examples

$M(NITF/File Header/OName)

$M(//C:/Data/mydata.ntf-I1//NITF/File Header/ONAME)

$M(//[View 1]//NITF/File Header/ONAME)

Other Special Replacements

Other special replacements are specified with $S(specification) where specification uses the same rules as $M() but instead of a metadata path, it uses a variable name. The only variable currently supported is:

  • CLASSIFICATION. Replace with the classification string of the specified data set.
