Annotation Drop-Down Menu

The Annotation Drop-Down Menu allows you to position the objects on the annotation layer.

When working with annotation objects, be sure the Annotation layer is active. See Activate a Layer.

To access the drop-down menu:

On the Annotation Toolbar, click the Annotation button.

To select a single object:

Click on the object with your mouse.

To select multiple objects:

Hold down the Shift key and click on each object with your mouse button.

Group

Allows you to Group or Ungroup the selected objects.

Order

Allows you to move the selected object to the Front or Back.

Nudge

Allows you to move the selected object Up, Down, Left, or Right. You can also nudge objects using the Ctrl key plus the arrow keys on your keyboard.

Align & Distribute

Allows you to align the selected objects: Right, Center (vertical center), Left, Top, Middle (horizontal middle), or Bottom.

Also allows you to distribute the selected object Horizontally or Vertically. This creates even spacing between the object centers.

Select All

Selects all the annotation objects in the current Annotation Layer. You can also select all objects using Ctrl + A.

Properties

Accesses the Annotation Layer Properties. See Annotation Layer Properties.
