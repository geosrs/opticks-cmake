Annotation Layer

The annotation layer on an image or a product window enables you to define items of interest. Using the Annotation Toolbar, you can create and modify objects, lines, and text on the image.

To begin working with the annotation layer, you must activate the layer. See Activate a Layer.

If the image does not have an annotation layer, you can create one. See Create a New Layer.
