Working with Annotation Text Boxes

When working with annotation text boxes, be sure the Annotation layer is active. See Activate a Layer.

Note that special formatting can be used to display metadata values or classification markings. See Special Formatting for Annotation Text.

To create an Annotation Text Box:

  1. On the Annotation Toolbar, click the New Annotation Object button.
  2. In the Annotation Objects grid, navigate to and select the text object.
  3. Click the place on the window where you want the text to appear.
  4. The Enter text dialog is displayed. Enter the annotation text and click OK.

To move the text box:

  1. Select the text box.
  2. Move your mouse inside the text box. Hold down the (left) mouse button and move the text box to the new position.
  3. Release the mouse button.

To change the font and font size:

  1. Select the text box.
  2. On the Annotation Toolbar, use the font drop-down list to select the font.
  3. On the Annotation Toolbar, use the font size drop-down list to select the font size.

To change the font color:

  1. Select the text box.
  2. On the Annotation Toolbar, use the Annotation Text Color box to select the new color. For more information on working with color, see Edit Color.

To make the text bold or italic or underlined:

  1. Select the text box.
  2. On the Annotation Toolbar, click the Bold button , the Italic button , or the underline button .
