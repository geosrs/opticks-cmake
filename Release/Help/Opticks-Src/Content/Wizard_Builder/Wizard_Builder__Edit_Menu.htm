Wizard Builder: Edit Menu

The following menu items are available from the Edit Menu on the Wizard Builder window:

Menu Item

What It Does

Shortcut Keys

Toolbar Button

Cut

Cuts the selected item(s) and stores it in the Clipboard. Connections are included when connected items are selected. Connections from a selected item to a non-selected item are not included.

Ctrl + X

Copy

Copies the selected item(s) to the Clipboard. Connections are included when connected items are selected. Connections from a selected item to a non-selected item are not included.

Ctrl + C

Paste

Pastes the item(s) from the Clipboard to the wizard canvas. Connections are included when connected items are selected. Connections from a selected item to a non-selected item are not included.

Ctrl + V
