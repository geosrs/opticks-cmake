Wizard Builder Toolbar

Button

What it does

Creates a new, empty wizard in the current Wizard Builder window. If you have an unsaved wizard open, the application displays a dialog box asking if you want to save the wizard.

Opens an existing wizard.

Saves the current wizard.

Prints the items as represented in the wizard canvas.

Cuts the selected item(s) and stores it in the Clipboard. Works across multiple Wizard Builder windows. Connections are included when connected items are selected. Connections from a selected item to a non-selected item are not included.

Copies the selected item(s) to the Clipboard. Works across multiple Wizard Builder windows. Connections are included when connected items are selected. Connections from a selected item to a non-selected item are not included.

Pastes item(s) from the Clipboard to the wizard canvas. Works across multiple Wizard Builder windows. Connections are included when connected items are selected. Connections from a selected item to a non-selected item are not included.

Zoom in.
Zoom out.
Zoom to fit. Displays the data area so that it fits within the current window size.

Edits the selected Value item. See Add/Edit a Value Item.

Deletes selected item(s) from the wizard.

Increases the execution order number of the selected wizard item, meaning the item is executed later in the process. The order is displayed in the wizard item rectangle. See Wizard Items.

Decreases the execution order number of the selected wizard item, meaning the item is executed earlier in the process. The order is displayed in the wizard item rectangle. See Wizard Items.
