Using the Wizard Builder

The Wizard Builder allows you to create, edit, save, and execute custom wizards of actions and processes you commonly perform in Opticks. Automating these procedures saves time and reduces potential error.

To open the Wizard Builder:

On the Opticks Main Application window, go to the Tools menu and select Wizard Builder....

The Wizard Builder window is displayed. The default wizard loaded into the Wizard Builder is empty, untitled, and ready for editing.

The Wizard Builder toolbar provides access to the common tasks used to create a wizard. For more information on this toolbar, see Wizard Builder Toolbar.

Wizard Item Palette

The Wizard Item Palette is organized into sections based on the type of item. The "All" palette lists all of the items available, regardless of type.

For more information on working with wizard items, see Wizard Items.

Wizard Canvas

The wizard canvas area provides a graphical method of adding items to the wizard. All items and connections in the wizard are displayed in this area.

The canvas area is 5000 x 5000 screen pixels. This area expands to the right and bottom if you drag an item off the canvas. You can’t drag an item off the top or left sides of the canvas.

All items are currently displayed on the canvas. The scroll bars move the canvas around based on the window size. (You can resize the window.)

The canvas view serves as a data flow diagram rather than a process flow diagram. This means that a single output value in one item may be connected to multiple input values on other wizard items.

For more information on working with wizard items, see Wizard Items.

Wizard Properties

The Wizard Properties area provides wizard information such as name, menu location, and execution mode. For more information, see Create a Wizard, Wizard Menu Location, and Wizard Execution Mode.

Wizard Item Properties

The Wizard Item Properties area displays the properties of the selected wizard item and enables you to set the execution order and execution mode.

For more information on working with wizard items, see Wizard Items.
