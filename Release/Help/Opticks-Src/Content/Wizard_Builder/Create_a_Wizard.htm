Create a Wizard

  1. On the Opticks Main Application window, go to the Tools menu and select Wizard Builder.

The Wizard Builder window is displayed. For more information about this window, see Using the Wizard Builder.

  1. Add wizard items.
  1. Connect the wizard items. See Connect Wizard Items.
  2. In the Name field, type in a name for the Wizard. It’s best to use a name that describes the wizard’s functionality.
  3. The Menu Location allows you to specify a location and command name for the wizard in the Opticks menu bars or toolbars. For more information, see Wizard Menu Location.
  4. By default, the entire wizard is set to run in interactive mode. To run the wizard in batch mode, select the Run Wizard in Batch Mode checkbox. It is also possible to have an interactive wizard with individual batch mode items. For more information on interactive and batch modes, see Wizard Execution Mode.

Execute the Wizard

The Execute button on this window allows you to run the wizard from the Wizard Builder window.

This button is primarily used to test the wizard when you create it. Use it to ensure that the wizard is running correctly. Clicking this button is the same as selecting the wizard’s menu command (if set).
