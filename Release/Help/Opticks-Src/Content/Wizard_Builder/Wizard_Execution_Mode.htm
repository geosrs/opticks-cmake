Wizard Execution Mode

A wizard can operate in one of the following modes:

  • Interactive mode. Interactive mode indicates that the Opticks Main Application window is available and dialogs or other interface items may be displayed. Wizard operating in interactive mode may depend on information from the user during execution.
  • Batch mode . Batch mode wizards run without input from the user. A batch mode wizard can be executed in an environment outside the Opticks Main Application window or from an interactive Opticks wizard.

Note: The execution mode of individual wizard items is separate from the execution mode of the entire wizard.

If the wizard is set to run in batch mode, all items will be forced to run in batch mode also.

If the wizard is set to run in interactive mode, individual items can run in either interactive or batch mode.

See also Execution Mode of Wizard Items.

Setting the Execution Mode

To change the wizard's execution mode:

  1. On the Wizard Builder window, under Wizard Properties, go to the Execution Mode field.

  1. Use the drop-down list to select the execution mode: interactive or batch.
