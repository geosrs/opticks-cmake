Wizard Builder: Item Menu

The following menu items are available from the Edit Menu on the Wizard Builder window:

Menu Item

What It Does

Shortcut Keys
(if applicable)

Toolbar Button
(if applicable)

Select All

Selects all items in the wizard canvas.

Ctrl + A

 

Edit

Opens the Wizard Value dialog for the selected Value item. For information about this dialog box, see Add/Edit a Value Item.

 

Remove Item Deletes selected item(s) from the wizard. Ctrl + R

Increase Order

Increases the execution order number of the selected wizard item, meaning the item is executed later in the process. The order is displayed in the wizard item rectangle. See Wizard Items.

 

Decrease Order

Decreases the execution order number of the selected wizard item, meaning the item is executed earlier in the process. The order is displayed in the wizard item rectangle. See Wizard Items.
