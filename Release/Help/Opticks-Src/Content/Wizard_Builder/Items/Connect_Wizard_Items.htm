Connect Wizard Items

Wizard items are connected from an output node of one item to an input node of another item. Only nodes of the same type can be connected.

  1. Place your cursor over the (A) input node or (B) output node.
  2. Click using your left mouse button and drag the mouse to the (A) output node or (B) input node.
  3. Release the mouse button.

A connection line is added to the view in the same color as the nodes. If the connection cannot be made, the line disappears when you release the mouse.

Remove Connections

There are several ways to remove connections between wizard items.

To remove an individual connection:

  1. Click on the connection line to select it. Selection handles appear at both ends of the connection line.
  2. Press the Delete key on your keyboard.

OR

On the Wizard Builder Toolbar, click the Remove Item button.

If you change the execution mode of a wizard item to batch mode, the connections to that item are removed. See Execution Mode of Wizard Items.

If you change the execution mode of the entire wizard from interactive to batch mode or from batch to interactive mode, all connections in the wizard are removed. See Wizard Execution Mode.
