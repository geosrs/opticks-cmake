Add/Edit a Value Item

To add a Value Item using drag-and-drop:

  1. On the Wizard Builder window, go to the Wizard Item Palette.
  2. Select the Value tab.

  1. Select the Value Item you want to add and drag it to the canvas area of the Wizard Builder Window

The Wizard Value Item dialog box is displayed.

  1. Enter the name of the value item in the Name field.

The value item type is automatically selected in the Type list when you drop the wizard item onto the canvas. For more information, see Value Item Types.

  1. Enter the item value in the Value field.
  2. Click OK. The item is added to the Wizard Builder canvas.

To add a value item from a node:

  1. If you already have a wizard item on the canvas, double-click on the node.

The Wizard Value Item dialog box is displayed. The name of the item is automatically filled in.

  1. Enter the item value in the Value field. (If the item is going to be in Interactive mode, you can also add the value when you execute the wizard. See Enter Wizard Values When Executing the Wizard.)
  2. Click OK. The item is added to the Wizard Builder canvas.

To add multiple value items:

  1. On the Wizard Builder window, go to the Wizard Item Palette. It's probably easiest to select the Value tab, but you can also select items from the All tab.
  2. Select the value items you want to add. You can click the items while holding down the Ctrl key on your keyboard or you can drag a selection rectangle around the items.
  3. Drag the items to the Wizard Builder canvas.
  4. For each value item you selected, the Wizard Value Item dialog box is displayed. Fill in the value item information and click the OK button.

To edit a value item:

  1. Select the value item on the Wizard Builder canvas.
  2. On the Wizard Builder Toolbar, click the Edit Item button.

OR

Double-click the item.

  1. In the Wizard Value Item dialog box, edit the value item fields.
  2. Click OK.
