Wizard Items

A wizard item is a basic element of a wizard. It contains the information necessary to execute a single process in a wizard.

The wizard items are displayed in the Wizard Item Palette on the left side of the Wizard Builder. The "All" tab displays all of the available wizard items. The other tabs display the items by group such as algorithms, exporters, importers.

Each item has a name, type, a variable number of input and output nodes, an execution mode, and an execution order number. Note that the execution order may not be the same as the display order.

Adding Items to the Wizard

To add items to your wizard, simply drag them from the palette and drop them onto your canvas.

To add several items at a time. hold down the Shift or Ctrl key on your keyboard, select the items, and drag them onto the canvas.

Anatomy of a Wizard Item

Name

When you select a wizard item, the name is displayed beneath the rectangle.

The name of a Value item is user-defined. When you drag a value item onto your canvas, the Wizard Value Item dialog box is automatically displayed. Use this dialog box to enter the value item information. For more information on adding value items, see Add/Edit a Value Item.

All other wizard items have names that identify their functionality. The name cannot be changed.

Type

A wizard typically executes plug-ins in a certain sequence. Since plug-ins do not always have common input and output values, other item types are available to provide greater flexibility to connect items.

Input and Output Nodes

A wizard item can contain a variable number of input and output nodes. A Value item has no input nodes and a single output node that contains the hard-coded value.

The input nodes represent the input values that the item accepts. Input nodes are always located on the left side of the item rectangle.

The output nodes represent the output values that the item generates. Output nodes are always located on the right side of the item triangle.

The nodes are color-coded. To see the node name and type(s), move your mouse over the node. A tool tip is displayed.

Some nodes support multiple input types. These nodes have a down arrow button next to them.

The available types are displayed in the tool tip for the node.

To change the type, click the down arrow button. The node changes color, indicating the current type.

"Get..." Items

"Get..." items allow you to select items when running a wizard. For example, the "Get Layer" wizard item displays a dialog that lists all available layers and allows you to select which layer will be used in the next step. If you double-click on an input node of a compatible type, the appropriate "Get..." item is automatically created.

Execution Mode

Wizard items can operate in one of the following modes:

  • Interactive mode . Interactive mode indicates that the Opticks Main Application window is available and dialogs or other interface items may be displayed. Wizard items operating in interactive mode may depend on information from the user during execution.
  • Batch mode . Batch mode items run without input from the user. Results are not displayed in Opticks. Results are stored internally.

Items in interactive mode have a black border.

Items in batch mode have a red border.

Note that the execution mode of wizard items is separate from the execution mode of the entire wizard. If the wizard is set to run in batch mode, all items will be forced to run in batch mode also. If the wizard is set to run in interactive mode, individual items can run in either interactive or batch mode.

For more information, see Execution Mode of Wizard Items.

Execution Order Number

Each item contains an execution order number that identifies the item’s position in the wizard execution sequence. This means that an item numbered “1” is the first wizard item executed, an item numbered “2” is the second item executed, and so on.
