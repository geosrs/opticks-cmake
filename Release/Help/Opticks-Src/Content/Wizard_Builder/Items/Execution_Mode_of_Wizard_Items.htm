Execution Mode of Wizard Items

Wizard items can operate in one of the following modes:

  • Interactive mode. Interactive mode indicates that the Opticks Main Application window is available and dialogs or other interface items may be displayed. Wizard items operating in interactive mode may depend on information from the user during execution.
  • Batch mode. Batch mode items run without input from the user.

Note: The execution mode of individual wizard items is separate from the execution mode of the entire wizard.

If the wizard is set to run in batch mode, all items will be forced to run in batch mode also.

If the wizard is set to run in interactive mode, individual items can run in either interactive or batch mode.

The mode type must be set before making connections between input and output nodes in the Wizard Builder. Switching modes destroys the connections between wizard items. Only Value item connections are preserved.

A wizard item may have different input and output nodes for interactive and batch mode. This is because more inputs are usually necessary for batch mode.

Interactive mode items have a black border in the wizard canvas.

Batch mode items have a red border.

The inputs and outputs may vary based on the mode.

Occasionally, wizard items may not support execution in a specific mode. When a mode is unsupported, the item shows no input or output nodes and has a brown border, indicating that it’s disabled.

Setting the Execution Mode for Wizard Items

By default, a wizard item is set to run in interactive mode (unless the entire wizard is set to batch mode).

To run an item in batch mode:

Right-click on the item on the wizard canvas and select the mode from the drop-down list.

OR

  1. Select the item on the wizard canvas.
  2. Go to Wizard Item Properties.
  3. Go to the Execution Mode field and select Batch.

Change the Execution Mode

To change the execution mode for a wizard item:

Right-click on the item on the wizard canvas and select the mode from the drop-down list.

OR

  1. Select the item on the wizard canvas.
  2. Go to Wizard Item Properties.
  3. Go to the Execution Mode field and select the mode from the drop-down list.
