Wizard Menu Location

The Menu Location in the Wizard Builder allows you to add the wizard command to the Opticks Main Application window. You can add the command to:

  • A custom drop-down menu on the Opticks menu bar
  • An existing Opticks drop-down menu
  • A custom drop-down menu on a custom toolbar

Note: You can only create one menu location for a wizard.

The changes to the drop-down menus or the toolbars go into effect the next time you open Opticks or if you go to the Tools drop-down menu and select Update Wizard List.

Adding the wizard command to the Opticks Main Application window allows you to run the wizard from the main window without opening the Wizard Builder.

To add the wizard to a custom drop-down menu on the Opticks menu bar:

Follow these steps to add the wizard to a new drop-down menu. For example, you can create a separate drop-down menu called “My Wizards” and access your custom wizards from this drop-down menu.

Note: The custom drop-down menu will be located between the View drop-down menu and the Layout drop-down menu.

  1. On the Wizard Builder window, with the wizard open, type in the location in the Menu Location field. The format should be as follows:

Drop-down menu name\Menu item

Example: “Wizards\MyWizard” will look like this:

  1. Save the wizard. To make the drop-down available immediately, select Tools, Update Wizards List. Otherwise, the drop-down will be available the next time you open Opticks.

You can also create sub-menu items as follows:

Drop-down menu name\Menu item\Sub-menu item

To add the wizard to an existing Opticks drop-down menu:

Follow these steps to add the wizard to a drop-down menu that’s part of the Opticks Main Application window. Be sure you put an ampersand (&) in front of the menu name.

  1. On the Wizard Builder window, with the wizard open, type in the location in the Menu Location field. The format should be as follows:

&Drop-down menu name\Menu item

Example: “&Help\My Test” will look like this:

  1. Save the wizard. The next time you open Opticks, the wizard command will be available from the drop-down menu.

You can also create sub-menu items as follows:

&Drop-down menu name\Menu item\Sub-menu item

To add the wizard to a customer drop-down menu on a custom toolbar:

Adding a wizard to a toolbar creates a new toolbar if a toolbar with that name does not already exist.

To add a wizard to a toolbar, be sure you enclose the toolbar name in brackets.

  1. On the Wizard Builder window, with the wizard open, type in the location in the Menu Location field. The format should be as follows:

[Toolbar name]\Menu item

Example: “[ Wizards]\MyWizard” will look like this:

  1. Save the wizard. To make the drop-down available immediately, select Tools, Update Wizards List. Otherwise, the drop-down will be available the next time you open Opticks.

You can also create sub-menu items as follows:

[Toolbar name]\Menu item\Sub-menu item

Note: You can display and move the custom toolbar as you would the other toolbars in the Opticks Main Application window.
