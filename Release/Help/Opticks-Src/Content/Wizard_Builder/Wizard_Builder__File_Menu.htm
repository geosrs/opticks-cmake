Wizard Builder: File Menu

The following menu items are available from the File Menu on the Wizard Builder window:

Menu Item

What It Does

Shortcut Keys
(if available)

Toolbar Button
(if available)

New

Creates a new, empty wizard in the current Wizard Builder window. If you have an unsaved wizard open, the application displays a dialog box asking if you want to save the wizard.

Ctrl + N

Open

Opens an existing wizard. If you have an unsaved wizard open, the application displays a dialog box asking if you want to save the wizard.

Ctrl + O

Save

Saves the current wizard.

Ctrl + S

Save As

Saves the current wizard with a new name.

 

 

Print

Prints the items as represented in the wizard canvas.

Ctrl+ P

Close

Closes the Wizard Builder window. If you have an unsaved wizard open, the application displays a dialog box asking if you want to save the wizard.
