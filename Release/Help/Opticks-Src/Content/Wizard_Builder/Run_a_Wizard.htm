Run a Wizard

There are two ways to run a wizard:

  • Open the wizard in the Wizard Builder and run it by clicking the Execution button.
  • Run the wizard from the Opticks Main Application window. This requires you to save the wizard with a menu location.

Run the Wizard from the Wizard Builder

  1. From the Opticks Main Application window, go to the Tools menu and select Wizard Builder. This opens the Wizard Builder window.
  2. Open the wizard.
  3. Click the Execute button.

Run the Wizard from the Main Window

To run a wizard from the Opticks Main Application window, you have to first save it with a menu location. See Wizard Menu Location.

After you save it with a menu location, you must also update the list of wizards by going to the Tools menu and selecting Update Wizard List.

To run the wizard, go to the menu location and select the wizard’s command.
