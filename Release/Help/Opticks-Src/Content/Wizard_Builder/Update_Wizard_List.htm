Update Wizard List

Update Wizard List Use the Update Plug-In List option to scan the Wizard directory for new wizards, remove deleted wizards from the list, and refresh the Wizard list.

On the main Opticks window, go to the Tools menu and select Update Wizard List.

Any changes you have made to the Wizard directory since opening the current session of Opticks or the last update will be applied. If you have added wizards to your drop-down menus or a toolbar, the wizards will now be available from those locations.
