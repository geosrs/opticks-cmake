Open a Wizard

You can open existing wizards to view, edit, or run them.

To open a wizard:

  1. On the Opticks Main Application window, go to the Tools menu and select Wizard Builder.... This opens the Wizard Builder window.
  2. On the Wizard Builder window, go to the File menu and select Open.

OR

Select Ctrl + O.

OR

On the Wizard Builder Toolbar, click the Open button.

  1. In the Open dialog box, navigate to the wizard you want to open. Click the Open button. The wizard items are now displayed in the Wizard Builder window.
