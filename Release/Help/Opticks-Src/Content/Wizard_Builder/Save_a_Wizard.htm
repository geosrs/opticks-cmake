Save a Wizard

To save a wizard:

  1. In the Wizard Builder window, go to the File menu and select Save.

OR

Press Ctrl + S.

OR

On the Wizard Builder Toolbar, click the Save button.

  1. If this is a new wizard, the Save As dialog box is displayed. It displays a default name and location for the wizard. The default directory of the Wizard Path is set in the File Locations area of Session Options. This is different from the wizard name that appears in the Wizard Builder.

You can change the location. You can also change the wizard name.

  1. Click the Save button to save the wizard.
