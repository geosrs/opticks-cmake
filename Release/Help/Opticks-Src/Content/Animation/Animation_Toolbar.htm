Animation Toolbar

The Opticks Animation Toolbar contains the controls that enable you to select how the animation will be played.

To access the Animation Toolbar:

Go to the View menu and select Create Animation.

OR

If the image already has animation associated with it, right-click in the toolbar area and select Animation.

Button/Control

Animation

Stop. Stops the animation and sets the current frame to the first frame (if in play forward mode) or the last frame (if in play backward mode). This button remains depressed to indicate the current animation mode.

(Play)

(Pause)

Play/Pause. Plays or pauses the animation. This button remains depressed to indicate the current animation mode. It switches to the Pause button while the animation is playing.

Step backward. Steps the animation back one frame.

Step forward. Steps the animation forward one frame.
Slow Down. Slows down the frame speed. This is reflected in the frame speed field.

Frame speed. Use this drop-down list to select the frame speed. For more information, see Frame Speed.

You can set the values that appear in this drop-down list using the Animation Options.
Speed Up. Increases the frame speed. This is reflected in the frame speed field.
Change Direction. Reverses the direction of the playback.

Animation Cycle. Determines the behavior of the animation player after the final frame is displayed. See Animation Cycle.

You can set the default using the Animation Options.
Animation Bumpers. Enables you to set bumpers that limit the playback range of an animation. See Animation Bumpers.

Progress slider. Shows the progress of the animation as it plays. You can click and drag the slider value or click in the slider channel to increment the current frame by a small amount.

Frame text. The current frame value of the movie player.

For players that animate according to frame number, the text is displayed as:

Frame x/y

x is the current frame number. y is the frame number of the last frame.

For players that animate according to time value, the text is displayed as:

yyyy/MM/dd hh.mm.ss.zzzZ

yyyy is the year. MM is the month. dd is the day. hh is the hour. mm is the minute. ss is the second. zzz is the millisecond. Z is Zulu time.
