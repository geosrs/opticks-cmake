Frame Speed

The frame speed is a measure of how fast the animation player moves through the frames of the animation. It is not a frame rate or a strict number of displayed frames per second.

For players that animate according frame numbers, the frame speed value is a multiplier based on one frame per second. Below are examples of frame speeds and their displays.

Frame Speed ...

Displays....

5

5 frames per second

1

1 frame per second

0.25

1 frame every 4 seconds

For players that animate according to time values, the frame speed value is a multiplier based on one second of movie time equal to one second of real time.

Time Value ...

Displays....

5

5 seconds of movie time in 1 second of real time

1

1 second of movie time in 1 second of real time

0.25

0.25 seconds of movie time in 1 second of real time

Note: If the Can Drop Frames command has been selected and the requested frame speed cannot be achieved due to system hardware limitations, frames will be dropped or not displayed to preserve the requested speed.
