Play Animation

To cycle through the entire animation:

On the Animation Toolbar, select the Play button.

OR

1. Go to the Session Explorer Window and select the Animations tab.

2. Select the image name and right-click. Select Play.

To move forward or back one frame:

On the Animation, select the Step Forward button or the Step Backward button.

OR

1. Go to the Session Explorer Window and select the Animations tab.

2. Select the image name and right-click. Select Step Forward or Step Backward.
