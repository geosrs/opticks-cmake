Animation Bumpers

Animation Bumpers enable you to limit the playback range of an animation to a start and stop frame ID or frame time.

Typically, the "left bumper" is the "start" frame and the "right bumper" is the "end" frame. If you change the direction of the animation, the start and end frames are reversed.

Setting the Bumpers

To set the bumpers using the Animation Toolbar:

  1. Move through the animation until the frame that you want to use as the left bumper is displayed.
  2. On the Animation Toolbar, click on the arrow next to the Bumpers Tool button.
  3. Select Snap Left Bumper to current frame.
  4. Move through the animation until the frame that you want to use as the right bumper is displayed.
  5. On the Animation Toolbar, click on the arrow next to the Bumpers Tool button.
  6. Select Snap Right Bumper to current frame.

To set the bumpers using the Animation Tab in the Session Explorer:

Move through the animation until the frame that you want to use as the left bumper is displayed.

  1. On the Session Explorer, select the Animation Tab.
  2. Select the name of the image animation, right-click, and select Left Bumper to Current Frame.
  3. Move through the animation until the frame that you want to use as the right bumper is displayed.
  4. Select the name of the image animation, right-click, and select Right Bumper to Current Frame.

To set the bumpers using shortcut keys:

You can create shortcut keys for the following:

  • Adjust Bumpers
  • Bumpers Enabled
  • Reset Bumpers
  • Restore Bumpers
  • Snap Left Bumper to Current Frame
  • Snap Right Bumper to Current Frame
  • Store Bumpers

For instructions, see Creating Shortcut Keys.

After you've set the bumpers, click the Bumpers Tool button to toggle the bumpers on and off.

Adjusting the Bumpers

After you set the bumpers, you can adjust them.

  1. Go to the Animation Toolbar. Click on the arrow next to the Bumpers Tool button and select Adjust Bumpers....

OR

On the Animation Tab of the Session Explorer, right click on the name of the animation and select Adjust Bumpers....

The Adjust Bumper dialog box is displayed.

  1. Enter the Start Frame and Stop Frame. You can select the start and stop frames using the sliders or by typing in the number or by using the up and down arrows.
  2. Click the OK button.

Resetting the Bumpers

Resetting the bumpers returns the bumpers to the first and last frames of the animation.

Go to the Animation Toolbar. Click on the arrow next to the Bumper Tools button and select Reset Bumpers.

OR

On the Animation Tab of the Session Explorer, right-click animation name and select Reset Bumpers.

The bumpers are restored to the first and last frames of the animation.

Storing the Bumpers

Storing the bumpers saves the current values of the bumpers and the on/off status of the bumpers in your configuration settings. Only one instance is saved. The values will be overwritten each time you store the bumpers..

Go to the Animation Toolbar. Click on the arrow next to the Bumper Tools button and select Store Bumpers.

OR

On the Animation Tab of the Session Explorer, right-click animation name and select Store Bumpers.

The bumpers are stored for the current session.

Restoring the Bumpers

Restoring the bumpers returns the right and left bumpers to the frames you selected before resetting them.

Go to the Animation Toolbar. Click on the arrow next to the Bumper Tools button and select Restore Bumpers.

OR

On the Animation Tab of the Session Explorer, right-click animation name and select Restore Bumpers.

The bumpers are restored to the right and left bumpers.
