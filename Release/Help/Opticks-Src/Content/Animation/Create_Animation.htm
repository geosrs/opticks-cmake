Create Animation

In Opticks, an animation cycles through the bands of a displayed data set in grayscale display mode.

To create an animation:

  1. Be sure the data set window is active.
  2. On the Opticks menu bar, go to View and select Create Animation.

Opticks creates an animation with the same name as the active data set and activates the Animation Toolbar. The animation control is automatically set to:

Animation mode: Stop

Frame speed: 1

Animation cycle: Play once

You can set the default animation cycle and the frame speed values. See Animation Options.

You can also set "bumpers" to start and stop the animation at frames other than the first and last frames. See Animation Bumpers.

If you are using raster layers, you can use basic RGB full color video playback. Each color channel of a raster layer must display a separate raster element. See also Raster Layer Properties.
