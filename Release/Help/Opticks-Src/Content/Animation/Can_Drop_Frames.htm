Can Drop Frames

The "Can Drop Frames" command enables an animation to drop frames to maintain the requested frame speed. When "Can Drop Frames" is not selected, the animation plays all frames, slowing down as needed.

To toggle this command on and off:

  1. On the Session Explorer, select the Animations tab.
  2. Right-click on the top-level animation controller item and select Can Drop Frames.

When this command is enabled, a checkmark is displayed in front of the "Can Drop Frames."
