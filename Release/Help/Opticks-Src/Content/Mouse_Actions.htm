Mouse Actions

Mouse Action What it does
Middle button (pressed) Zooms to point by dragging the mouse, independent from a mouse mode.
Middle button (clicked) Pans the image at a speed relative to the clicked point, independent from a mouse mode.
Wheel up Zooms in or out on point, based on value in Options. See the Mouse Wheel Zoom Direction information in Session Options.
Wheel down Zooms in or out on point, based on value in Options. See the Mouse Wheel Zoom Direction information in Session Options.
Right button + Ctrl Activates the dynamic inset (the zoomed-in region of the scene.)
Right button Opens a context menu.
Left button + Ctrl Copies currently selected annotation objects.

Annotation mode only.

Left button + Shift Extends the current annotation selection.

Annotation mode only.
