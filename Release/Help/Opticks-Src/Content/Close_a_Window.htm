Close a Window

To close an image display window in Opticks:

  1. If you have several files open, select the window of the file you want to close.
  2. On the Opticks Main Application window, go to the File menu and select Close.

OR

On the Standard Toolbar, click the Close a File button.

OR

Click the button in the upper right corner of the image window.

If you have set the "Prompt to save" option, you will be prompted to save the session. See Session Options for more information.
