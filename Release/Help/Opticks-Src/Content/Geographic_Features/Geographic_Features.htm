Geographic Features

Before using the geographic features, you must have an image loaded and georeferenced.

Geographic Features enables you to set up files and add them to the menu to quickly load them at a later time. The files you can load are:

  • Shape file without ArcGIS
  • Shape files with ArcGIS
  • Shape files from the ArcSDE database.

To import the geodatabase:

Go to the Geographic Features Toolbar. Select Geographic Features, Import geodatabase.

To edit the menu:

Go to the Geographic Features Toolbar. Select Geographic Features, Edit this menu.

Click the add or remove buttons to add or remove features.

The Geographic Feature window or the Feature Menu Editor Window is displayed. Each window has three tabs.

Connection Tab

Display Tab

Clipping Tab

The Test connection button enables you to test the Opticks connection to the database.
